# 5.日期
## 5.1 30days
       1)保全
         1.1）证据、财产、行为 保全后30day内 不起诉的，法院裁定解除保全措施
         1.2）国内外相同
       2）对裁定二审的审限
         2.1）立案之日起30days
         2.2）报院长批准可以延长
       3）特别程序的审限
         3.1）立案之日起30days || 公告期满之日起30days
         2.2）报院长批准可以延长
              exp:选民资格案件
      4）管辖权异议
         涉外 答辩期结束前 30days + 可延长（相对不变期间）
         国内 答辩期结束前 15days + 绝对不变期间
      5) 调解协议的司法确认
         调解协议生效之日起 30days 向调解组织所在地 基院 提出
      6）公益诉讼 调解、和解 的公告期
         >= 30days 
      7) 检院 拟 提起公益诉讼 的公告期
         30days

# 6.离婚
## 6.1 管辖
       1）原告住所地
          一方离开住所地超过1years，另一方起诉离婚
       2）被告住所地
          双方离开住所地超过1years，另一方起诉离婚

## 6.2 提三撤不予受理
      婚姻无效、撤销||解除婚姻关系 的 判、裁、调

## 6.3 调解
       1）身份关系的确认不能调解
          身份关系的解除可以调解
              like:离婚
       2）必须制作调解书
             if 一审 判决解除离婚、收养关系 + 二审调解和好、维持收养关系；
                   必须制作调解书
                       cuz:二审要制约一审裁判，否则一审裁判就没有法律文件否定
       3）可以不制作调解书
          调解和好的离婚案件

## 6.4 撤诉
       if 离婚案件 + P 撤诉||按照撤诉处理 + 没有新情况、新理由 + 6mon 内起诉；
                不予受理

## 6.5 一事不再理的例外
      if 判决不准离婚、调解和好的 D + 有新情况、新理由 + 6mon 后起诉；
              应当受理

## 6.6 特殊规定
       1）F 怀孕、分娩1years 内||中止妊娠6mon 后；
          usu:M 不得提离婚
          exp:确有必要
       2）if 一方下落不明 + 另一方只起诉离婚；
             应当受理 + 对下落不明的人公告送达
       3）if 夫妻关系存续期间 + 单独提出精神损害赔偿 + 不离婚；
             不予受理
