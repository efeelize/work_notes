# ch_2 制度
# I.管辖
## 1.级别管辖
### 1.1 一般规则
       1）上可审下
          ie.必要时
       2）下不可审上
          usu:应当由上级管辖的，不能再指定下级
          exp:if 缓刑、假释期间又犯新罪
                 审判新罪的法院 一并 撤销缓刑、假释
              if 上级法作出的缓刑、假释
                 下级法可以撤销
       3）就高不就低
          usu:一人||一罪 属于 上级法，全案移送
          exp:未成年人可以分案

### related 
           1）漏罪
              原审地法院管辖
              更为合适的；罪犯服刑地||犯罪地
           2）新罪
              服刑地法院管辖
              unless 犯罪地抓获+发现其在脱逃期间犯的罪；犯罪地
              if 死缓期间脱逃+在犯罪地抓获;死缓执行地中院 执行死刑

### 1.2 指定管辖
#### 1.2.1 情形
           1）管辖不明
              inti:B 中院为受案法，高院 指定 A中院审理, 
                   if 公诉；
                      案卷退回 B 检院；B 检院 移送 A 检院；A 检院 向 A 中院起诉
                   if 自诉
                      案卷直接移送 A中院
           2）管辖不宜
              ie.A省 省委书记 不能在 A省 审判
                 上一级法 自己管||指定 提出请求的同级其他法
           3）规避管辖
              ie.针对检院规避法院的级别管辖
                 inti: 中院 一审，高院 二审 ，A 上诉
                       if 高院 二审发回重审，市检撤诉，
                          市检 让 区检 向 基院 提起公诉
                       else 基院 应当层报 高院
                            高院 可以决定移送 中院 || 其他法院
           4）监察法
              ie.可上可下
              4.1）上级  可以指定 下级||其他 监察机关管辖
              4.2）重大复杂 ，可以 报请 上级  监察机关管辖   
#### 1.2.2 程序
           1）争议
             1.1）都有管辖权
                  S_2.1 3）
             1.2）管辖权争议
                  1）应当在审限内协商
                  2）不成的，分别逐级报请 共同上级法 指定管辖
             1.3）监察法
                  由共同上级 监察机关确定
           2）上级法的处理
              需要移送中院的，应当在报请院长决定后,最迟审限届满前15day前+书面请求移送
              
              if 受理前；中院接到申请后10days 内做出决定
                       if 不同意移送；下达不同意移送决定书
                          else if 同意；下达同意移送决定书 + 书面通知同级检 
                
              if 受理后；应当依法审判

           2）申请 书面 || 口头
           3）决定 回避
           4）甲申请乙回避，乙被决定回避，乙无权复议

### 1.1 身份不当回避

### 1.2 任职回避
#### 1.2.1 法
           审判人员 && 其他工作人员 
           1）+ 离职后2years 内 + 不得以律师身份
              ie.可以 非律师 身份担任
           2）终身 + 不得担任原任职法院 辩护人
              exp:作为被告人的监护人、近亲属
           3）配偶、父母、子女 + 不得担任原任职法院 辩护人
              exp:作为被告人的监护人、近亲属

#### 1.2.2 检
           检察人员
           1）2)S 法
           3）配偶、子女 + 不得担任原任职法院 辩护人
              notes:无父母、无例外

#### 1.2.3 监察委
           监察人员
           1）+ 离职后3years 内 + 不得 从事与 监察&&司法工作 相关联&&可能发生利益冲突 的职业
           2）涉密的 遵守托秘期的规定

### 1.3 跨诉讼阶段
        ie.只能参与一个诉讼阶段
        if 二审裁定发回||按照再审重新审理；一审合议庭需要重组
        if 二审裁定发回 + 又上诉到二审||死刑复核；二审法院没有审过，合议庭直接审理无需重组

### 1.4 违法违规
        1）only 这一项是申请人举证，else 是办案机关举证
        2）其他不当行为 + 可能影响公正审判的

## 2.种类
### 2.1 自行回避
        1）回避决定的做出不需要告知当事人

### 2.2 申请回避
        1）当事人 及其 法、诉、辩、
           Incl:all 当事人
           1.1）if 不服回避决定，可以申请复议
                   if 不服公；5days 内+原机关复议 + 1次，原机关5days 内做出复议
                   if 不服检；5days 内+原机关复议 + 1次，原机关3days 内做出复议
                   if 不服法；接到决定时复议

        2）diff && 申请变更强制措施
           被羁押的 嫌疑人、被告人 法、近、辩、
           ！incl：被害人

### 2.3 指令回避
        ie.领导责令
        1）回避决定的做出不需要告知当事人

## 3.程序
     3.1）决定回避的主体决定 之前行为的效力
     3.2）老大决定
          if 审判长；找院长
             if 院长；找审委会

          if 检察人员；找检察长
             if 检察长；找同级检委会

          if 其他人员；找公安机关负责人 ！incl:派出所长
             if 公安机关负责人；找同级检委会


# III.证据制度
## 1.基本原则
### 1.1 证明力
        1）关联性规则
           1.1）证据本身与案件联系的紧密程度
           1.2）关联性是证据证明你的原因
                but  有关联性的证据未必具有可采性
        2）补强证据规则
           2.1）2个证据都要有证据能力
           2.2）有独立的来源
                   like:有利的人说了有利的话
                        限人作证 + 可以正确认知、表达
                   ！:根据被告人供述找到的证据，只是互相印证

### 1.2 证据能力
        1）证据裁判规则
              diff && 神明裁判

        2）自由心证
              like：存在多个鉴定意见的时候，由法官进行判断

        3）自白任意||非任意自白排除规则
           3.1)^基本确立
           3.2)严禁刑讯逼供
           3.3)diff && 非法证据排除

        4）传闻证据规则
           ie.真的证人+没有出庭
           ^ 未确立严格的传闻证据排除规则

        5）意见证据规则
           5.1）only 证人
                ！incl:专家辅助人、鉴定人
           5.2）可以采纳
                5.2.1）醉酒、麻醉、中毒 + 能正常感知||正确表达
                5.2.2）猜测、评论性、推断性 + 根据一般生活经验判断符合事实

        6）最佳证据规则||原始证据规则
           6.1）only 文字材料
           6.2）^ 未明确规定

        7）证据转换规则
           7.1）行政 转 刑事
                7.1.1）物证可以在刑诉中作为证据
                       1）物证、书证、视听资料、电子数据
                       2）勘验检查笔录 + 检院审查认为符合要求
                7.1.2）言辞
                       1）usu 应当重新收集
                       2）exp
                         2.1)鉴定意见 + 检院审查认为符合要求
                         2.2）if 供述、证言、陈述;
                                +来源、收集程序合法+其他证据相互印证+检院审查认为符合要求
           7.2）监察 转 刑事
                都可以转，cuz 是按照刑事标准收集的

#### related：8）非法证据排除规则
                8.1）言辞证据 绝对排除
                     8.1.1）嫌疑人、被告人 + 供述
                            1）！incl:辩解
                            2）威胁+本人 && 近亲属+痛苦
                            3）限制自由
                            4）刑讯+痛苦
                            5）重复性供述
                               if 先受到刑讯，后基于先前影响作出重复性供述；排除
                                  else if 更换了侦察主体||更换了诉讼阶段
                     8.1.2）证人证言、被害人陈述
                            1）暴力、威胁
                            2）限制自由
                8.2）物证书证  相对排除
                     只要可以补正||作出合理解释

## 2.分类
### 2.1 物证
        1)USU 间接证据
          like:痕迹
        2）直接排除的书证、物证
          % 真伪不明、来源不明、无法解释 %
          % 不能、未通过  %  

### 2.2 书证
        ie.通过内容来证明案件事实
           like:缴获的淫秽书籍->既是书证，也是物证

### 2.3 证人证言
#### 2.3.1 trait
           1）only 自然人
              单位开的证明是书证
           2）当事人不能做证人
              有冲突的时候，证人身份优先
                  like:警察在执勤中目睹现场->证人
                       侦查人员就刑讯合法性进行说明->专门机关
                       就刑讯合法播放了录音录像->证明当时打还是没有打->视听资料
                       播放讯问的过程的录音录像->嫌疑人供述、辩解
           3）补助：交通、住宿、就餐
                  ！incl:误工费
                         cuz:单位不得克扣、变相克扣工资、奖金、其他福利待遇

#### 2.3.2 出庭作证
           1）有异议：公诉人、当事人、辩护人、诉代
           +
           2）有影响：法院认为对定罪量刑
           ！incl:有必要

#### related:鉴定人出庭作证
     1）有异议：公诉人、当事人、辩护人、诉代
     +
     2）有必要：法院认为


#### 2.3.3 强制出庭
           1）incl：证人、被告人的配偶父母子女
           2)!incl:被害人、鉴定人
           3）院长签发+强制证人出庭令
              ！强制措施、！审判长签发
           4）不来的
              训诫||情节严重+院长批准+10days 以下拘留
              ！incl:罚款

#### related：鉴定人不得强制出庭
              1）usu:不来的 不作为定案依据
              2) exp:if 理由正当，法院可以决定延期||重新鉴定
                         like:不可抗力

#### 2.3.4 可以不出庭，但是要作证
           % 有病太远在国外 %
           % 民诉：有病太远遇灾害 %

#### 2.3.5 矛盾证言
           1)庭前 庭审 证言矛盾
             usu:庭前证言
             exp:证人做出合理解释 + 有相关证据印证 == 庭审证言
           2)if 不来||来了不作证；
                usu:不排除
                exp:庭前证言真实性无法确认的

#### 2.3.6 保护
           1）罪名
              incl:危国、恐、黑、毒
           2）对象
              incl:鉴定人、证人、被害人 本人&&近亲属
           3）scope
              3.1)信息：姓名、住址、单位
              3.2）外貌、声音
              3.3）被 特定的人员接触
              3.4）人身、住宅 专门性保护措施

#### 2.3.7 直接排除
           % 麻醉猜测未个别，核对翻译暴限胁，拒绝出庭不真实 %
           1）处于明显醉酒、麻醉、中毒 + 不能正确表达
           2）猜测性、评论性、推断性 + 一般生活经验判断不符
           3）询问 + 没有个别
           4）没有证人+ 核对确认+签名、捺印 的书面证言
           5）聋哑人+没翻译
           6）不懂当地方言的证人 + 没翻译
           7）暴力、威胁 手段取得
           8）拒绝出庭 + 真实性无法确认

### 2.4 鉴定意见
#### 2.4.1 trifle
           1）only 专门机关指派||聘请的专门人员 + 判断性意见
              倾向性意见是 参考
           2）针对 专门性问题
           3）出了问题 鉴定人负责，单位只是盖章证明来自机构
           4）鉴定意见、辨认笔录 属于言辞证据

#### 2.4.2 直接排除
           % 见错就排 %
