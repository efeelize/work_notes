### 2.1 财产保全的查扣冻
#### 2.1.1 保管
           usu:法院
           exp:被保全人
           exp's exp:他人、申请保全人

           if 担保财产；
               usu:担保物权人保管（无需担保物权人的同意）
               exp:法院保管 + 不消灭 质权、留置权 + 不消灭 优先受偿权

#### 2.1.2 对保管财产的使用
           usu:不得使用、处分
           exp:if 不动产||特殊动产 + 被保全人保管 + 使用对价值无影响；
                     可以使用，不得处分
                     only 被保全人可以使用，其他人不行

               if 鲜活易腐；
                     责令当事人及时处理 || 法院变卖  + 法院保存价款 

#### 2.1.3 第三人到期债权
           法院依 债权人的申请 裁定第三人不得对本案债务人清偿
           if 第三人要求偿付；
                     法院提存

### 2.2 保全的救济
#### 1） if 不服保全裁定
          自收到保全裁定之日起 5days 内 向作出法 复议

#### 2） if 保全造成损失
         1）诉中
            向受诉法院起诉
         2）诉前
            2.1）if 30days内未起诉
                 向采取保全措施的法 起诉
                    cuz 此时无受诉法院
            2.2）if 起诉了
                   1)向采取保全措施的法 起诉
                        此时也是受诉法院
                   2)向其他法院起诉
                     ie.财产所在地、被保全人住所地、其他法院
                     此时选择权：
                        向受诉法 || 采取保全措施的法 起诉

### 2.3 执行前保全
        1）only 依申请
        2) 在法律文书指定的 履行期届满后 5days 内申请 && 尚未进入执行前
           if 逾期不申请执行；
                     法院应当解除保全措施

### 2.4 保全措施的衔接
        1）if 二审法 对一审法 续保||采取新的保全措施;
                     自己实施 || 委托一审法实施
        2）if 再审法 对原保全措施 续保||采取新的保全措施;
                     自己实施 || 委托一审法实施 || 委托执行法
        3）if 一审后的上诉期间；
                      一审法保全

### 2.5 保全的解除
        1）诉前保全 + 30days 内未起诉
        2）财产纠纷 + 被申请人提供担保
        3）申请人在保全期间撤回申请 + 法院准许
        4）被申请人 复议 + 法院认为原行为错 + 做出新的裁定 + 撤销原保全裁定
        5）被申请人 依法履行了 法院判决的义务
        notes:撤销一审裁判，并不一定会解除保全 

# II.一审
# 1.普通
## 1.1 受理
       if P 交起诉状；
           法院应当接收 + 审查
              if 符合起诉条件；
                     7days 内 立案 + 通知当事人
                  if 当场不能判断；
                          应当接收 + 出具书面凭证
              if 欠缺材料；
                     应当及时告知
                       if 补齐材料后；
                          7days 内 决定是否立案
              if 不符合起诉条件；
                   7days 内 做出不予受理的裁定书（必须书面）
      
      if 立案后 + 不符合起诉条件；
             裁定驳回起诉（书面||口头）
      
      if 超过诉讼时效；
             应当受理
            if D 未提出诉讼时效抗辩；
                      法院不进行审查
            else if D 提出抗辩 + 法院审查成立的；
                         判驳
            if 到了二审才提出诉讼时效抗辩；
                  usu:不允许
                  exp:出现了新的事实理由

## 2.简易
### 2.1 积极条件
        1）一审基院
        2）事实清楚、关系明确、争议不大
    
          

### 2.2 消极条件
        % 下落不明重再审，人多公益要撤销 %
        1）起诉时 + D 下落不明
            cuz D 不能提出书面答辩状
        2）发回重审、再审
        3）共同诉讼中 一方||双方当事人 人数众多
           ie.代表人诉讼
        4）非争议案件
           incl:特别、督促、公示催告、破产
        5）涉及国社利益的
              like:公益诉讼（C_ch_6）
        6）三撤
             cuz 三撤应当合议庭开庭审理

### 2.3 转化
#### 2.3.1 普通转简易
           ie.当事人约定适用简易 + 开庭前提出 + 双方自愿 + 法院决定
           1）在一审普通案件中，当事人选择适用简易
           2）口头提出，记录笔录，双方当事人签字||捺印 确认
           3）已经适用简易程序，only 法院可以决定转为普通，当事人不得约定

#### 2.3.2 简易转普通
           1)方式
             1.1）依申请
                  当事人异议 + 法院审查同意
             1.2）依职权
                  1）法院发现 + 案情复杂 + 裁定转化
                  2）if P 提供了 D 准确地址 + 法院无法向 D 送达；
                          应当裁定转入普通
                       if P 未提供 D 准确地址 + 法院查证后仍不能确定 D 的送达地址；
                              裁定驳回起诉（cuz D 不明确）
           2）期限
              2.1）从简易程序立案之日起 
              2.2）连续计算
                   diff && 刑诉：重新计算

### 2.4 简化
#### 2.4.1 审判
           1）起诉可口头（不是 exp）
              委托可口头 （普通程序 usu 要书面）
              传唤可口头
              开庭通知可以 打电话、捎口信、传真、邮件 + 确认收悉（else 不可以缺席判 ）
              可以远程网络异地审判 + 双方同意
              可以当庭宣判（有 exp）
              usu 开一次庭（有 exp）
           2) if 同时到案；
                    当即审理（双方同意不需要举证期、答辩期） || 另定日期审理
           3）期限
              3.1）协商确定举证期 =< 15days
              3.2）diff && 普通 >= 15days
                           小额 =< 7days

#### 2.4.2 可以简化文书的情形
           1）达成调解协议的
              ie.当事人达成调解协议 + 需要制作调解书的
           2）承认的
              ie.一方明确表示 + 承认对方全部||部分诉讼请求的
           3）关涉私密的
              ie.国、个秘 + 一方要求简化文书 + 法院认为理由正当
           4）一致同意的
              ie.双方一致同意的

## 3.小额
### 3.1 积极条件
        1）符合简易程序的条件
        2）标的额为 省级上年度平均工资 30% 以下（包括本数）

### 3.2 消极条件
        1）重大财产法益
           ie.财产确权纠纷
        2）人身关系
        3）涉外
        4）IP
        5) 需要评估、鉴定的
        6）简易的消极范围

### 3.3 告知义务
       适用小额，应当告知 incl:
       审判组织、一审终审、审理期限、诉讼费用缴纳标准
