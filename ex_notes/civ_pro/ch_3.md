### related:诉前财产保全管辖
      财产所在地、被申请人住所地、其他（depends 具体案件类型）
      if 采取了诉前保全 ；
          if 在法定期间起诉；
                向采取保全的法院 || 其他有管辖权 的法院 起诉
             if 在法定期间起诉 + 造成损害；
                   受理起诉 || 采取保全 的法院 管辖
             else if 没有在法定期间起诉 + 造成损害；
                     采取保全措施的法院管辖

# II.一审
# 1.普通
## 1.1 受理
       if P 交起诉状；
           法院应当接收 + 审查
              if 符合起诉条件；
                     7days 内 立案 + 通知当事人
                  if 当场不能判断；
                          应当接收 + 出具书面凭证
              if 欠缺材料；
                     应当及时告知
                       if 补齐材料后；
                          7days 内 决定是否立案
              if 不符合起诉条件；
                   7days 内 做出不予受理的裁定书（必须书面）
      
      if 立案后 + 不符合起诉条件；
             裁定驳回起诉（书面||口头）
      
      if 超过诉讼时效；
             应当受理
            if D 未提出诉讼时效抗辩；
                      法院不进行审查
            else if D 提出抗辩 + 法院审查成立的；
                         判驳

## 1.5 暂停
## 1.延期 （决定）
     1）新证
       通知新的证人出庭、调取新的证据、重新鉴定、勘验、需要补充调查
    2）回避
       usu:暂停
       exp:保全 || 先予执行
    3）必须出庭的当事人、其他诉讼参与人 + 有正当理由 没来
    4）D 经过一次传唤拒不到庭

###  related:刑诉延期（决定）
     % 新证、补侦、回避 %

     3）变更、追加起诉 
     4）简易转普通
     5）检院二审 阅卷 超期
     6）检院发现需要补侦，合议庭应当同意
