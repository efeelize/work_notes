# ch_1 101

# I.概念
    1.民事诉讼法律关系
      ie.必须有一方是法院
         like: 法院 && 原告、法院 && 被告
    2.所有诉讼法 都是 公法
    3.在中国领域内的 任何人 都适用 我国民诉

# II.基本原则 
## 1.平等、同等、对等 原则
     1.1 平等
         1）P && D 权利相同
         2）P && D 权利对应
              like:起诉状-答辩状
                   本诉-反诉
     1.2 同等
         ie.一视同仁
            外国人 == 我国当事人
     1.3 对等
         ie.以牙还牙
            外国法对我国人进行限制，我国法也限制该国当事人

## 2.调解原则
     C_ch_3

## 3.辩论原则
     3.1 贯穿全过程
     3.2 也可使用书面形式
            ie.庭前提交起诉状、答辩状
     3.3 可以辩论的范围 incl:
         事实、法律、程序、实体
     3.4 主体 incl:
         第三人是广义的本案当事人
            But 不是本诉当事人

## 4.处分原则
    4.1 适用全过程
        incl：执行和解
    4.2 一般授权 && 特别授权的人 都有处分权

## 5.支持起诉
### 5.1 公益诉讼
#### 5.1.1）反诉
            不能提
            incl:环境、消费

#### 5.1.2）和解调解
           可以
           和解、调解协议 + 公告 + >=30days
              cuz 进行社会监督

#### 5.1.3）撤诉
           1）if 在法庭辩论终结 前 撤诉；
                      准予撤诉
                 else if 在法辩终结 后 撤诉；
                          不准撤诉
           2）普通诉讼规则
              2.1）在法辩 终结前 撤诉
              2.2）在法辩 终结后 + D 同意 == 撤诉

#### 5.1.4）管辖
           usu 侵权行为地、被告住所地 + 中院
           exp 环境公益诉讼 + 中院认为有必要 + 高院批准；
                 裁定 交 基院

#### 5.1.5）主体
           1）法律规定的机关 && 有关组织
              以自己的名义
           2）检院 
              if 1）的主体起诉；
                   检院 支持起诉
                else if 1）的主体不起诉 + 检院公告 + 满30days；
                     检院 提公益诉讼

#### 5.1.6）trait
           1）不排斥私益诉讼
           2）不以存在实际损害为前提

## 6.诚实信用
### 6.1 保证书制度
#### 6.1.1 证人
           usu:法院在证人出庭作证前 应当告知 + 责令签署
           exp:无限人
               if 拒签；
                    不得作证+自行承担相关费用
                    exp:无限人

#### 6.1.2 询问当事人
           usu:法院在询问当事人之前 可以要求 签署
               if 拒签 + 待证事实欠缺其他证据证明；
                    对其主张的事实不予认定

### 6.2 对恶意诉讼的强制措施
#### 6.2.1 诉讼
           if 当事人恶意串通+ 通过诉讼调解等方式侵害他人合法权益的；
                 驳回诉求 + depends 情节罚款拘留

#### 6.2.2 执行
           if 当事人恶意串通+ 通过诉讼调解等方式逃避履行义务的；
                 depends 情节罚款拘留

## 7.法律监督
     ie.检院有权监督
     7.1 全程监督
          incl:执行
     7.2 方式
         1）上级再审抗诉
         2）同级提出检察建议
     7.3 向相关人有调查核实权

# III.基本制度
## 1.合议庭组成
     1）人民陪审员一审普通可以有
     2）简易一审独任
     3）特别程序 
        usu 一审独任
             incl:督促/支付令程序、公示催告的公告审理阶段
        exp:必须合议 + 审判员组成
            选民资格、重大疑难、担保财产标的额超过基院管辖范围的 担保物权实现 、
            公示催告的除权判决阶段

## 2.回避
### 2.1 回避的申请
        1）方式
           1.1）口头 || 书面
           1.2）申请 || 职权
        2）时间
           usu:案件开始审理时
           exp:法庭辩论终结前
                  if 开始审理后才知道回避事由的

### 2.2 回避的决定
        1）审判人员->院长决定
             incl:陪审员、书记员
        2) 其他人员->审判长决定
             incl:鉴定员、勘验员、翻译
        3) 院长->审委会

### 2.3 回避的程序
        if 申请回避；
              被申请的人要暂停执行职务
                 exp:需要采取紧急措施的
                       ie.保全 || 先予执行
              法院应当在 3days 内 做出决定（文书是决定）
                 if 决定回避 + 更换人员；
                       诉讼程序继续进行 + 先前行为有效
                 if 不服；
                    向原法院复议；
                       复议期间，被申请回避的人不停止执行职务
        
        if 二审发现一审未回避（程序错）；
               裁定撤销原判 + 发回重审

## 3.公开审判
     1）绝对不公开
        1.1）国秘、个人隐私
        1.2）合议庭评议
     2） 依申请可以不公开
        2.1）商业秘密
        2.2）离婚
     3） 一律公开
        案件宣判

## related:公开质证
     1.usu:未经公开质证，不得作为定案依据
       exp:法院依职权调取的证据
     2.不公开质证的
       1）国密、个人隐私
       2）商业秘密
       3）if 庭前当事人认可 + 审判人员在庭审中说明 ;
                视为质证过

## 4.两审终审的例外
     1）最高院一审
     2）调解书
     3）非争议案件
        incl:特别程序、督促、公示催告、企业法人破产还债程序
     4）小额
        对小额提管辖异议的，法院的裁定一经做出，立即生效
     5）确认婚姻效力
        diff && 离婚是变更之诉，是两审终审

# IV.诉
## 1.class
     1.1 确认之诉
         depends 原告的诉讼请求
     
     1.2 变更之诉/形成之诉
         改变||消灭 既存的法律关系
     
     1.3 给付之诉
         1）诉讼标的物存在与给付之诉中
         1）停止侵害、排除妨害是给付内容为不作为的给付之诉

## 2.反诉
### 2.1 diff 反诉 && 反驳
        反诉 == 承认本诉 + 提出独立的反请求

### 2.2 反诉的条件
        1）反诉的当事人 是 本诉的原被告
        2）反诉 && 本诉 之间有牵连关系
        3）时间
           usu:举证期限届满前
           exp:if 法庭辩论结束前 + 可以合并审理；
                     法院应当合并审理
        4）对反诉有管辖权 + 适用同一诉讼程序
              cuz 一般合并审理


# V.管辖
# 1.管辖恒定
    1）incl:地域、级别管辖恒定
    2）exp:诉讼标的额的变化，应当调整 级别管辖
           if 提交答辩状期间届满后 + 原告增加诉讼请求金额 致使超过受诉法院 级别管辖；
               if 被告提出管辖异议；
                     法院进行审查；
                         if 异议成立；
                               裁定将案件 移送有管辖权的法院
                             esle if ；
                                  裁定驳回异议

### 5）公益诉讼

# 3.地域
   
## 3.1 人身关系
### 3.1.1 两地管
          1)if 监护案件（不服指定监护||变更监护关系）；
                被监护人住所地 || 被告住所地

### 3.1.2 原告所在地
          1)不在国内居住的人  +  有关身份关系
          2）下落不明 || 宣告失踪  +  有关身份关系
          3）一方被采取强制性教育措施
          4）一方被监禁
          5）一方被注销户籍
          6）追索 赡养费、抚育费、扶养费 + 几个被告  + 住所地不在同一辖区的；
               ie.必要共同诉讼
          7）一方离开住所地超过1years，另一方起诉离婚

### 3.1.3 被告所在地
          1）双方被采取强制性教育措施
          2）双方被监禁
          3）双方被注销户籍
          4）双方离开住所地超过1years，另一方起诉离婚
          5）else 的普通情形

## 3.2 合同关系
### 1st 不动产纠纷专属管辖
        不动产所在地
        1）不动产所在地
           if 已经登记；
                 登记簿记载的所在地
              else if 未登记；
                     不动产实际所在地
        2）class
         2.1）物权类
              不动产的确权、分割、相邻关系 引起的无权纠纷
         2.2）债权纠纷
              房屋租赁、政策性房屋买卖（安置房）、建设工程施工合同、农村土地承包经营合同
              ！商品房买卖（按照合同法一般原则：被告住所地||合同履行地）

### 2nd 书面协议管辖
        1）可约定权益
           incl：财产权益纠纷
                 因 同居||解除婚姻、收养关系后 发生的财产争议
        2）only 书面
        3）only 就第一审的案件 + 地域管辖 进行约定
           不得违反 级别 && 专属管辖
        4）可以选择的法院：沾边就管
        5）incl 涉外
           涉外合同||涉外财产权益纠纷 + 书面协议 + 有实际联系的法院管辖
        6）if 约定 >= 2个 法院；
               P 有选择权

### 3rd 合同管辖一般规则
        被告住所地||合同履行地

#### 3.1）以是否履行确定合同履行地
          if 履行；
                  被告住所地||合同履行地
                if 履行 + 不在约定的履行地履行；
                       被告住所地||约定的履行地
           else if 没履行 + 双方住所地都不在约定的履行地；
                       被告住所地
                          cuz:约定地点与案子毫无关系

#### 3.2）对合同履行地点没有约定||约定不明
          if 争议标的为给付货币；
                接收货币一方所在地
          if 交付不动产；
                不动产所在地
          if 其他；
                履行义务一方所在地
          if 即时结清；
                交易行为地
          if 信息网络方式订立合同；
               if 通过网络交付标的物；
                     买受人住所地
               if 其他方式；
                     收货地

# 4.裁定管辖
## 4.1 移送管辖（同级法院之间）
### 4.1.1 条件
          1)已经受理
            if 在受理之前发现没有管辖权；
                 裁定不予受理
          2）受理的法院 没有管辖权 || 其他法院已经先立案了
          3）法院 主观认为 有必要
             不要求一定送对了

### 4.1.2 exp
          1）if 受移送的法院也没有管辖权；
                     不得再次移送 + 报上级法院指定管辖
                cuz 只能移送一次
          2）if A法院 先立案  + AB法院 都有管辖权；
                2.1）A 不得将案件移送给 B
                2.2）B 发现 A 已经立案；不得重复立案
                2.3）B 立案后发现 A 已经立案的；裁定将案件移送给 A

## 4.2 管辖权的移转（上下级法院之间）
       % 能上能下，下要批准 %
       if 确有必要交下级法；
             应当报请其上级法批准
       like:中院认为需要交给基院审理的；应当报请高院批准

## 4.3 指定管辖
### 4.3.1 情形
          1）受移送法院认为自己无管辖权
          2）管辖不宜
                ie.需要回避
          3）管辖权争议 + 协商未能解决

### 4.3.2 程序
          if 管辖权争议；
                协商解决；
                   if 协商不成；
                      报共同上级法指定管辖
                      下级法 应当中止审理
                         if 下级法 在指定管辖做出前抢先下判决裁定的；
                               上级法裁定指定管辖的同时 + 一并撤销下级法的判决、裁定 

# 5.管辖权异议
## 5.1 条件
       1）本诉的当事人
         ie.only D && P
         ！incl:有独三、无独三
       2）only 向第一审法院 提出
          !incl:发回重审后按照一审审理的法院
                   if 此时提；法院不予审查
       3）可以对级别、地域管辖 异议，也可以同时异议
       4）在答辩期间提出
       5）必须书面 
       6）国内：收到起诉状副本 15days 内 提出
          涉外：收到起诉状副本 30days 内 提出

## 5.2 程序
### 5.2.1 法院处理
       if 异议成立；
             裁定将案件 移送有管辖权的法院
          esle if ；
                裁定驳回异议
       if 当事人不服；
           10days 内向上一级法上诉
       if 案件移送 + 当事人未上诉 + 受移送法院认为有错误；
           可以依职权裁定撤销

### 5.2.2 应诉管辖
       if 当事人未提管辖异议 + 应诉答辩（Incl就案件的实体内容进行答辩、陈述、反诉）；
             视为法院有管辖权
                exp:违反专属、级别管辖

### 5.2.3 exp
         1）对小额诉讼 管辖的裁定一经做出，立即生效
         2）if 请求权竞合的案件 + 当事人选择 A 法院 + 在举证期限届满之前 又 选择 B 法院；
                  if 经审查异议成立；
                        应当裁定驳回起诉
            ie.先依照侵权选择法院，又依照违约选择法院
         3）if 同时对级别、地域都提出了管辖权异议；
                 应当一并做出裁定
         4）if 当事人未提管辖异议 + 法院自己发现没有管辖权；
                 应当移送
         5）if 在管辖权异议裁定做出前 + 原告撤诉 + 法院准予；
