# I.概念
    1.民事诉讼法律关系
      ie.必须有一方是法院
         like: 法院 && 原告、法院 && 被告
    2.所有诉讼法 都是 公法
    3.在中国领域内的 任何人 都适用 我国民诉

# II.基本原则 
## 1.平等、同等、对等 原则
     1.1 平等
         1）P && D 权利相同
         2）P && D 权利对应
              like:起诉状-答辩状
                   本诉-反诉
     1.2 同等
         ie.一视同仁
            外国人在我国进行民诉时，与中国人享有同等的诉讼权利义务
          $:外国人 == 我国当事人
     1.3 对等
         ie.以牙还牙
            外国法对我国人进行限制，我国法也限制该国当事人

## 2.调解原则
     C_ch_3

## 3.辩论原则
     3.1 贯穿 诉讼程序的 全过程
         1）非诉、执行中 无
         2）不仅仅 局限于 法庭上（P起诉 D答辩）
     3.2 也可使用书面形式
            ie.庭前提交起诉状、答辩状
     3.3 可以辩论的范围 incl:
         事实、法律、程序、实体
         法律适用问题当事人可以辩论，决定权在法官
     3.4 主体 
         incl 第三人
              ie.广义的本案当事人
                 But 不是本诉当事人
         ！incl:证人

## 4.处分原则
    4.1 适用全过程
        incl：执行和解
    4.2 一般授权 && 特别授权的人 都有处分权
        like:委托代理人
             if 委托||不委托；
                       处分原则
             if 当事人、TP 都可以委托；
                       平等原则

## 5.支持起诉

## 6.诚实信用
### 6.1 保证书制度
#### 6.1.1 证人
           usu:法院在证人出庭作证前 应当告知 + 责令签署
           exp:无限人
               if 拒签；
                    不得作证+自行承担相关费用

#### 6.1.2 询问当事人
           usu:法院在询问当事人之前 可以要求 签署
               if 拒签 + 待证事实欠缺其他证据证明；
                    对其主张的事实不予认定

### 6.2 对恶意诉讼的强制措施
#### 6.2.1 诉讼

                 驳回诉求 + depends 情节罚款拘留

#### 6.2.2 执行
           if 当事人恶意串通+ 通过诉讼调解等方式逃避履行义务的；
                 depends 情节罚款拘留

#### 6.2.3 证据
           if 故意毁灭书证（把借条烧掉）；
                     罚款拘留
           else if 无正当理由拒不提交；
                  不需要

# III.基本制度
## 1.合议庭组成
     1）陪审员
        一审普通可以有
     2）简易一审独任
     3）特别程序 
        usu 一审独任
             incl:督促/支付令程序、公示催告的公告审理阶段
        exp:必须合议 + 审判员组成
            3.1）选民资格
            3.2）重大疑难
            3.3）担保财产标的额超过基院管辖范围的 担保物权实现 、
            3.4）公示催告的除权判决阶段

## 2.回避
> 回避是申请再审的理由

### 2.1 回避的申请
        1）方式
           1.1）口头 || 书面
           1.2）申请 || 职权
        2）时间
           usu:案件开始审理时
           exp:法庭辩论终结前
                  if 审理过程中 才知道回避事由的

### 2.2 回避的决定
        1）审判人员->院长决定
             incl:陪审员、书记员
        2) 其他人员->审判长决定
             incl:鉴定员、勘验员、翻译、执行人员
        3) 院长->审委会

### related ：仲裁中的回避
        1）仲裁院->仲裁委主任
        2）主任->仲裁委集体

### 2.3 回避的程序
        if 申请回避；
              被申请的人要暂停执行职务
                 exp:需要采取紧急措施的
                       ie.保全 || 先予执行
              法院应当在 3days 内 做出决定（文书是决定）
                 if 决定回避 + 更换人员；
                       诉讼程序 继续进行 + 先前行为有效
                       notes:无需重组

                 if 不服；
                    向原法院复议；
                       复议期间，被申请回避的人不停止执行职务
        
        if 二审发现一审未回避（程序错）；
               裁定撤销原判 + 发回重审

### related：仲裁回避，效力待定
        1）当事人可以请求 重新仲裁
        2）仲裁庭决定
          2.1）自己决定
          2.2）根据当事人决定

### 2.4 回避的理由
    1）股东||持有股权
       if 上市公司；
             要看是否有其他回避的理由
       else if 非上市公司；
             应当回避

## 3.公开审判
> 诉讼程序中无书面审理的说法

### 3.1 一审必开庭
        1）绝对不公开
          1.1）国秘、个人隐私
          1.2）合议庭评议
        2） 依申请可以不公开
          2.1）商业秘密
          2.2）离婚
        3） 一律公开
           案件宣判

#### 3.1.1 简易程序
           1）审理
              必须开庭
              当事人可以选择采用视听传输技术等方式开庭
           2）宣判
              usu 当庭宣判
              exp 法院认为不宜当庭宣判的
              $:判决结案的，必须公开宣判

### 3.2 二审
        usu:开庭审理
        exp:1）针对裁定上诉
              incl:不予受理、驳回起诉、管辖权异议
            2）驳回上诉
               ie.上诉请求明显不成立
            3）应当改判
               ie.适用法律错误
            4）应当发回
               ie.程序错

### 3.3 调解
     1）审判方式
        usu:过程不公开
        exp:当事人同意公开
     2）调解方式
        usu：双方 同时在场
        exp: 分别 调解
     3）文书
        usu:调解书内容不公开
        exp:国社、公共利益 + 法院认为有必要
            公益诉讼调解内容要公告
               C_ch_1

### 3.4 实现担保物权
        usu: 审判员独任 + 书面审查 + 不开庭
              if 对材料的合法性 审查；
                    可以询问 被申请人、申请人、利害关系人
                    可以依职权 调查事实
        exp: 担保财产超过基法管辖范围
             应当组成合议庭 + 书面审查 + 不开庭

### 3.5 确认调解协议
        审判员独任 + 书面审查 + 不开庭
           if 对材料的合法性 审查；
                    应当通知 双方共同到场 核实
                    必要时 向调解组织 核实有关情况

### 3.5 债务人对支付令的异议
        法院书面、形式审查

### 3.6 公示催告
> 公示催告期间，转让票据权利的行为无效

#### 3.6.1 公示催告阶段
        审判员 + 独任 + 书面审查（审查票面是否一致）
        if 利害关系人申报；
               法院应当通知 利害关系人出示票据 + 申请人在指定期间查看票据
            if 符合条件；
                    裁定终结催告程序
                        cuz 出现争议
#### 3.6.2 除权判决阶段
        审判员 + 合议庭

### 3.7 仲裁
        1）usu 开庭审理
        2) usu 不公开审理
           exp 涉及国秘 

### related :公开质证
     1.usu:未经公开质证，不得作为定案依据
       exp:法院依职权调取的证据
     2.不公开质证的
       1）国密、个人隐私
       2）商业秘密
       3）if 庭前当事人认可 + 审判人员在庭审中说明 ;
                视为质证过

### related：文书公开制度
     usu.公众可以查阅 + 生效的+ 裁、判
     exp.三秘

## 4.一审终审，不能上诉
     1）最高院一审
     2）调解书
     3）非争议案件
        incl:特别程序、督促、公示催告、企业法人破产还债程序
     4）小额
        对小额提管辖异议的，法院的裁定一经做出，立即生效
     5）确认婚姻效力
        diff && 离婚是变更之诉，是两审终审

# IV.诉
## 1.class
     1.1 确认之诉
         depends 原告的诉讼请求
         ie.双方之间存在||不存在某种法律关系
               case_1)甲起诉要求确认自己对乙不构成侵权 T
                    2)甲请求确认妻子乙是无人 F
                         cuz 是特别程序，不属于诉

     1.2 变更之诉/形成之诉
         改变||消灭 既存的 法律关系
            like:离婚
                 甲起诉前妻，要求将女儿判归前妻抚养
                 甲起诉撤销合同
     
     1.3 给付之诉
         1）诉讼标的物存在与给付之诉中
         2）停止侵害、排除妨害是给付内容为不作为的给付之诉
               like:甲起诉乙要求拆除影响甲家房屋通风的临时建筑
                    甲起诉乙要求把车修好，其后又变更诉讼请求，要求乙赔偿损失+赔礼道歉

## 2.反诉
### 2.1 diff 反诉 && 反驳
        反诉 == 承认本诉 + 提出独立的反请求
        like:甲诉乙支付租金
              1）租赁合同无效；反诉
              2）已经支付了租金：反驳
              3）超过诉讼时效：抗辩

### 2.2 反诉的条件
        1）反诉的当事人 
           ie.是本诉的原被告
        2）反诉 && 本诉 之间有牵连关系
           2.1)同一法律关系
           2.2)同一事实
           2.3)具有因果关系
        3）时间
           usu:本诉举证期限届满前（quote 邱振启 在法辩终结前提反诉）
           exp:if 本诉法庭辩论结束前 + 可以合并审理；
                        应当合并审理
               if 在判决后 + 另行起诉提出反诉；
                        裁定不予受理
        
        4）对反诉有管辖权 + 适用同一诉讼程序
              cuz 一般合并审理

### 2.3 反诉的提出
        if 发回重审 + D 反诉；
                      一并受理

# 3.地域
     自然人：经常居住地 > 户籍所在地
     法人：主要办事机构所在地 > 登记地、注册地

## 3.1 人身关系
### 3.1.1 两地管
          1)if 监护案件（不服指定监护||变更监护关系）；
                被监护人住所地 || 被告住所地
          2）追索 赡养费、抚育费、扶养费 + 几个被告  + 住所地不在同一辖区的；
               ie.必要共同诉讼
          3)离婚
            一方离开住所地 > 1years，另一方起诉离婚
            3.1）没离开的一方仍然在 原住所地
            3.2）没离开的人起诉

### 3.1.2 原告所在地
          1) 在中国境内找不到 +  有关身份关系
             1.1）不在国内 居住  
             1.2）下落不明 || 宣告失踪  
                  ！incl:宣告死亡
             1.3) incl:婚姻、收养、监护、抚养、继承
                  
             like:A区甲 && B区乙 签订货物买卖合同 + 约定C 区履行 + 尚未履行甲下落不明；
                  analyze:不是有关身份关系的诉讼，仍然合同规则->
                          尚未履行 + 双方都不在履行地 ->
                          被告住所地法 管辖

          2）一方被关起来 + 外面的人 诉 被关的人
             2.1）一方被采取强制性教育措施
             2.2）一方被监禁
             2.3）被关起来 >= 1years;
                                 被告被监禁地 || 被采取强制性教育措施地 
             2.4）if 被关的人 诉 外面的人；
                                 被告住所地
                  if 双方都被关起来；
                                 两地管
          3）一方被注销户籍
             if 户籍签出尚未落户；
                    经常居住地->原户籍所在地
  
### 3.1.3 被告所在地
          1）双方被采取强制性教育措施
          2）双方被监禁
          3）双方被注销户籍
          4）离婚案件
             if 双方都离开  > 1years  + 另一方起诉离婚；
                          被告经常居住地->原告起诉时被告居住地
                              ie.看 D 在什么地方
          5）else 的普通情形

## 3.2 合同关系
### 1st 不动产纠纷专属管辖
        不动产所在地
        1）不动产所在地
           if 已经登记；
                 登记簿记载的所在地
              else if 未登记；
                     不动产实际所在地
        2）class
         2.1）物权类
              不动产的确权、分割、相邻关系 引起的无权纠纷
         2.2）债权纠纷
              房屋租赁、政策性房屋买卖（安置房）、建设工程施工合同、农村土地承包经营合同
              ！商品房买卖（按照合同法一般原则：被告住所地||合同履行地）
                like:甲乙订立房屋买卖合同
                        适用的是一般合同原则：合同履行地 || 被告住所地

### 2nd 书面协议管辖
        1）only refer 财产权益纠纷
             like: 因 同居||解除婚姻、收养关系后 发生的财产争议
        2）only 书面
        3）only 就第一审的案件 + 地域管辖 进行约定
           不得违反 级别 && 专属管辖
        4）可以选择的法院：沾边就管
        5）incl 涉外
           涉外合同||涉外财产权益纠纷 + 书面协议 + 有实际联系的法院管辖
        6）if 约定 >= 2个 法院；
               P 有选择权
           but 必须是 起诉时 能够确定管辖法院

        like:A区甲公司 && B区乙公司 签订购货合同，约定履行发生争议，由 P 所在地||合同签订地管辖；
             不属于专属管辖，按照当事人的约定确定管辖法院

### 3rd 特殊案件管辖
#### 3.1）保险合同
         被告住所地 || 保险标的物 所在地
         1）财产保险
            被告住所地、运输工具登记注册地、运输目的地、保险事故发生地
            ！incl:运输始发地
         2）人身保险
            被告住所地 、被保险人 住所地 

#### 3.2）担保合同
          depends 主合同

#### 3.3）运输
          1）% 合同看两头 %
             运输始发地、目的地、被告住所地
          2）% 事故到现场 %

# 4.裁定管辖
## 4.1 移送管辖（纠正管辖错误，也可纠正违反级别管辖的规定）
> usu同级法院之间,exp上下级之间（A区法受理专利案件后，移送B市中院）

### 4.1.1 条件
          1)已经受理了
            if 在受理前 发现没有管辖权；
                 裁定不予受理
            else if 在受理后 发现没有管辖权；
                      应当移送
          2）受理的法院 没有 管辖权 || 其他法院已经先立案了
          3）法院 主观认为 有必要
             不要求一定送对了

### 4.1.2 exp
          1）if 受移送的法院也没有管辖权；
                     不得再次移送 + 报上级法院指定管辖
                cuz 只能移送1次
          2）if A法院 先立案  + AB法院 都有管辖权；
                2.1）A 不得将案件移送给 B
                2.2）B 发现 A 已经立案；不得重复立案
                2.3）B 立案后发现 A 已经立案的；裁定将案件移送给 A

### related：执行管辖的冲突
         if 在立案前 其他法已经立案；
                 不得重复立案
            else if 在立案后发现 ；
                      应当撤销案件
         if 两个法院冲突；
                最先立案

## 4.2 管辖权的移转（上下级法院之间）
       % 能上能下，下要批准 %
       if 确有必要交下级法；
             应当报请其上级法批准
       like:中院认为需要交给基院审理的；应当报请高院批准

## 4.3 指定管辖
### 4.3.1 情形
          1）受移送法院认为自己无管辖权
                cuz 不能退还原法院
          2）管辖不宜
                ie.法院本身就是当事人，需要回避
          3）管辖权争议 + 协商未能解决

### 4.3.2 程序
          if 管辖权争议；
                协商解决；
                   if 协商不成；
                      层报 共同 上级法指定管辖
                      下级法 应当中止审理
                         if 下级法 在指定管辖做出前抢先下判决裁定的；
                               上级法裁定指定管辖的同时 + 一并撤销下级法的判决、裁定 
          if 没有产生管辖权争议；
                 报请自己的 上级法 指定

          like:甲市A区法 && 甲市B区法 发生管辖权争议；
                         if 协商不成；
                               报请该省高院指定管辖（共同的上级）
