# I.基本理念
# 1.trifle
     1)自诉可以附民，附民是民诉
     2）检院的监督是全程、书面、事后、以检院的名义
     3）检院的公益诉讼只能在刑附民中，在追究犯罪的同时代表国家||集体的时候提
     4）简易程序体现在公正前提下追求效率

## 2.diff 
### 2.1 独立价值/程序正义
        1）本身体现着 秩序、公正、效益
              like:审判公开、辩护制度
        2）创制
              like:解说、争辩、论证、评议、表决
        3）阻却
              like:不告不理、和解后量刑从轻、上诉不加刑、排除口供后检院证据不足不诉

### 2.2 惩罚犯罪 && 保障人权
        1）最重要、罪核心 保护 嫌疑人、被告人
           ie.也保护其他人
        2）usu 同等重要
           exp 发生冲突，侧重 保障人权
               like:非法证据排除

### 2.3 程序公正 && 实体公正
        1）实体公正
          1.1）犯罪事实的证据确实、充分
          1.2）准确适用刑法
          1.3）罪行相适应，适度判定刑罚
          1.4）冤假错案 采取救济方法 及时纠正、补偿
        2) usu 同等重要
           exp 发生冲突，程序优先
               like:二审发现一审程序错，应当发回

### 2.4 诉讼效益 && 司法公正
        1）效益
           ie.减少 人力、物力、财力 的投入
               like: 1.1）尽量 取保
                     1.2）集中审理原则
                          ie.减少案件拖延、积压
                     1.3）严格限定羁押期间
                     1.4）简易程序、速裁程序
        2）效率 常常是 公正的保障
           追求效率 以公正为前提
        3）公正优先、兼顾效率
