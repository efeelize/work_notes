# I.概念
    1.民事诉讼法律关系
      ie.必须有一方是法院
         like: 法院 && 原告、法院 && 被告
    2.所有诉讼法 都是 公法
    3.在中国领域内的 任何人 都适用 我国民诉

# II.基本原则 
## 1.平等、同等、对等 原则
     1.1 平等
         1）P && D 权利相同
         2）P && D 权利对应
              like:起诉状-答辩状
                   本诉-反诉
     1.2 同等
         ie.一视同仁
            外国人 == 我国当事人
     1.3 对等
         ie.以牙还牙
            外国法对我国人进行限制，我国法也限制该国当事人

## 2.调解原则
     C_ch_3

## 3.辩论原则
     3.1 贯穿全过程
     3.2 也可使用书面形式
            ie.庭前提交起诉状、答辩状
     3.3 可以辩论的范围 incl:
         事实、法律、程序、实体
     3.4 主体 incl:
         第三人是广义的本案当事人
            But 不是本诉当事人

## 4.处分原则
    4.1 适用全过程
        incl：执行和解
    4.2 一般授权 && 特别授权的人 都有处分权

## 5.支持起诉

## 6.诚实信用
### 6.1 保证书制度
#### 6.1.1 证人
           usu:法院在证人出庭作证前 应当告知 + 责令签署
           exp:无限人

                    不得作证+自行承担相关费用

### 6.2 对恶意诉讼的强制措施

#### 6.2.2 执行
           if 当事人恶意串通+ 通过诉讼调解等方式逃避履行义务的；

# III.基本制度
## 1.合议庭组成
     1）人民陪审员一审普通可以有
     2）简易一审独任
     3）特别程序 
        usu 一审独任
             incl:督促/支付令程序、公示催告的公告审理阶段
        exp:必须合议 + 审判员组成
            选民资格、重大疑难、担保财产标的额超过基院管辖范围的 担保物权实现 、
            公示催告的除权判决阶段

## 2.回避
### 2.1 回避的申请
        1）方式
           1.1）口头 || 书面
           1.2）申请 || 职权
        2）时间
           usu:案件开始审理时
           exp:法庭辩论终结前
                  if 开始审理后才知道回避事由的

### 2.2 回避的决定
        1）审判人员->院长决定
             incl:陪审员、书记员
        2) 其他人员->审判长决定
             incl:鉴定员、勘验员、翻译
        3) 院长->审委会

### related ：仲裁中的回避
        1）仲裁院->仲裁委主任
        2）主任->仲裁委集体

### 2.3 回避的程序
        if 申请回避；
              被申请的人要暂停执行职务
                 exp:需要采取紧急措施的
                       ie.保全 || 先予执行
              法院应当在 3days 内 做出决定（文书是决定）
                 if 决定回避 + 更换人员；
                       诉讼程序继续进行 + 先前行为有效
                 if 不服；
                    向原法院复议；
                       复议期间，被申请回避的人不停止执行职务
        
        if 二审发现一审未回避（程序错）；
               裁定撤销原判 + 发回重审

### related：仲裁回避，效力待定
        1）当事人可以请求 重新仲裁
        2）仲裁庭决定
          2.1）自己决定
          2.2）根据当事人决定

## 3.公开审判
### 3.1 一审必开庭
        1）绝对不公开
          1.1）国秘、个人隐私
          1.2）合议庭评议
        2） 依申请可以不公开
          2.1）商业秘密
          2.2）离婚
        3） 一律公开
           案件宣判

### 3.2 二审
        usu:开庭审理
        exp:1）针对裁定上诉
              incl:不予受理、驳回起诉、管辖权异议
            2）驳回上诉
               ie.上诉请求明显不成立
            3）应当改判
               ie.适用法律错误
            4）应当发回
               ie.程序错

### 3.3 调解
     1）审判方式
        usu:过程不公开
        exp:当事人同意公开
     2）调解方式
        usu：双方 同时在场
        exp: 分别 调解
     3）文书
        usu:调解书内容不公开
        exp:国社、公共利益 + 法院认为有必要
            公益诉讼调解内容要公告
               C_ch_1

### 3.4 实现担保物权
        usu: 审判员独任 + 书面审查 + 不开庭
              if 对材料的合法性 审查；
                    可以询问 被申请人、申请人、利害关系人
                    可以依职权 调查事实
        exp: 担保财产超过基法管辖范围
             应当组成合议庭 + 书面审查 + 不开庭

### 3.5 确认调解协议
        审判员独任 + 书面审查 + 不开庭
           if 对材料的合法性 审查；
                    应当通知 双方共同到场 核实
                    必要时 向调解组织 核实有关情况

### 3.5 债务人对支付令的异议
        法院书面、形式审查

### 3.6 公示催告
#### 3.6.1 公示催告阶段
        审判员 + 独任 + 书面审查（审查票面是否一致）
        if 利害关系人申报；
               法院应当通知 利害关系人出示票据 + 申请人在指定期间查看票据
            if 符合条件；
                    裁定终结催告程序
                        cuz 出现争议
#### 3.6.2 除权判决阶段
        审判员 + 合议庭

### 3.7 仲裁
        1）usu 开庭审理
        2) usu 不公开审理
           exp 涉及国秘 

### related :公开质证
     1.usu:未经公开质证，不得作为定案依据
       exp:法院依职权调取的证据
     2.不公开质证的
       1）国密、个人隐私
       2）商业秘密
       3）if 庭前当事人认可 + 审判人员在庭审中说明 ;
                视为质证过

### related：文书公开制度
     usu.公众可以查阅 + 生效的+ 裁、判
     exp.三秘

## 4.一审终审，不能上诉
     1）最高院一审
     2）调解书
     3）非争议案件
        incl:特别程序、督促、公示催告、企业法人破产还债程序
     4）小额
        对小额提管辖异议的，法院的裁定一经做出，立即生效
     5）确认婚姻效力

# 4.裁定管辖
## 4.1 移送管辖（同级法院之间）
### 4.1.1 条件
          1)if 在受理前 发现没有管辖权；
                 裁定不予受理
            else if 在受理后 发现没有管辖权；
                      应当移送
          2）受理的法院 没有管辖权 || 其他法院已经先立案了
          3）法院 主观认为 有必要
             不要求一定送对了

### 4.1.2 exp
          1）if 受移送的法院也没有管辖权；
                     不得再次移送 + 报上级法院指定管辖
                cuz 只能移送一次
          2）if A法院 先立案  + AB法院 都有管辖权；
                2.1）A 不得将案件移送给 B
                2.2）B 发现 A 已经立案；不得重复立案
                2.3）B 立案后发现 A 已经立案的；裁定将案件移送给 A

### related：执行管辖的冲突
         if 在立案前 其他法已经立案；
                 不得重复立案
            else if 在立案后发现 ；
                      应当撤销案件
         if 两个法院冲突；
                最先立案

## 4.2 管辖权的移转（上下级法院之间）
       % 能上能下，下要批准 %
       if 确有必要交下级法；
             应当报请其上级法批准
       like:中院认为需要交给基院审理的；应当报请高院批准

## 4.3 指定管辖
### 4.3.1 情形
          1）受移送法院认为自己无管辖权
          2）管辖不宜
                ie.需要回避
          3）管辖权争议 + 协商未能解决

### 4.3.2 程序
          if 管辖权争议；
                协商解决；
                   if 协商不成；
                      报共同上级法指定管辖
                      下级法 应当中止审理
                         if 下级法 在指定管辖做出前抢先下判决裁定的；
                               上级法裁定指定管辖的同时 + 一并撤销下级法的判决、裁定
