# ch_5 非诉程序 && 执行

# I.特别程序
    1）非争议都在基院
    2）审判员一人独任
       exp:2.1）选民资格
           2.2）重大疑难
           2.3）担保财产标的额超过 基院管辖范围的 担保物权实现
    3）有错：找原法院：做新判，撤原判

## 1.选民资格
    1）选委会先行处理/申诉处理前置
             cuz 宪法权利
    2）任何人都可以起诉
    3）参加（3个）：起诉人、选委会、选民
       送达（2个）：起诉人、选委会
       通知（1个）：选民

## 2.认定财产无主
    if 公告内 有人对财产提出请求；
           应当裁定终结特别程序 + 告知另诉普通诉讼程序
    if 公告期满 1years；
          判决收归国家||集体

## 3.督促程序
### 1.条件
    1） only 金钱、有价证券
        ！incl:劳务
    2）已到清偿期 + 数额确定
        even if 预期违约，也不能适用督促
    3）没有对待给付义务、其他纠纷
        like:货物的瑕疵担保义务
    4）能够送达
        ie.对不在我国领域内、在我国领域内&&下落不明 不得督促程序
        cuz:支付令可以留置，但是不能公告送达
    5）没有申请过诉前保全
        cuz else if 有走诉讼的意思         

### 2 支付令的效力：督促力（限期履行）+执行力（强制履行）
#### 1）if 债务人 在收到支付令之日起15days 内；
              1）清偿 ：裁定终结督促程序   
              2）+ 向原法院 + 书面 异议；
                               法院书面、形式审查
        
                 if 债务人向 其他法院 提出异议||超期提异议；
                                             视为未提出异议
                 if 债务人 异议内容不是针对债权债务本身（没欠钱、未到清偿期）；
                                             构成异议
        
        else if 期满债务人什么都不做；
                         债权人 向法院申请 强执

#### 2）if 对 设有担保的主债务人发出支付令；
                          对担保人没有约束力
#### 3）if 就担保关系单独提起诉讼；
                          支付令自法院受理案件之日期失效
#### 4）if 债权人向法院起诉；无论哪一个法院，支付令都失效


### 3. 支付令失效后，直接转入诉讼
      if 异议成立；
            终结督促程序 + 支付令失效 + 直接转入诉讼程序
                if  申请支付令一方 不同意 转入诉讼（未明确表示反对 即视为同意）；
                           自收到终结督促程序的裁定之日起 7days 内 向原法院提出
                if 向其他法院起诉；
                           视为向原法院起诉 

# II.执行
# 1.class
## 1.1 usu 申请执行
       1）only 针对财产、行为
       2）所有主体 + 2years 内 申请 + 可以中止、中断 （！incl:延长）
           usu:履行期的最后一日起算
               if 分期；
                    每次履行期的最后一日
               If 未规定履行期；
                    文书生效之日
           exp:if 超过2years 申请；
                        不审查 +  应予受理
                    if 被执行人 提异议 + 法院审查成立；
                                          裁定不予执行
                    if 被执行人 没提 + 履行||部分履行后 + 要求执行回转；
                                         不予支持

## 1.2 exp 移送执行（法院内部直接给执行庭）
        1）四费一金劳动报酬
        2）民事制裁决定书
           ie.罚款拘留
        3）刑附民
        4）公益诉讼案件 + D 不履行
        5）执行回转
           usu：if  执行完毕后 + 原来文书有错被撤销 + 权利人拒绝返还已经执行的财产;
                              法院根据 新的文书 重新采取措施 + 恢复到 执行开始前的状
           exp：IP 案件；
                  商标被撤销、专利宣告无效 ,没有新的文书，不存在执行回转的问题
                      cuz 没有出现新的生效文书，无执行根据

# 2.法院的委托执行
       if 受托法 接到委托函件；
            15days 内开始执行 + 不得拒绝
                else if 不执行；
                         委托法 请求 受托法的上级法 指令受托法 执行
            if 当事人、利害关系人、案外人 对原法院的执行行为 提出异议；
                         受托法审查

# 3.执行行为异议（***）
> 执行行为异议审查 && 复议期间，不停止执行

    if 当事人、利害关系人 + 执行错 ；
            法院收到异议之日起 15days 内 审查；
               If 理由成立；
                     裁定撤销||改正
               else if ;
                     裁定驳回
    
    if 当事人、利害关系人 不服；
             自裁定送达之日起 10days内 向上一级法 申请 + 书面复议
                    上一级法 合议庭 + 30days 内审查完毕 + 院长批准可延长

# 4.案外人执行（标的）异议 && 案外人申请再审 （***）
    Notes:1)不得对标的物进行处分
             1.1）审查期间、
             1.2）驳回异议裁定送达案外人之日起15days 内（救济期）
          2)无论是案外人||申请执行人 提出 执行异议之诉，都是案外人承担举证证明责任
          3）if 异议成立；裁定中止
             else if;裁定驳回，执行继续
          4）异议之诉都按照一审普通程序 + 执行法院管辖

## 4.1 案外人申请
      if 是必共人；
           if 按照一审；追加
           else if;调解不成，撤销原判，发回重审
           $:追加是一审法的事情
      else if;
            only 审理原裁判调对其民事权益造成损害的内容

## 4.2 程序（不能自由选择）
    depends 原判标的物 && 执行标的物 是否同一；
            if 同一；原判错了->再审
            else if;执行错了->执行异议（原判有错误令案救济）

     case_法院判决甲将汽车交付给乙
          1）if 甲申请执行 +  案外人丙异议，认为车是自己的
                  （都是汽车）-> 法院自收到 异议之日 起15days内审查
                  if 审查 + 认为成立；
                             裁定中止执行 + 甲申请再审
                     else if 认为不成立；
                             裁定驳回  + 丙申请再审

          2）if 汽车无法执行，甲申请执行房屋 +  案外人丙异议，认为房子是自己的
                  （标的不一样）-> if 审查 + 认为成立；
                             裁定中止执行 + 甲申请执行异议
                      else if 认为不成立；
                             裁定驳回  + 丙申请执行异议
