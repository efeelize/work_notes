# II.调解
##  1.diff 和解 && 调解
     1）和解
        双方当事人 + 自主的 + 达成的协议
           cuz 协议就是合同，合同可以书面||口头
                 like:执行和解可以口头
     2) 调解
        三方 + 自愿 + 达成协议
     3）执行力
        3.1）if 给付内容 + 生效调解书;
                 有强制执行力
                if 调解协议 diff && 调解书；
                       裁定补正
        3.2）达成和解协议的，可以请求法院 确认和解协议 制作调解书

## 2.一审
### 2.1 genernal principle
        1.先行调解原则
          usu:适宜调的，都要先调
          exp:当事人拒绝
        2.调解不区分审理阶段
             like:检院抗诉案件也可以调解
                     ie.再审
        3.审判权 + 争议 + 私权处分
          3.1）审判权
               执行阶段不调解
          3.2）争议
               非诉案件不调解
                  cuz 此阶段无辩论
          3.3）私权处分
               身份关系 确认 不调解
                   解除身份关系可以调节
                      like:离婚
        4.调解书、调解协议 一裁终局 (C_ch_1)
          exp:检院抗诉
              notes:有限抗诉
                    ie. only 只在损害国家 && 社会利益的前提下
                            like:公益诉讼
        5.自愿
          5.1）启动自愿
               双方都需要同意
          5.2）接受自愿
               1）调解书签收生效
               2）签收之前都可以反悔
       6.法院
         1）主持调节
         2）可以委托第三方进行调节

### 2.2 不能调解
           1）非争议
              Incl:特别、督促、公示催告、破产
           2）执行
              but 执行中可以和解
           3）身份关系的确认
              身份关系的解除可以调解
                 like:离婚

### 2.3 调解协议的内容
           1）可以超出诉讼请求
           2）约定不履行 调解协议 的民事责任
              if 约定了 + 一方不履行；
                   if 申请法院判；
                         法院不予准许
                   only 只能依调解书 申请法院执行
           3）约定担保
              法院应当准许
              but 担保人不签，不影响调解书的效力
                     cuz 之前有担保合同存在，条件成就时直接强执
           4）if 就部分内容达成调解；
                     就这部分 先行确认 + 制作调解书
           5）if 调解书&&调解协议 diff;
                    以调解协议为准，裁定补正

### 2.4 结案方式
#### 1）usu 调解书
        1.1）审判人员、书记员签名、盖法院章
        1.2）if 签收之前反悔；
                 及时裁判
        1.3 生效
            usu:签收生效
            exp:if 不需要制作调解书；
                      审判人员、书记员 在调解协议上签名 || 盖法院章 后 生效 
        1.4）必须制作调解书
             if 一审 判决解除离婚、收养关系 + 二审调解和好、维持收养关系；
                   必须制作调解书
                       cuz:二审要制约一审裁判，否则一审裁判就没有法律文件否定

#### 2）exp 1：一审可以不制作调解书，二审必须有调解书
       2.1）调解和好的离婚案件
       2.2）维持收养关系
       2.3）即时履行
            ie.一次性+当场履行

#### 3）exp 2：制作判决书
      3.1）涉外
           应当制作调解书 -> 当事人要求制作判决书的，可以依据协议的内容制作判决书送达当事人
      3.2）无人的离婚案件

## 3.二审（***）
### 3.1 调不成发回（法院之过）
        1）漏事
           ie.一审已经提出诉讼请求 + 原审法未做处理
        2）漏人
           ie.必共人、有独三
        3）if 一审判不准离婚 + 二审认为应当离 ;
               if 当事人自愿；
                     可以与子女抚养、财产问题 一并调解；
                          调不成发回
               if 双方同意由 二审一并审理的；
                        可以一并裁判

### 3.2 调不成告知另诉（当事人之过）
        1）增加独立诉讼请求
        2）二审提反诉
        if 1）2）双方当事人同意 由 二审法 一并审理；
                        可以一并裁判，立即生效，不能上诉、另诉
                                     cuz 反诉&&本诉 是2个独立的诉

### related：再审中增加独立的诉讼请求
        1）usu
           不调，直接告知另诉
              cuz 不属于再审的审理范围
        2）exp 一并审理
           2.1)涉及国社利益
           2.2)原审中要求增加、变更 + 原审没有加

### 3.3 调解成功
        1）必须制作调解书
              不做调解书only 一审
        2)调解书送达签收后，原审法院判决视为撤销，不需要在调解书上写明

## 4.再审
     1）达成调解协议的 + 应当制作调解书 
     2）签收生效+ 原判决裁定视为被撤销（无需写明撤销原判）
     3）漏人，调不成，发回 原一审
          cuz 民诉中的发回都是发回 原一审（一发到底）
              diff && 刑诉：only 死刑复核可以发一审，也可以发二审
                           else 都是发原审

## 5.人民调解
### 5.1 genernal principle
        1) 调解不是必经阶段
        2） 调解不影响起诉
        3） 没有 严格的 程序要求
        4）if 不履行调解协议；
                    起诉 + 诉调解协

### 5.2）确认调解协议（***）
      双方共同 + 自调解协议生效之日起30days 内 + 书面||口头 + 调解组织所在地 + 基院
             if 法院受理后；
                      书面审查 
                 if 裁定调解有效 + 一方不履行 || 不全部履行；
                         向法院申请执行
                 if 裁定驳回申请；
                         1）通过调解方式
                            1.1）变更原调解协议
                            1.2）达成新的调解协议
                         2）向法院起诉 

## 6.仲裁调解
     1）应当制作调解书
     2）裁决书
        2.1)不能形成多数意见，按照首仲的意见
        2.2）可签可不签，可记可不记
     3）裁决自 作出之日 起生效


# III.和解
## 1.一审和解结案方式
       1）依据和解协议制作调解书
            spec:公益诉讼only 调解书结案（C_ch_1）
                    if 公益诉讼调解、和解 公告期满 + 法院审查不损害国社公共利益；
                                 应当出具调解书
                          if 当事人以达成和解协议 申请撤诉；
                                   不予准许
            exp:1)涉外
                  判决书结案
                2）无人 的离婚
       2）撤回起诉
          可以再次起诉

## 2.二审和解结案方式
      1）依据和解协议制作调解书
      2）因和解而撤诉 + 符合撤诉条件 ；应予准许
            if 撤回 上诉 + 对方不履行；
                        依据一审判决要求强执
            if 撤回 起诉 + 对方不履行；
                        不能再起诉，且 没有生效法律文书可以强执
                                cuz 法院会撤销原判决

## 3.仲裁和解
     1）裁决书
     2）撤回仲裁申请
        if 撤回仲裁申请后反悔；
               根据 原协议 申请仲裁
               重新达成新协议 变更仲裁

## 4.执行和解（无强执力）
     1）可以口头 + 记入笔录 
     2）if 达成 + 履行完毕；
               裁定执行终结 + 也要将和解协议计入笔录
     3）if 欺诈、胁迫 || 部分履行 
        当事人有选择权
        3.1）only 依申请（！依职权）恢复执行（原生效文书，cuz 和解协议无强执力）；
                    法院 按照 原生效文书执行，已经履行的部分扣除
        3.2）根据 和解协议 起诉
             ！原纠纷

## 2 一审撤诉
### 2.1 主体
          P、有独三（相当于 P,only D:管辖权异议）、上诉人及其法代

### 2.2 撤回起诉
        if 在立案受理后，进行宣判前 + 书面||口头 申请撤回起诉；
             usu:准予撤诉

                   2.1）+ D不同意
                        可以 不准撤
                   2.2）公益诉讼法辩终结后 + （C_ch_1）
                        应当 不准撤
        if P 在开庭审理前申请撤诉 + D 不同意；
                            可以裁定 准予撤诉

### related：按照一审再审 +  P 撤回起诉
       if D 同意 + 不损害国社、TP利益 ；
              可以准许 + 一并裁定撤销原裁判调；
                 if P 撤回后重复起诉；
                              不予受理

### 2.3 后果
#### 1）本案诉讼程序终结
#### 2）诉讼时效重新计算

#### 3）撤诉视为未起诉
##### 3.1）usu 还可以再起诉
           1)不予受理、驳回起诉
           2)if 判决生效后 + 发生新情况、新理由 ；
                应当 作为新案件 受理
##### 3.2)exp：不予受理
         1) 二审 + 撤回起诉 
         2）再审 + 按照一审审理 + 撤回起诉
            if P 撤回起诉 + D 同意 + 法院审查 不损害 国社他人利益；
                    可以准许 + 应当一并撤销 裁判调（视为未起诉，没有文书） + 不得再起诉
            if P 再审撤回起诉后 再起诉的；
                              裁定不予受理
         3）if 离婚案件 + P 撤诉||按照撤诉处理 + 没有新情况、新理由 + 6mon 内；
                                      不予受理
                             if D + 有新情况、新理由 +  6mon 后起诉；

                    法院根据案件的具体情况决定是否准许

## 3 二审撤诉
### 3.1 撤回上诉，不能再上诉

                 不能再上诉（上诉权1次用尽），一审判决期满生效
          if 在上诉期满 后 + 二审法审查；
                  usu 应当裁定准予撤诉 + 裁定准许后，一审判决立即生效 + 可强执 一审判决

                      2）双方恶意串通 损害 国社利益

### 3.2 撤回起诉
          if 在二审中P 撤回起诉 + D 同意 + 法院审查 不损害 国社他人利益；
                    可以准许 + 应当一并撤销一审裁判（视为未起诉，没有文书） + 不得再起诉
                       if P 二审撤回起诉后 再起诉的；
                                    裁定不予受理

### 3.3 再审
#### 1）按撤诉处理，可再审
       1）P 不交诉讼费、不来、先走
       2）P 是 无人的法代，不来
       3）有独三 不来、先走
             cuz 相当于 P.撤诉之后可再起诉

#### 2） 撤回再审申请（了解）
          1）自己撤
             申请撤回 + 法院裁定 
               if 裁定准许 + 终结再审程序
                   if 撤回后 又 申请再审的；
                         usu:不予受理
                         exp:自知||应知 6mon 内 + 
                             1）伪证 + 认定事实的 主要证据
                             2）新证 + 足以推翻 原判决裁定
                             3）据以做出原判、裁的法律文书被撤销、变更
                             4）审理人员有不当行为
                                incl:贪污受贿、徇私舞弊、枉法裁判
          2）按照撤回处理：再审申请人、申请抗诉当事人 
             2.1）+ 不来、先走
             2.2）+ 无正当理由不接受 询问
