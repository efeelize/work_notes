# II.调解
##  1.diff 和解 && 调解
     1）和解
        双方当事人 + 自主的 + 达成的协议
           cuz 协议就是合同，合同可以书面||口头
                 like:执行和解可以口头
     2) 调解
        三方 + 自愿 + 达成协议
     3）执行力
        3.1）if 给付内容 + 生效调解书;
                 有强制执行力
                if 调解协议 diff && 调解书；
                       裁定补正
        3.2）达成和解协议的，可以请求法院 确认和解协议 制作调解书

## 2.一审
### 2.1 genernal principle
        1.先行调解原则
          usu:适宜调的，都要先调
          exp:当事人拒绝
        2.调解不区分审理阶段
             like:检院抗诉案件也可以调解
                     ie.再审
        3.审判权 + 争议 + 私权处分
          3.1）审判权
               执行阶段不调解
          3.2）争议
               非诉案件不调解
                  cuz 此阶段无辩论
          3.3）私权处分
               身份关系 确认 不调解
                   解除身份关系可以调节
                      like:离婚
        4.调解书、调解协议 一裁终局 (C_ch_1)
          exp:检院抗诉
              notes:有限抗诉
                    ie. only 只在损害国家 && 社会利益的前提下
                            like:公益诉讼
        5.自愿
          5.1）启动自愿
               双方都需要同意
          5.2）接受自愿
               1）调解书签收生效
               2）签收之前都可以反悔
       6.法院
         1）主持调节
         2）可以委托第三方进行调节

### 2.2 不能调解
           1）非争议
              Incl:特别、督促、公示催告、破产
           2）执行
              but 执行中可以和解
           3）身份关系的确认
              身份关系的解除可以调解
                 like:离婚

### 2.3 调解协议的内容
           1）可以超出诉讼请求
           2）约定不履行 调解协议 的民事责任
              if 约定了 + 一方不履行；
                   if 申请法院判；
                         法院不予准许
                   only 只能依调解书 申请法院执行
           3）约定担保
              法院应当准许
              but 担保人不签，不影响调解书的效力
                     cuz 之前有担保合同存在
           4）if 就部分内容达成调解；
                     就这部分 先行确认 + 制作调解书

### 2.4 结案方式
#### 1）usu 调解书
        1.1）审判人员、书记员签名、盖法院章
        1.2）if 签收之前反悔；
                 及时裁判
        1.3 生效
            usu:签收生效
            exp:if 不需要制作调解书；
                      审判人员、书记员 在调解协议上签名 || 盖法院章 后 生效 
        1.4）必须制作调解书
             if 一审 判决解除离婚、收养关系 + 二审调解和好、维持收养关系；
                   必须制作调解书
                       cuz:二审要制约一审裁判，否则一审裁判就没有法律文件否定

#### 2）exp 一审可以不制作调解书，二审必须有调解书
       2.1）调解和好的离婚案件
       2.2）维持收养关系
       2.3）即时履行
            ie.一次性+当场履行

## 3.二审
### 3.1 调不成发回
        1）漏事
           ie.一审已经提出诉讼请求 + 原审法未做处理
        2）漏人
           ie.必要共同诉讼人、有独三
        3）if 一审判不准离婚 + 二审认为应当离 ;
               if 当事人自愿；
                     可以与子女抚养、财产问题 一并调解；
                          调不成发回
               if 双方同意由 二审一并审理的；
                        可以一并裁判

### 3.2 调不成告知另诉
        1）增加独立诉讼请求
        2）二审提反诉
        if 1）2）双方当事人同意 由 二审法 一并审理；
                     可以一并裁判

### related：再审中增加独立的诉讼请求
        1）usu
           不调，直接告知另诉
              cuz 不属于再审的审理范围
        2）exp 一并审理
           2.1)涉及国社利益
           2.2)原审中要求增加、变更 + 原审没有加

### 3.3 调解成功
        应当制作调解书
        调解书送达签收后，原审法院判决视为撤销

## 4.再审
     1）达成调解协议的 + 应当制作调解书 
     2）签收生效+ 原判决裁定视为被撤销（无需写明撤销原判）
     3）漏人，调不成，发回 原一审
          cuz 民诉中的发回都是发回 原一审（一发到底）
              diff && 刑诉：only 死刑复核可以发一审，也可以发二审
                           else 都是发原审

## 5.人民调解
### 5.1 genernal principle
        1)调解不是必经阶段
        2）调解不影响起诉
        3）没有 严格的 程序要求

### 5.2）确认调解协议
           双方共同 + 自调解协议生效之日起30days 内 + 书面||口头调解组织所在地 + 基院
              if 法院受理后；
                      书面审查 
                    if 裁定调解有效 + 一方不履行 || 不全部履行；
                         向法院申请执行
                    if 裁定驳回申请；
                         1）通过调解方式
                            1.1）变更原调解协议
                            1.2）达成新的调解协议
                         2）向法院起诉 

## 6.仲裁调解
     1）应当制作调解书
     2）裁决书
        2.1)不能形成多数意见，按照首仲的意见
        2.2）可签可不签，可记可不记
     3）裁决自 作出之日 起生效


# III.和解
## 1.一审和解结案方式
       1）依据和解协议制作调解书
            spec:公益诉讼only 调解书结案（C_ch_1）
                    if 公益诉讼调解、和解 公告期满 + 法院审查不损害国社公共利益；
                                 应当出具调解书
                          if 当事人以达成和解协议 申请撤诉；
                                   不予准许
            exp:1)涉外
                  判决书结案
                2）无人 的离婚
       2）撤回起诉
          可以再次起诉

## 2.二审和解结案方式
      1）依据和解协议制作调解书
      2）因和解而撤诉 + 符合撤诉条件 ；应予准许
            if 撤回上诉 + 对方不履行；
                            依据一审判决要求强执
            if 撤回起诉 + 对方不履行；
                            不能再起诉，且 没有生效法律文书可以强执
                                cuz 法院会撤销原判决

## 3.仲裁和解
     1）裁决书
     2）撤回仲裁申请
        if 撤回仲裁申请后反悔；
               根据 原协议 申请仲裁

## 4.执行和解
     1）可以口头 + 记入笔录 
     2）if 达成 + 履行完毕；
               裁定执行终结
     3）if 欺诈、胁迫 || 部分履行 
        当事人有选择权
        3.1）only 依申请（！依职权）恢复执行；
                    法院 按照 原生效文书执行，已经履行的部分扣除
        3.2）根据 和解协议 起诉
             ！原纠纷

# IV.撤
## 1 自认的撤回
        if 在法辩 终结前 + 撤回承认 + 对方同意；
               对方仍然要举证
          else if 有充分证据证明 + 自认是在 胁迫、重大误解做出 + 与事实不符；
                        对方仍然要举证

## 2 一审撤诉
### 2.1 主体
          P、有独三、上诉人及其法代

### 2.2 申请撤诉
        if 在立案受理后，进行宣判前 + 书面||口头 申请撤回起诉；
             usu:准予撤诉
             exp:1)违法行为需处理
                 2）法辩终结后 + 被告不同意
                     可以不准撤
                 3）法辩终结后 + 公益诉讼（C_ch_1）
                     应当不准撤

### 2.3 按撤诉处理，可再审
       1）P 不交诉讼费、不来、先走
       2）P 是 无人的法代，不来
       3）有独三 不来、先走
             cuz 相当于 P.撤诉之后可再起诉

### 2.4 后果
       1）本案诉讼程序终结
       2）撤诉视为未起诉，还可以再起诉
       3）诉讼时效重新计算

## 3 二审撤诉
### 3.1 撤回上诉
          if 在上诉期 内 + 撤回上诉；
                 不能再上诉（上诉权1次用尽），一审判决期满生效
          if 在上诉期满 后 + 二审法审查；
                  usu 应当裁定准予撤诉 + 裁定准许后，一审判决立即生效
                  exp 1）一审错
                      2）双方恶意串通 损害 国社利益
                          裁定不准撤

### 3.2 撤回起诉
          if 在二审中P 撤回起诉 + D 同意 + 法院审查 不损害 国社他人利益；
                    可以准许 + 应当一并撤销一审裁判（视为未起诉，没有文书） + 不得再起诉
                       if P 二审撤回起诉后 再起诉的；
                                    裁定不予受理

### 3.3 再审撤诉
#### 3.3.1 按照一审撤回起诉
          if 在一审中P 撤回起诉 + D 同意 + 法院审查 不损害 国社他人利益；
                    可以准许 + 应当一并撤销 裁判调（视为未起诉，没有文书） + 不得再起诉
                       if P 再审撤回起诉后 再起诉的；
                                    裁定不予受理

#### 3.3.2 撤回再审申请
          1）自己撤
             申请撤回 + 法院裁定 
               if 裁定准许 + 终结再审程序
                   if 撤回后 又 申请再审的；
                         usu:不予受理
                         exp:自知||应当知道 6mon 内 + 
                             1）伪证 + 认定事实的 主要证据
                             2）新证 + 足以推翻 原判决裁定
                             3）据以做出原判、裁的法律文书被撤销、变更
                             4）审理人员有不当行为
                                incl:贪污受贿、徇私舞弊、枉法裁判
          2）按照撤回处理
             2.1）再审申请人、申请抗诉当事人 不来、先走
             2.2）再审申请人、申请抗诉当事人  无正当理由不接受 询问

## 4.申请宣告失踪、死亡后 撤回申请
    if 做出前  + 申请人撤回申请；
           应当裁定终结特别程序
             exp:利害关系人加入程序要求继续审理

## 5.确认调解协议 撤回申请
    if 裁定做出前 + 当事人申请撤回；
                    可以裁定准许
    if 当事人无正当理由 + 未在期限内补充陈述、补充证明材料 || 拒绝接受询问；
                    可以按照撤回处理

## 6.撤回公示催告
    If 公示催告前 撤回；
                 应当准许
    if 公司催告期间 撤回；
                 裁定终结公示催告程序               

# IV.二审
# 1.法院
## 1.1 上诉状的提交
       通过原审法院 + 书面
       if 直接交二审法；
             二审法5days 内 把上诉状 交给 原审法；
                    原审法 在 把案卷材料、上诉状 交给二审法

## 1.2 二审法院
       only 上一级法

## 1.3 裁判
       if 一审对；
             判决驳回上诉，维持原判决（笔误裁定补正）
             diff && 刑诉：裁驳，维持原判
       if 事实错、法律错；
             应当改判
       if 事实不清；
             可改可发
       if 程序违法；
             应当发（只能发1次，再次上诉的应当改判）
       if 二审法认为 不应由法院审理的；
                     裁定撤销原判，驳回起诉


# V.再审
    genernal principle
    1)有限再审，1次法（申请再审）+1次检（申请抗诉||申请建议）
         alike 两级申诉
    2）法院纠错先行，检察监督断后

# 1.启动
## 1.1 法院启动
       1）针对本院 +  裁判调
          院长提交 + 审委会决定
       2）其他法院 +  裁判调
          最高院、上级法 + 一律提审

## 1.2 检院启动
> 再审抗诉：上级检 抗 下级法，文书同级送

     inti:基院 一审生效裁判 + 
          1）if  市检 认为有错；
                    向 基院 再审抗诉 + 向 中院 送文书
                if 中院 30days 内做出再审的裁定
                      if 有错；
                           提审 （按照二审）
                              if 5个证据原因;
                                        提审 || 交 基院(按照一审)
                                           but:经该下一级法再审的 Only 提审
          2）县检 
             2.1）if 县检 认为有错 + 检委会 讨论决定 后；
                        向 基院 提检察建议 + 报 市检 备案;
                    if 基院 收到 检察建议后 + 合议庭 + 3mon 内审查；
                          if 有错；
                                裁定再审 + 通知当事人
                            else if  ；
                                书面回复 + 县检院
             2.2）提请 市检 向 基院 再审抗诉

## 1.2.1 5个证据原因
         1）新证 + 足以推翻原判决、裁定的
         2）缺乏证据 + 原判决裁定据此认定的
         3）伪证 + 主要证据 + 原判决裁定据此认定的
         4）未经质证 + 主要证据 + 原判决裁定据此认定的
         5）法院未调取证据 + 当事人书面申请

## 1.2.2 经该下一级法再审的 Only 提审

## 1.3 当事人申请
### 1.3.1 申请的主体
          1）原审P\D\TP（有独三、承担义务的无独三）、上诉人、被上诉人
          2）案外人、被遗漏的必要共同诉讼人
          3）死者的权利义务继受人
            ！incl:债权转让的受让人（非原审中的当事人）

### 1.3.2 申请的对象
          1）裁、判
          2）调解书 + 证明
             1）违反自愿
             2）调解协议内容违反法律

### 1.3.3 申请期限
          1)usu：
            客观标准：裁、判、调 生效后 + 6mon（不中止、中断、延长）
          2）exp:
            主观标准：知||应当知道 + 6mon
            2.1）新证 + 足以推翻原判决、裁定的
            2.2）伪证 + 主要证据 + 原判决裁定据此认定的
            2.3）据以做出的原 判决、裁定 被撤销||变更
                 ！incl:调解书
            2.4）审判人员审理时贪污贿赂、徇私舞弊、枉法裁判
            2.5）被 遗漏 的必共人 未接到厂家诉讼的通知

### 1.3.4 法院
          if 法院受理申请后；
                 应当合议庭 + 自收到再审申请书之日起 3mon 内审查完毕 + 院长批准可延长
                                if 审查 + 理由是 新证 + 足以推翻原判决、裁定的；
                                            应当询问 当事人
                                   else if;
                                            可以询问 当事人
                    if 再审理由成立；
                           裁定再审
                      else if 超过再审范围||超期申请；
                                  裁定驳回 再审申请

#### 1)usu 向上一级法再审 + 最低中院审理

          2）交原生效审 中院 一审程序
          3）交 与原审同级 的其他法院 一审程序

     case_中院 二审生效裁判
          向 高院 交再审申请
          1）自己审 提审 二审程序
          2）交原生效审 中院 二审程序
          3）交 与原审同级 的其他法院 二审程序

#### 2）exp 向原审申请再审 + 可以 基院审理
        if 一方人数众多||双方都是公民；
                 原审||上一级法
                    if 一方当事人向原审申请，一方向上一级申请；
                            协商不成，原审优先

       case_基院 一审生效裁判 + 一方人数众多||双方都是公民 
            1）向 中院 交再审申请

                          中院 only 提审  二审程序

#### 3）exp 高院、最高院 不能指令再审
       3.1）原审法院再审审理过1次的
       3.2）原判、裁 是审委会做出的

       3.4）原审法院无管辖权的
       3.5）需要同意适用||裁量权行使标准的

# 2.审理
    if  原审对；
             判决驳回上诉，维持原判决（有瑕疵纠正后维持）
    if 事实不清；
             可改可发（usu 改判；exp:发回有利于查清事实才发回）
    if 原审严重违反程序；
             应当发回
    if 不符合受理条件
             裁定撤销一、二审 判决 + 驳回起诉（撤销所有的文书）

# 3.小额的再审
    depends 申请再审的事由
    1)if 以再审的13个事由之一 申请再审；
           原审法 应当受理
             if 理由成立；
                   应当裁定再审 + 合议庭审理 + 再审判决裁定 不得上诉
    2）if 以 程序错（不应当按照小额审理） 申请再审；
           原生发 应当受理
              if 理由成立；
                  应当裁定再审 + 合议庭审理 + 再审判决裁定 可以上诉
