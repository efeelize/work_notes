### 消保法的规定
#### 1）一般商品
       if 一般瑕疵（电视没声音）；
              only 追销售者；销售者向生产者、其他销售者追偿
       else if 缺陷（造成人身损害、财产损害）；
              选择权：销售者 && 生产者（only 缺陷产品承担绝对无过错） 连带责任
              受害人incl:购买者、使用者、第三人
              一定要题目中说是“认定为缺陷”，才能适用缺陷产品的规则
#### 2）食品领域；
        % 生产者首负责制 % 
#### 3）具体规定
        1）展销会、租赁柜台
           usu 销售者、服务者
           exp:if 租赁期满 ；
                    展销会、租赁柜台
        2）网络交易平台
           usu:销售者、服务者
           exp:if 1）平台不能提供真实名称、地址、有效联系方式 == 先赔
                  2）平台做出更有利于消费者的承认的 == 先赔
                  3）明知||应知 + 未采取必要措施 == 连带
        3）虚假广告
           usu：经营者
           exp:1）S 网络；
               2）if经营者、发布者设计、制作、发布 关系消费者生命健康产品||服务的虚假广告；
                                                     连带
               3）if 社会团体、其他组织、个人 +关系生命健康产品||服务的虚假广告 + 推荐；
                                                     连带

# 3.无过错
### 1）承运人责任
       1.1)旅客自身原因||故意重大过失免责
       1.2)必须首先要证明在运输过程中受到了损害 ：
             if 甲在火车上失踪，生死未卜;
                          未能证明与运输有关,承运人不承担责任 

### 2）产品责任
#### 2.1）消费者主张
         1)基于合同
            only 找销售者 承担 违约责任；
         2）基于加害：
            1）要求承担违约责任：
               销售者 
            2）要求承担侵权责任：
               生产者||销售者 
         3）基于加害给付：不能既主张违约又主张侵权
         4）if 运输者、仓储者的过错；
                        消费者也不能起诉仓储者、运输者
#### 2.2）内外部的规则
          1）内部
            1.1）生产者
                 相对无过错
            1.2）销售者
                 绝对无过错
          2）外部
             都是无过错


### 3)帮工
> int A被帮工的人；B帮工人；C 第三人/侵权人
#### 3.1）A明确拒绝
          1）B致人损害、B 自己伤害自己
             A不担责 + 根据公平原则补偿
          2) C 伤害了 B
             C 赔偿 
#### 3.2）A未拒绝
          1）B致人损害
             A 担责
             if B 故意||重大过失；
                           A\B 连带
          2）B 自己伤害自己
             A 承担全部责任
             diff && 劳务用工：根据各自过错担责
          3）C 伤害了 B
             C 赔偿 
#### 3.3 ）C 
          1）下落不明
             不列其为当事人
          2）没有赔偿能力
             列 C&&A 为共同 D
