## 1.1 产品存在缺陷造成损害的
       生产者||销售者->谁造成的向谁追偿

### 消保法的规定
#### 1）一般商品
       if 一般瑕疵（电视没声音）；
              only 追销售者；销售者向生产者、其他销售者追偿
       else if 缺陷（造成人身损害、财产损害）；
              选择权：销售者 && 生产者（only 缺陷产品承担绝对无过错） 连带责任
              受害人incl:购买者、使用者、第三人
              一定要题目中说是“认定为缺陷”，才能适用缺陷产品的规则
#### 2）食品领域；
        % 生产者首负责制 % 
#### 3）具体规定
        1）展销会、租赁柜台
           usu 销售者、服务者
           exp:if 租赁期满 ；
                    展销会、租赁柜台
        2）网络交易平台
           usu:销售者、服务者
           exp:if 1）平台不能提供真实名称、地址、有效联系方式 == 先赔
                  2）平台做出更有利于消费者的承认的 == 先赔
                  3）明知||应知 + 未采取必要措施 == 连带
        3）虚假广告
           usu：经营者
           exp:1）S 网络；
               2）if经营者、发布者设计、制作、发布 关系消费者生命健康产品||服务的虚假广告；
                                                     连带
               3）if 社会团体、其他组织、个人 +关系生命健康产品||服务的虚假广告 + 推荐；
                                                     连带

## 1.2 不合格血液致人损害
       生产者||血液提供者||医疗机构->医疗机构向另外两个追偿
## 1.3 第三人过错污染环境
       污染者||第三人->向第三人追偿
## 1.4 第三人过错动物致人损害
