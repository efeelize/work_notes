             like:if 当事人不懂事，拿着澳门判决在内地多个法院申请了（没有约束住当事人）；

      2）调取证据、送达文书

      2）！拒绝其他司法协助事项

# ii.具体规定

# 1.涉外仲裁
## 1.1 仲裁协议的效力认定
### 1）认定机构
       1.1）仲裁机构
       1.2）中院||专门法院 + 
            约定的仲裁机构所在地、仲裁协议签订地、（被）、申请人住所地 

### 2）适用法律
       if 当事人选择；
             意思优先
       else if 没有选择；（***）
          仲裁机构所在地||仲裁地法（仲裁地实际开庭&&做出生效裁决的地方） 
                if 适用这两个地方的法对仲裁协议的效力做出 不同认定的；
                                       应当适用 确认有效 的法律
       else if 没有选择适用的法律、仲裁机构、仲裁地||约定不明；
                                      中国法

### 3）结果
      if 认定有效；
            有管辖权的法院自己做出裁定
      else if 认定无效（***）；
            层报最高院 + 最高院出审核意见 + 依照做出裁定

## 1.2 涉外仲裁裁决的执行
### 1）管辖法院
     被执行人住所地 || 被执行财产所在地 + 中院
### 2）不予执行
#### 2.1）only当事人申请
         1）无仲裁条款||仲裁协议书
         2）不是被申请人指定的仲裁员
         3）组成||仲裁程序违法
         4）超裁
            ie.越权仲裁||超过了可仲裁的范围
#### 2.2）法院主动审查 
         违背社会公共利益
### 3）程序
       if 裁定执行；
             有管辖权法院自己做出裁定
       else if 不予执行；
             层报最高院 + 依最高院的审核意见做出裁定
### 4）救济
       if 裁决刚做出；
            当事人就同一纠纷 再 申请仲裁||向法院起诉；
                               不予受理（cuz 此时应申请执行）
       if 裁决做出 + 申请了执行 + 裁定不予执行；
                                重新达成仲裁协议申请仲裁 || 起诉

### related：港澳台判决的救济
    if 对内地做出的不服；
                向上一级法 申请复议
    if 对港澳做出的不服；
                依照当地法律 上诉
     case_甲乙在台北因合同纠纷涉诉，乙败诉，甲向乙财产所在地的大陆法申请认可台湾判决
          1）if 甲申请；
                   应当提供相关证明：文书生效+真实
          2）if 法院受理;
                   自立案之日起 6mon 审结 + 特殊情况报 上一级法 延长
          3）if 法裁定不予认可 + 申请人再次提出申请；
                   法院不予受理 + 申请人可就同一争议向法院起诉

# 2. 外国仲裁裁决的 承认 && 执行
## 1）管辖法院
        1）usu
           被执行人住所地 || 其财产所在地 + 中院
        2) exp 
           if 两个地方都不在我国（cuz 没有执行的可能）；
                  受理关联案件（拿着外国仲裁裁决在中国法起诉）的法院
## 2）保留
      1）仲裁地在缔约国
         ！care else 
      2) only 商事仲裁裁决
         ！incl:与人身密切相关
      3）incl：临时仲裁裁决（临时仲裁庭作出）
      4）争议双方都是私法主体
         ie.任何一方都不能是国家
## 3）程序
      if 当事人 + 仲裁裁决生效后2years 内 提出申请
               if 分开提（时效中断）；
                     自法院承认裁定生效之日起重新计算2years 期间
                          
          法院在2mon 内做出裁定
             if 裁定承认；
                  6mon内执行
             else if ；
                  层报最高院

      notes:1)only 当事人申请
              我国法 usu 无权主动审查，
                        依被执行人的申请，拒绝承认&&执行外国仲裁裁决
                    exp 违反我国社会公共利益 || 我国保留的不具有可仲裁性的事项
            2）only 审查程序
               if 程序有问题；
                       only 拒绝承认 && 执行，不能撤销
            3）if 承认 && 执行申请被裁定驳回；
                               可向有管辖权法起诉
                                  cuz 当事人之间的具体民商事争议尚未解决

## 3.法院判决的承认 && 执行
### 1）我国执行外国法判决、裁定
       if 当事人（直接申请）、外国法（依照条约||互惠原则） 
          + 向被执行人住所地||财产所在地 
          + 中院 
          + 2years 内 申请；
               
               法院审查 条约||互惠关系
                  if N;
                     不承认 && 执行 + 裁定驳回申请
                         exp:外国法做出的 + 发生法律效力的离婚判决（难以满足条件）
                       if 不服；
                          当事人向法院起诉
                 
                  else if Y + 继续审查;
                          incl:1)已经发生法律效力的判决||裁定
                               2）原判决国法院必须有管辖权
                                     like:专属管辖
                               3）审判程序公正
                               4）不与正在我国国内进行||已经终结的诉讼相冲突 
                               5）不损害我国的社会公共利益
                                if 符合条件；
