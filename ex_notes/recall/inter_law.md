# 1.涉外仲裁
## 1.1 仲裁协议的效力认定
### 1）认定机构
       1.1）仲裁机构
       1.2）法院
            incl:仲裁协议约定的仲裁机构所在地、仲裁协议签订地、申请人住所地、被申请人住所地 + 中院||专门法院

### 2）适用法律
       if 当事人选择；
             意思优先
       else if 没有选择；（***）
          仲裁机构所在地||仲裁地法（仲裁地实际开庭&&做出生效裁决的地方） 
                if 适用这两个地方的法对仲裁协议的效力做出 不同认定的；
                                       应当适用确认仲裁协议 有效 的法律
       else if 没有选择适用的法律、仲裁机构、仲裁地||约定不明；
           中国法

### 3）结果
      if 认定有效；
            有管辖权的法院自己做出裁定
      else if 认定无效（***）；
            逐级层报最高院 + 依照最高院的审核意见做出裁定

## 1.2 涉外仲裁裁决的执行
### 1）管辖法院
     被执行人住所地 || 被执行财产所在地 + 中院
### 2）不予执行
#### 2.1）only当事人申请
         1）无仲裁条款||仲裁协议书
         2）不是被申请人指定的仲裁员
         3）仲裁庭的组成||仲裁程序违法
         4）越权仲裁||超过了可仲裁的范围
#### 2.2）法院主动审查 
         违背社会公共利益
### 3）程序
       if 裁定执行；
             有管辖权法院自己做出裁定
       else if 不予执行；
             层报最高院 + 依最高院的审核意见做出裁定
### 4）救济
       if 裁决刚做出；
            当事人就同一纠纷 再申请仲裁||向法院起诉；
                                         不予受理（cuz 此时应申请执行）
       if 裁决做出 + 申请了执行 + 裁定了不予执行；
                当事人可以 重新达成仲裁协议申请仲裁 || 起诉
### related：港澳台判决的救济
    if 对内地做出的不服；
                向上一级法院申请复议
    if 对港澳做出的不服；
                依照当地法律上诉

### 2.IP担保义务

          1）买方营业地
          2）合同预期的货物转售地||使用地      
#### 2.2）卖方免责

## 2.2 买方
### 1）接收货物
       if 货物瑕疵（即使构成严重违约）；
                      接收是义务（没有拒收权） + 不等于接受 + 仍然有退换货权利
          cuz 否则一定对承运人产生损失
 
       if 提交交货 || 多交货（only 拒收多交的部分）；
                     only 这两种情况下才有拒收的权利
#### 2）检验货物权利的限制
       1）usu 切实可行的最短时间 + 目的地
       2）exp 卖方已知货物需要转运（约定须转运） + 新目的地

# II.运输法
# 1.bill of lading
    1)class
      if depends 有无记载船名、航次；
           已装船 && 收货待运
      if depends 有无不良批注（保护承运人利益）；
           清洁 && 不清洁
      if depends  收货人一栏的填写方式；
            if 明确载明收货人姓名、名称
                        记名：收货人唯一，非流通
            else if 空白||交持有人；
                        不记名：认单不认人，交付达到转让的效果
            else if 凭指示；
                        指示提单：背书转让

# 2.承运人交货责任
## 2.1 赔偿
       if 无单放货；
             正本提单持有人 选择诉 承运人||取货人（连带责任）
                if 诉承运人；               
                      1）违约 && 侵权 竞合
                      2）赔偿额为 CIF 价格
                      3）足额赔偿
                         ！适用海事赔偿限额
                      4）诉讼时效1year + only 适用海商法
## 2.2 承运人免责
       1）依法免责
       2）记名提单（收货人唯一性）+ 托运人明确要求（中止运输、返还货物、变更到达地点）
             cuz 承运人本来就是按照托运人的要求对特定收货人交货
       3）签发了2份以上正本提单 + 其他正本提单已提货 

# 3.承运人货损责任
     1st:depends 致损原因
         if 违反最低限度的义务（适航||管货）；
                           赔（无论什么规则）
         if 承运人对货损没有过失（incl 无过失导致的火灾）；
                          不赔（无论什么规则）
     2nd:if 航行过失；
              if 海牙||维斯比规则；
                             不赔
              if 汉堡；
                             赔
     3rd:if 火灾；
              if 雇用人代理人过失导致；
                           if 海牙||维斯比规则；
                             不赔
                           if 汉堡；
                             赔
              if 承运人实际过失（船舶不适航）；
                             赔
              if 私谋导致火灾；
                             赔
