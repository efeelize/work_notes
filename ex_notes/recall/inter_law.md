        中外合资、中外合作、中外合作勘探开发自然资源合同

### 2nd:允许 + 限制意思自治
#### 1）消费者合同 （***）
        if 消费者选择(避免经营者滥用优势地位)；
               商品、服务提供地（平衡经营者利益）
        else if 没有选择；
               depends 消费者经常居住地是否从事相关经营活动
                         like:在中国已经开设了实体店
                   if Y;
                        经常居所地法
                   else if；
                        商品、服务提供地
#### related:产品责任
        if 被侵权人选择；
               侵权人主营业地法 || 损害发生地法
        else if ；
               depends 侵权人 在被侵权人经常居住地 是否从事相关经营活动
                  if Y;
                        被侵权人经常居所地法
                   else if；
                        侵权人主营业地法 || 损害发生地法

#### 2）劳动合同
##### 2.1）聘用合同；
             if 能够确定劳动者工作地（在xx国）；
                   劳动者工作地法
             else if（是船长）;
                   用人单位主营业地法
###### 2.2）劳务派遣合同 （***）
             if 能够确定劳动者工作地；
                      劳动者工作地||劳务派出地法（无条件选择的冲突规范）
                      notes:if 从 A派到 B；
                               A地为劳务排除地法
             else if ;
                   用人单位主营业地法||劳务派出地法

## 3.2 侵权之债
### 1st:法律规定
#### 1）侵害人格权
        被侵权人经常居所地法

#### 2）船舶 && 民用航空器侵权

               船旗国法
        if 不同国籍；
             if 碰撞在内水||领海；
                       侵权行为地法
             else if 碰撞在公海；
                       法院地法（cuz公海不是任何一个国家的领土）

### 2nd:允许 + 限制意思自治
#### 1）产品责任
#### 2）IP侵权 （***）

#### related：IP
        if 归属 && 内容；
             被请求保护地法
        if 转让 && 许可；

# 4.家庭关系
## 4.1 夫妻关系
### 1st:特殊
        1）诉讼离婚
           法院地法
        2）婚姻手续 满足之一就有效
           incl:婚姻缔结地法、一方当事人经常居所地法、国籍国法

### 2nd:允许 + 限制意思自治
        1）夫妻财产关系
           if 选择；
               一方经常居所地法||国籍国法||主要财产所在地法
           else if;
               共居法 > 共同国籍国法
        2）离婚协议
           if 选择；
               一方经常居所地法||国籍国法
           else if;
               共居法 > 共同国籍国法 > 办理离婚手续机构所在地法
### 3rd：usu:共居法 > 共同国籍国法 > 行为地法
        1）结婚条件
           共居法 > 共同国籍国法 > 婚姻缔结地法
        2）夫妻人身关系（姓氏权）
           共居法 > 共同国籍国法
### 4th: 最密切联系（兜底）


## 4.2 与其他人之间
> 原则上都要：保护弱者利益
### 1）父母子女关系
       共居法 > 一方经常居所地法||国籍国法中 + 有利于保护弱者利益
### 2）抚养（抚养、赡养、扶养）
       一方当事人经常居所地法、国籍国法、主要财产所在地 + 有利于保护被抚养人权益
### 3）监护
       一方当事人经常居所地法、国籍国法 + 有利于保护被抚养人权益

### 4）收养
     4.1）条件&&手续 （only 重叠适用的冲突规范 ）
         收养人 && 被收养人 经常居所地法
     4.2）效力
        （cuz 收养的发生depends 收养人的行为 + 鼓励保护善意收养人）
         收养时 + 收养人 经常居所地法
     4.3）解除
         （cuz 被收养人处于弱势一方+ 谨慎解除）
         收养时 + 被收养人 经常居所地法 || 法院地法
     4.4）程序要求
          1）转交申请（cuz 收养关系的程序要符合两国的收养法）
             通过外国收养组织 向 中国收养组织转交申请
          2）亲自办手续
             cuz 与人身密切相关
             if 一方来不了；
                  Only 委托 另一方
          3）书面收养协议
