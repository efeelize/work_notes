                           港判能，else 是 X 的交叉关系

    3）专属管辖
       3.1） only拒绝法院判决
       3.2）！拒绝其他司法协助事项

# 1.涉外仲裁
## 1.1 仲裁协议的效力认定
### 1）认定机构
       1.1）仲裁机构
       1.2）法院
            incl:仲裁协议约定的仲裁机构所在地、仲裁协议签订地、申请人住所地、被申请人住所地 + 中院||专门法院

### 2）适用法律
       if 当事人选择；
             意思优先
       else if 没有选择；（***）
          仲裁机构所在地||仲裁地法（仲裁地实际开庭&&做出生效裁决的地方） 
                if 适用这两个地方的法对仲裁协议的效力做出 不同认定的；
                                       应当适用确认仲裁协议 有效 的法律
       else if 没有选择适用的法律、仲裁机构、仲裁地||约定不明；
           中国法

### 3）结果
      if 认定有效；
            有管辖权的法院自己做出裁定
      else if 认定无效（***）；
            逐级层报最高院 + 依照最高院的审核意见做出裁定

## 1.2 涉外仲裁裁决的执行
### 1）管辖法院
     被执行人住所地 || 被执行财产所在地 + 中院
### 2）不予执行
#### 2.1）only当事人申请
         1）无仲裁条款||仲裁协议书
         2）不是被申请人指定的仲裁员
         3）仲裁庭的组成||仲裁程序违法
         4）越权仲裁||超过了可仲裁的范围
#### 2.2）法院主动审查 
         违背社会公共利益
### 3）程序
       if 裁定执行；
             有管辖权法院自己做出裁定
       else if 不予执行；
             层报最高院 + 依最高院的审核意见做出裁定
### 4）救济
       if 裁决刚做出；
            当事人就同一纠纷 再申请仲裁||向法院起诉；
                                         不予受理（cuz 此时应申请执行）
       if 裁决做出 + 申请了执行 + 裁定了不予执行；
                当事人可以 重新达成仲裁协议申请仲裁 || 起诉
### related：港澳台判决的救济
    if 对内地做出的不服；
                向上一级法院申请复议
    if 对港澳做出的不服；
                依照当地法律上诉


# 2. 外国仲裁裁决的 承认 && 执行
## 1）管辖法院
        1）usu
           被执行人住所地 || 其财产所在地 + 中院
        2) exp 
           if 两个地方都不在我国（cuz 没有执行的可能）；
                  受理关联案件（拿着外国仲裁裁决在中国法起诉）的法院
## 2）保留
      1）仲裁地在缔约国
         ！care else 
      2) only 商事仲裁裁决
         ！incl:与人身密切相关
      3）incl：临时仲裁裁决
         ie.临时仲裁庭作出
      4）争议双方都是私法主体
         ie.任何一方都不能是国家
## 3）程序
      if 当事人 + 仲裁裁决生效后2years 内 提出申请
               if 分开提；
                     自法院承认裁定生效之日起重新计算2years 期间
                          ie.发生时效中断的效力
          法院在2mon 内做出裁定
             if 裁定承认；
                  6mon内执行
             else if ；
                  层报最高院

      notes:1)only 当事人申请
              我国法 usu 无权主动审查，
                    exp 违反我国社会公共利益 || 我国保留的不具有可仲裁性的事项
            2）only 审查程序
               if 程序有问题；
                       only 拒绝承认&&执行，不能撤销

## 3.法院判决的承认 && 执行
### 1）我国执行外国法判决、裁定
       if 当事人（直接申请）、外国法（依照条约||互惠原则） 
          + 向被执行人住所地||财产所在地 
          + 中院 
          + 2years 内 申请；
               法院审查 条约||互惠关系
                  if N;
                     不承认&&执行 + 裁定驳回申请
                         exp:外国法做出的 + 发生法律效力的离婚判决（难以满足条件）
                       if 不服；
                          当事人向法院起诉
                  else if Y + 继续审查;
                          incl:1)已经发生法律效力的判决||裁定
                               2）原判决国法院必须有管辖权
                                     like:专属管辖
                               3）审判程序公正
                               4）不与正在我国国内进行||已经终结的诉讼相冲突 
                               5）不损害我国的社会公共利益
                                if 符合条件；
                                      裁定承认执行 + 发出执行令
### 2）外国执行我国法判决裁定
      if 当事人直接申请||我国法院依照条约、互惠原则；
