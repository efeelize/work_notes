# ch_3 行政复议 && 国赔

# I.行政复议 **结合行政诉讼一起考查

## 1.受案范围
- 1）许可、处罚、强制、登记、协议、裁决、给付
- 2）确认自然资源所有权||使用权
- 3）红头文件

## 2.当事人
### 2.1 申请人
- 相对人&&相关人

### 2.2 被申请人
- 1）谁有行政主体资格，谁是被申请人
- 2）谁批准谁是被申请人

### 2.3 第三人
- 复议中的第三人 only 可以通知，cuz 复议不是最终程序

## 3.管辖
> 1st：找被申请人；2nd:找复议机关；3rd:县政府收材料的，有权就管，无权的7days 内转

## 4.审理程序
### 4.1 申请
- 1）原则上自知道行为之日起60days 内，exp>60days

> <反不正当竞争法>规定的是15days,还是要适用60days 的规定

- 2) 书面+口头：BUT 企业不可以口头委托
- 3）不得收费
- 4）鉴定可以当事人自行委托，鉴定时间不计入复议审理期间

### 4.2 受理
- 1）5days 内补正。一次性（！）
- 2）竞合的时候由最先收到的机关受理。最先立案（！）

### 4.3 审查
- 1）原则上书面审查，exp 重大复杂+可以+听证
- 2）复议自愿撤回之后，即使未超过期限，也不允许再诉。exp:申请人能够证明撤回违背真实意思表示。
- 3）复议的撤回需要复议机构的同意。

### 4.4 决定
- 1）不得作出对申请人更不利的复议决定
- 2）原则上90days;exp<60days

## 5.结案&&执行
- 1)可以调解、和解结案
- 2）可以主动作出附带赔偿的决定：cuz行为确实违法，是肯定要赔偿的
- 3）if 被申请人行为违法；撤销||确认||变更判决
- 4）if 被申请人不作为+可以履行：履行判决
- 5）if 被申请人行为合法：决定维持
- 6）if 不应受理||对不作为的申请不成立；决定驳回

# II.国赔
## 1.构成要件
- 1）关了打了才要赔

> 1.1)人身自由赔偿 only 实际羁押，else 不赔；1.2）侵犯人身权->精神损害赔偿

- 2）谁主张谁举证；exp 被限制人身自由的人死亡||丧失行为能力的
- 3）违法归责：违反法律规范||从普通人的角度未尽到合理注意义务
- 4）赔偿机关：最后一个做出错误的法律文书的机关为赔偿义务机关

## 2.赔偿范围
### 2.1赔
    1)错误实际羁押赔偿，重判不赔
      1.1)违法实际羁押；
      1.2）合法拘留+超期羁押（57days）+无罪 == 赔57days；
      1.3）逮捕+撤销案件；
      1.4）逮捕+存疑不起诉；
      1.5）逮捕+判决宣告无罪;
      1.6)数罪并罚，一个无罪，无罪的部分赔偿

### 2.2不赔
    2.1)合法拘留+无罪+合期限羁押（37days）；
    2.2)合法拘留+有罪+超期羁押；
    2.3）逮捕+法定/酌定不起诉；
    2.4）有罪改为无罪，只赔偿判决生效前被实际羁押的部分；
    2.5）无刑事能力的人||免于追究刑事责任的人逮捕羁押 + 判处刑罚 == 之赔偿后面执行刑罚的部分，之前羁押的部分不赔

## 3. 程序
> depends 哪部分侵权

### 3.1 行政行为侵权
#### 3.1.1 way1:在诉讼||复议中一并解决
#### 3.1.2 way2:先向赔偿义务机关申请赔偿+单独提起行政赔偿诉讼

##### 3.3 赔偿义务机关先行处理
- 1）申请材料不全的，5days 内一次性告知补正
- 2）受到材料的 2mon 内做出是否赔偿的决定
- 3）if 赔偿；10days 送达->不服自期限届满3mon 内向起诉
- 4）if 不赔：10days 书面通知+说明理由->不服自决定之日期3mon 内起诉 

##### 3.4 对3.3结果不服的，再按照分类找上一级

    if 赔偿请求人知道合法权益受到损害；2years 请求赔偿义务机关先行处理
       but ！incl:被羁押等限制人身自由的期间
           适用民诉时效的中止
       if {赔偿义务机关是法院}；30days 内向 该法院的上一级法赔委会申请赔偿；
       if {赔偿义务机关非法院：like 公、检、看守所、监狱}；30days 内向 该机关的上一级机关申请复议/未告知的2years 内申请复议->30days 内向 复议机关所在地 同级法赔委会申请赔偿
    $:赔委会的决定第1次就生效，不能上诉，立即生效

### 3.2 事实行为侵权：先向赔偿义务机关申请赔偿+单独提起行政赔偿诉讼
- 1）行政赔偿诉讼不是行政诉讼，only 是国赔中的行政赔偿程序

> like:城管打人了

## 4. 内容 
### 4.1 方式
- 1）返还财产、恢复原状、金钱赔偿、精神损害抚慰金
- 2）垃圾：消除影响、恢复名誉、赔礼道歉

### 4.2 计算标准
#### 4.2.1 侵害人身
- 1）限制人身自由：误工费->上年度->做出的赔偿决定内容没有被更改的的上一年度

> like:2017年做出，2018年维持，按照2016年的标准赔偿

- 2）造成身体伤害：医疗费、护理费、误工费
- 3）丧失劳动能力

> 全残+抚养的无劳动能力的人 == 应当支付生活费；

- 4）死亡：总额为上年度20倍，if抚养的无劳动能力的人;应当支付生活费
- 5)精神损害：后果严重的才支付精神损害抚慰金

#### 4.2.2 侵害财产
- 1）损失发生时/侵权行为时的价格计算
- 2）必要的经常性费用开支

> incl:设备租金、设备折旧费、留守职工工资、水电费、场地租金

> !incl：精神损害、误工费

- 3）only 赔偿直接损失
- 4）返还罚款||没收的金钱

> 4.1)同期银行利息；4.2）存在金融机构里的存款的，合同约定利息；4.3）贷款的，按照贷款利息

### 4.3 赔偿金的支付
赔偿请求人->赔偿义务机关7days内找->财政部门15days 内赔
