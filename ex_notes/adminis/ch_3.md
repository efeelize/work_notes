# II.复议管辖（D 确定级别，地域都是2地管）
# 1 复议维持
## 1） 判断
     1st:if 结果没有改;
                一定是维持
     
     2nd:if 认为 复议请求理由不成立 + 驳回复议请求;
                维持
     
     3rd:if 改了一部分；
                复议决定 维持 && 改变 && 不予受理申请内容 都有；
                改了一部分结果也是维持
             case_2个行为合并执行，改变一部分，维持了一部分，属于复议维持
     
     4th:if 复议机关以违反法定程序为由确认原机关违法;
                视为复议维持：
             cuz 这样原机关可以基于同一事实理由做出同样的裁决，对当事人未提供有效救济
## 2） 审理
     1）告漏了：
        $:先通知追加被告->
        不加的列为共同被告：

        diff:1.1)共同行为告漏了：
                 先通知追加D->
                 不加的列为被告型TP
             1.2)P起诉的D不适格，
                 先告知变更D->
                 不变更的裁驳
     2）管辖
       2.1) 先级别:原机关（% 就低 %）
       2.2）后地域：两地管
     3）起诉期：
        收到复议决定之日15days
     4）举证责任要共担，举证责任可分工
## 3） 判决类型

| 原行为判决 | 复议维持判决 | 
| -------- |  --------- | 
| 不符合受理条件->裁驳  | 复议维持->S（裁定一并驳回对原行为&&复议决定的起诉） | 
| 原行为合法->判驳 | 复议维持合法->S | 
| 原行为合法->判驳 | （***）复议维持程序违法->确认违法(cuz 不然会相当于没有复议过) |
| 作为违法->撤销 + 判决原机关重作or 恢复原行为的法律效力 | 维持违法->S | 
| 不作为违法->履行or 给付 | 维持违法->撤销 |

# IV.国赔
# 1.构成要件
    1）关了打了才要赔
      1.1)人身自由赔偿 only 实际羁押，else 不赔；
      1.2）侵犯人身权->精神损害赔偿
    2）谁主张谁举证；
      exp 被限制人身自由的人死亡||丧失行为能力的
    3）违法归责：
       违反法律规范||从普通人的角度未尽到合理注意义务
    4）赔偿机关：
       最后一个做出错误的法律文书的机关为赔偿义务机关

# 2.赔偿范围
## 2.1赔
    1)错误实际羁押赔偿，重判不赔
      1.1)违法实际羁押；
      1.2）合法拘留+超期羁押（57days）+无罪 == 赔57days；
      1.3）逮捕+撤销案件；
      1.4）逮捕+存疑不起诉；
      1.5）逮捕+判决宣告无罪;
      1.6)数罪并罚，一个无罪，无罪的部分赔偿

## 2.2不赔
    2.1)合法拘留+无罪+合期限羁押（37days）；
    2.2)合法拘留+有罪+超期羁押；
    2.3）逮捕+法定/酌定不起诉；
    2.4）有罪改为无罪，只赔偿判决生效前被实际羁押的部分；
    2.5）无刑事能力的人||免于追究刑事责任的人逮捕羁押 + 判处刑罚 == 之赔偿后面执行刑罚的部分，之前羁押的部分不赔
    2.6）执行行为本身没有错+依据的法律文书错 == 责令返还取得的财产

          30days 内向 该机关的上一级机关申请复议/未告知的2years 内申请复议->
          30days 内向 复议机关所在地 同级法赔委会申请赔偿

    $:1)赔委会的决定第1次就生效，不能上诉，立即生效
      2)赔委会自收到 赔偿申请之日起 3mon 内做出决定；重大疑难复杂的可以延长3mon
      3)不服赔委会的决定，可以向上一级法院的赔委会提出申诉


### 3.2 事实行为侵权：先向赔偿义务机关申请赔偿+单独提起行政赔偿诉讼
     1）行政赔偿诉讼不是行政诉讼

## 4. 内容 
### 4.1 方式
    1）返还财产、恢复原状、金钱赔偿、精神损害抚慰金
    2）垃圾：消除影响、恢复名誉、赔礼道歉

### 4.2 计算标准
#### 4.2.1 侵害人身
    1）限制人身自由：
       误工费->
       上年度->
       做出的赔偿决定内容没有被更改的的上一年度
          like:2017年做出，2018年维持，按照2016年的标准赔偿
    2）造成身体伤害：
       医疗费、护理费、误工费
    3）丧失劳动能力
       only 全残+抚养的无劳动能力的人 == 应当支付生活费；
    4）死亡：
       总额为上年度20倍，if抚养的无劳动能力的人;应当支付生活费
    5)精神损害：
      后果严重的才支付精神损害抚慰金

#### 4.2.2 侵害财产
    1）损失发生时/侵权行为时的价格计算
    2）必要的经常性费用开支
      2.1)incl
          设备租金、设备折旧费、留守职工工资、水电费、场地租金
      2.2)!incl
          精神损害、误工费
    3）only 赔偿直接损失
    4）返还罚款||没收的金钱
      4.1)同期银行利息；
      4.2）存在金融机构里的存款的，合同约定利息；
      4.3）贷款的，按照贷款利息

    2）可以协商
