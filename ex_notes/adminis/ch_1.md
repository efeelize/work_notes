      1.2）辅助性的职位

    2）trait：

               仲裁可以申请法院强制执行

     3）合同要备案：
        同级 公务员主管部门
     4）选拔
       4.1）公开招聘
       4.2）直接选拔

## 2.处分（only6种 + refer在职公务员）
    1）only 警告可以涨工资
    2）警告 6mon，记过12m；记大过18m ；降级24m；撤职24m；开除forever 
    3）免责、引咎辞职不属于处分
       3.1)引咎辞职 是辞去领导身份；
       3.2）责令辞去领导职务，还有公务员的身份
    4）职务、级别、待遇是挂钩的
    5）调任外到内；
       转任内到内；
       挂职锻炼内到外||内
    5）程序
      5.1）初步调查
           1）处分决定机关负责人同意
           2）被调查 + 不宜履行职务 == 暂停履行职务
           3）可以辞去领导职务
      5.2）讨论决定&&通知
           1）集体讨论
           2）书面 + 做出之日生效
           3）if 说+做：减轻
              else 只满足一个;从轻
      5.3）解除（exp 开除）
           1）集体讨论
           2）书面 + 做出之日生效
           3）条件：悔改 + 未再发生危机 + 处分期满

## 3.回避
    1）县乡 级不得衣锦还乡
       exp 自治县
    2）领导 3years||else 2years 内 + 要回避去与原业务有关的再就业
    3) 任职回避 must 上一级

## 4.退出
### 4.1 辞职
      1.1）不满国密期限
      1.2）未达最低服务年限
      1.3）重要公务尚未处理完毕 + 由本人继续处理
      1.4）审计、纪律审查、涉嫌犯罪 + 司法程序未终结

### 4.2 辞退
    ie.辞退不是开除，开除是处分的一种
     4.2.1 辞
           ie.能力不行、态度不好
           1）连续 + 2years 考核不称职
           2）旷工 + 连续15days || 累计30days/year

     4.2.2 ！辞
           1）因公 致残
           2）患病、负伤 + 医疗期内
           3）F + 孕期、产假、哺乳期

## 5.纠纷解决
### 5.1 先复核再申诉
        1）自知道结果30days 内 -> 原机关
        2）不服15days -> 同级 公务员主管部门 || 原机关的上一级机关
           like: 不服区食药局：区公务员主管部门、市食药局、区政府
### 5.2 直接申诉
        1）自知道结果30days 内-> 原机关的上一级机关
        2）60+30 days 内做出申诉决定

## 3.合法
    3.1）滥用职权是行政违法的独立理由：考虑了不相关因素

    3.3）合法侵权补偿，违法侵权赔偿

        2nd:确定力->

        3rd:执行力->

# 2.行政法规(科学立法、民主立法、党的领导)
## 2.1 权限
      1）不得转授权
      2）法律绝对保留事项incl:
           限制人身自由行政处罚、行政强制措施
      3）授权不得超过 5years
      4) 被授权机关在期限届满的6mon 以前报告实施情况 
      5）国务院办公厅：协助总理处理日常事务
      6）标准文本都是 xx公报
## 2.2 制定程序
      1)法规制定的程序较为严格，
         but 对行政法规的解释不用
      2)对法规的解释 只需要 国务院各部门 && 省级人民政府 向国务院 提出解释的要求
### 1) 立项
     1）Only 国务院下属部门：xx部
     2）司法部拟 年度立法工作计划，
        报党中央（***）、国务院 审批
### 2) 起草
    1）起草
      1.1) 重要行政管理的行政法规草案：
           司法部 组织 起草；
      1.2）专业性强的行政法规：
           吸收||委托专家
      1.3）与其他部门紧密相关的规定
           1）充分协商
           2）refer 部门职责分工、行政许可、财政支持、税收优惠
              应当征得 编制、财政、税务 等相关部门同意
    2）听取意见
       2.1) 可以 听证会；
       2.2）草案及其说明应当向社会公布，
            exp 国务院可以决定不公布；
       2.3）向社会征求意见 <30days
       2.4）应当广泛听取意见
    3）咨询论证
       重大利益，应当论证
    4）送审稿的签署
       4.1)由起草部门 + 主要负责人签署；
       4.2）共同起草的由几个部门的负责人共同签署
            cuz 谁签字谁负责
### 3) 审查
    1）可以 + 咨询论证
    2）可以 + 组织听证：
          incl:重大利益调整 、存在重大意见分歧、权利义务重大影响、人民群众普遍关注 
          cuz 会成本较高，效率较低
    3）if 觉得不够火候；
          缓办（搁置） || 退回（退回起草部门）
    4）if 有关部门不同意见；
              1）委托 专家、教学科研单位、社会组织 + 评估
                   if 充分协调不能达成一致意见；
                                       报国务院领导 协调 
              2）报国务院决定
### 4) 决定方式
     1）if 大事；
            国务院常务会议审议：总理、副总理、国务委员、秘书长；
        else if;
            也可以传批，不开会
     2）国务院审批 + 由 司法部||起草部门 说明情况
### 5) 公布
     1)usu：司法部根据国务院审议意见，形成修改稿->
                总理签署国务院令 公布施行：
     2）exp:有关国防建设的军事法令：总理&&军委主席一起签署->
                以国务院、中央军事委员会令 公布
     3）应当公布的载体：国务院公报；中国政府法制信息网；全国范围内发行的报纸
### 6) 施行
     1)usu 公布后 30days
     2)exp:立即实施：
           % 国外货 % 
           国家安全、外汇汇率、货币政策
### 7) 备案
       公布后 30days + 国务院办公厅 + 报全国人常备案
### 8) 立法评估结果只是废止、修改的重要参考
### 9）解释
       if 条文本身；
           省级政府（省长）、部级（部长） 报国务院 解释
       if 具体应用；
           省部级法制部门  报 国务院法制机构 解释
