# ch_2 行政诉讼法

# I.受案范围
## 1.行诉的受案范围
- 1）行诉只审查合法性，不审查合理性
- 2）行政复议两个都审查
- 3）作为行政行为依据+红头文件既可以提行政诉讼，也可以提行政复议：like:县政府发布全县征地补偿安置标准的文件

> 红头文件：国务院 && 各级政府工作部门就可以起草；开会就可以出台；名字很多：会议纪要、通知、操作指南、指示

> 规章：地规由省级||设区的市政府制定；经过一系列的程序才可以制定；名称为规定、办法、细则

- 4）行政给付可诉：救命钱
- 5）具体行政行为可诉

> 5.1)以行为作出时为判断标准。like:这条街上违章停车一律罚款：作出时可以确定范围；5.2）必须同时满足主观 + 客观 两个因素：否则申请国赔

## 2. 不可诉的行政行为：不产生权利的增减
- 1）公安局违约：民事行为
- 2）公安局不立案侦查：刑事司法行为
- 3）仲裁结果：仲裁是双方自愿的行为
- 4）信访办理行为：信访机构对材料进行登记、移交、转送、接受的
- 5）维持决定书：重复处理行为，内容上没有进行更改的2次决定
- 6）交通处罚通知单/告知书、材料补正通知单：过程性行为/阶段性行为 -> 对最终的权利义务没有进行安排
- 7）政府建议种植水稻：行政指导行为
- 8）国务院发布的具有普遍约束力的决定、命令
- 9）行政法规、行政规章：cuz 属于立法行为
- 10）结婚登记、不动产物权登记、预告登记：影响了当事人实施民事活动的法律效力

> if 登记行为不影响；不可诉。like:房屋购买合同去房管局备案

- 10）复议终局

## 3.行政协议
### 3.1 解题思路
- 1st:是行政行为||民事行为
- 2nd:官告民||民告官
- 3rd:诉行政机关违约||单方变更解除合同 -> 起诉期间、诉讼费用不同

> 1）民诉：诉行政机关违约 -> 3year 诉讼时效 -> 按照标的额收取费用
> 2）行诉：诉行政机关单方变更解除合同 -> 6mon -> usu 一件50

### 3.2 trifle
- 1）行政协议不能在合同中约定管辖法院，民诉可以
- 2）class:2.1）政府特许经营协议；2.2）土地、房屋等征收征用补偿协议
- 3）判决：P 主张不成立，驳回；P 请求解除||确认无效；解除协议||确认无效判决

## 4.规范性行政文件的附带审查
- 1）审查中发现可能不合法的：应当听取+制定机关意见
- 2）处理结果

> 2.1) 谁做出生效裁判找谁：二审维持->找二审法院；2.2）法院自裁判生效之日起3mon 向制定机关提出处理建议 || 抄送制定机关的：同级人民政府、上一级行政机关、监察机关、规范性文件的备案机关；2.3）联合制定的规范性文件：法院发送司法建议->规范文件的主办机关、共同上一级行政机关

- 3）对司法建议的处理

> 3.1)自接到司法建议之日起 60days内 + 书面答复 ；3.2）情况紧急可以建议立即停止执行：制定机关||上一级行政机关

## 5.行政复议 && 行政诉讼
### 5.1 自由选择
- 1）if 同时两个都提的：先立案的-> 当事人自由选择
- 2）复议结果没有出来不允许诉讼：已经申请复议的，在法定复议期间又向法院提出诉讼的，法院裁定不予立案

### 5.2 复议前置但不终局
- 1）纳税争议：交不交、交多少、谁来交、计算问题

> 有关纳税的行政处罚、行政强制措施、执行措施、滞纳金 可以直接诉

- 2）行政确权 + 已经取得 + 自然资源权利

> case_1)甲村土地确权；2）市政府说归甲；3）乙向省政府复议 

- 3）禁止 && 限制经营者集中：like 垄断

### 5.3 复议终局
- 1）出入境公安局针对外国人做出的相关决定

> incl:继续盘问、拘留审查、限制活动范围、遣送出境

- 2）国务院的复议决定：对国务院部门||省级政府的行为不服的

> case_2.1）不服上海市政府的决定：或诉或复；2.2）if 复议；先找自己->上海市政府 2.3） if 对上海市政府的复议不服的;或诉或复；2.4）if 选择第二次复议->国务院作出的复议决定终局

- 3）省级人民政府的自然资源权属复议决定

> 3.1)行为主体：省政府；3.2）行为性质：复议决定；3.3）行为内容：确认了自然资源的所有权||使用权；3.4）行为依据：根据国务院||省政府对行政区划的勘定、调整、征用土地的决定 

## 6.行政公益诉讼
### 6.1 范围
- 1）环保
- 2）食药安全
- 3）国有财产保护
- 4）国有土地出让

### 6.2 条件
- 1）损害结果：损害了国家利益||公共利益
- 2）诉前程序：检察院向行政机关提出了检察建议，否则不可以直接起诉

> 2.1)行政机关收到之日起 2mon内履行 + 书面回复; 2.2)exp紧急情况：15days 内回复

- 3）管辖法院：县检->县法

### 6.3 诉中被告改了
- 1）检院撤诉->应当裁定准许
- 2）检院不撤诉+被告改行为为合法了 == 确认原行为违法

### 6.3 检院不服一审
- 1）可以 + 上一级 + 上诉
- 2）谁提公益诉讼谁出庭，上一级检院打辅助：派员参加

## 2.D **（主+客）

> way:1)if 复议维持；2D;审2个行为；举证2个行为；判2个行为；2）if 复议改变：1D;审1个行为；举证1个行为；判1个行为

### 2.1 复议维持
#### 2.1.1 判断
- 1）认为复议请求不成立，做出驳回复议请求的
- 2）结果没有改一定是维持
- 3）复议决定 维持&&改变&&不予受理申请内容 都有；改了一部分结果也是维持

> case_2个行为合并执行，改变一部分，维持了一部分，属于复议维持

- 4）复议机关以违反法定程序为由确认原机关违法的，视为复议维持：cuz 这样原机关可以基于同一事实理由做出同样的裁决，对当事人未提供有效救济

#### 2.1.2 审理
- 1）告漏了：先通知追加被告->不加的列为共同被告：cuz 逼着行政机关做出复议改变

> 共同行为告漏了：先通知追加被告->不加的列为被告型第三人

> 原告起诉的被告不适格，先告知变更被告->不变更的裁定驳回起诉

- 2）管辖

> 2.1) 先级别->原机关：2.2）后地域：两地管

- 3）起诉期：收到复议决定之日15days
- 4）举证责任要共担，举证责任可分工

#### 2.1.3 判决类型
- 1）if 判决撤销原行为 && 复议决定；判决原机关重作
- 2）if 原行为合法，复议违法;撤销复议决定||确认复议决定违法 + 驳回原告对原行为的诉讼请求

> like:适用法律错误

- 3）if 原行为不符合受案范围；裁定一并驳回对原行为&&复议决定的起诉

### 2.2 复议改变
- 1）管辖

> 1.1)先级别：复议机关 1.2）后地域：两地管

- 2） 判决类型

> 2.1) 复议决定合法；驳回原告诉讼请求；2.2）原行为违法+复议改变错误 == 撤销复议决定 + 可以责令复议机关重作；2.3）原行为合法+复议改变错误 == 撤销复议决定 + 恢复原行为

### 2.3 复议不作为
- 1）以申请人的复议不符合受理条件为由驳回复议申请
- 2）不服谁告谁，按照谁的标准确定地域管辖和级别管辖

### 2.4 其他案件中的D
#### 2.4.1 开发区管理机构
- 1）国务院、省政府批准设立的开发区管理机构->告自己：cuz alike 政府
- 2）国务院、省政府批准设立的开发区管理机构的 职能部门->职能部门
- 3）法律法规授权的->开发区管理机构
- 4）无授权->无主体资格->设立该机构的地方政府

#### 2.4.2 经批准
- 1）诉讼：谁署名谁被告
- 2）复议：谁批准谁被告：cuz 复议要找到上一级领导

# IV.程序
## 1.起诉
### 1.1 起诉期限，单行法另有规定除外
#### 1.1.1 诉作为

> 1）全不知（like 行政决定书不送达相关人）：1.1）自实际知道之日起6mon;1.2）自行为作出之日起<5years;不动产案件<20years

> 2）全知道（内容、相对人、诉权、期限）：自实际知道之日起6mon

> 3）知一半（不知诉权、期限）：自知||应知起诉期限之日起，< 1year

> case_1）2018.1.1环保局发给排污证；2）2018.6.1造纸厂动工毒死了养殖户的鱼；3）2018.6.1起算养殖户的诉讼时效


#### 1.1.2 诉不作为

> 1）有履行期限，届满之后6mon

> 2）无履行期限：申请满2mon后+6mon

> 3）紧急情况：即时起诉，6mon

### 1.2 起诉方式
- 1）可以口头
- 2）一律接收无 exp + 出具书面凭证注明日期

> 起诉不符合条件的：一律接收+当场裁定不予受理

## 2.立案
- 1）当场不能判断是否符合起诉条件；7days 内决定 ->else 应当+先予立案
- 2）材料欠缺的一次性告知、应当指导释明：cuz 尽量避免不立案，提高立案率好交差
- 3）当事人拒绝补||补了也不符合起诉条件 -> 退回 && 记录在册
- 4）if 3)+坚持起诉 ->裁定不予立案+载明不予立案的理由 
- 5）救济：

> 5.1)if裁定不予立案：10days 内向上一级法 上诉；

> 5.2)if不当行为向 上级法 投诉：cuz涉及的是违纪行为->like 不收起诉状、不出具书面凭证、不一次性告知等

> 5.3)if既不立案，也不出具不予立案的裁定；向上一级法 起诉->上一级法自己审||交除了原法院的下级法审理

## 3.审理
### 3.1 一审普通
- 1）商秘+当事人申请 == 不公开审理
- 2）证人、鉴定人、勘验人、翻译->应当用通知书
- 3）可以延期
- 4）可以合并
- 5）法院对回避申请3days内+书面||口头；明显不属于的当庭驳回
- 6）对回避申请不服，可以向作出决定的法院申请 复议一次，复议期间不停止工作的审理
- 7）对复议维持案件：审理被诉行政行为的合法性+复议程序的合法性
- 8）1w 以下的罚款||15days 以下的拘留 + 院长批准 + 单独||合并适用 == 不服可以申请复议一次
- 9）审限为立案之日起6mon 

### 3.2 一审简易
- 1）法定简易

> 1.1)当场作出的行政行为；1.2）2k 以下；1.3）政府信息公开

- 2）各方可以约定简易
- 3）！incl:发回重审、再审
- 4）举证期限：法院定||当事人协商<15days
- 5) 1个审判员在45days内审结
- 6）简易转普通：审限届满前+书面通知双方

### 3.3 二审
- 1）上诉期:判决15days;裁定10days
- 2）例外书面审：无新的事实证据+合议庭认为不需要开庭
- 3）全面审查
- 4）审限是收到上诉状3mon 内
- 5）通过原审法院上诉||直接向二审法院上诉

### 3.4 再审
- 1）申请期限：在文书发生效力后6mon
- 2）审限：自再审申请案件立案之日起6mon 内审查，exp院长批准可以延长

### 3.5 行政机关负责人出庭应诉
- 1）至少来个工作人员+加盖行政机关印章的授权委托书

> if 负责人不来||只来了一个律师：法院记录在案+在裁判文书中载明+向任免机关、监察机关、上一级行政机关提出 给予处分的司法建议

- 2）负责人有正当理由：应当向法院提交情况说明+盖章/签字；if 不想说理由：案件继续审理+法院向监察机关、上一级行政机关 提出司法建议
- 3）行政机关负责人 incl:其他参与分管的负责人
- 4）行政机关工作人员 incl:地方政府所属法制工作机构的工作人员、具体承办机关的工作人员

### 3.6 行诉中对民事争议的例外：可以一并审理
#### 3.6.1 前提
- 1）only 许可、登记、征收、征用、裁决
- 2）!incl:

> 2.1)法律规定需要对行政先行处理：like 土地权属纠纷；2.2）违反了民事专属||协议管辖：like 合同约定的管辖法院与行诉法院不同；2.3）已经申请仲裁||民诉；

- 3）if 不予准许一并审理，可以申请复议一次

#### 3.6.2 审理
- 1）有正当理由的，在法庭调查中提
- 2）usu民事争议单独立案，同一组织审理；exp行政裁决案件本身就是对民事争议做出的裁决，可以不用另行立案
- 3）无论怎样，分别裁判
- 4）if只撤诉一个的;另一个继续审理
- 5）if一个超过起诉期，另一个还没有立案的；另诉
- 6）if一个超过起诉期，另一个立案的；原审判组织继续审理
- 7）if 只诉一部分，全部案件移送，全面审查
- 8）if 二审发现未上诉部分有错的，按照再审审理
- 9）诉讼费分别收取
- 10）对民事部分进行了调解，调解的结果不得作为行政部分的证据->行政行为的证据原则上要在做出行政决定之前收集

### 3.7 调解
- 1）incl：钱多钱少（赔偿、补偿）||自由裁量的案件
- 2）调解过程不公开，exp 当事人同意

### 3.8 trifle
- 1）P 撤诉+法院不准+拒不到庭||中途退庭 == 缺席判
- 2）辩论终结后+P撤诉==可以+准许；exp 涉及国家||公共利益
- 3）if D不来||走掉；公告||向检察机关、上一级行政机关 提出处分的司法建议
- 4）if third party 不来||走掉；无所谓
- 5）D诉讼中改变了原行为

> 5.1)书面告知法院；5.2）if D 改变为合法行为||作为；P 仍然要求确认原行为违法||不撤诉->做确认判决；5.3）if P 对新的判决还是不服->再一次提起新的诉讼

- 6）行诉中发现了犯罪材料，立即移送+不影响行政审理。cuz 否则会串供、逃跑

### 3.9 保全 alike 民诉
- 1）诉中：48h 内裁定；可以申请复议；法院可以责令提供担保
- 2）诉前：48h 内裁定；可以申请复议；应当提供担保；

### 3.10 先予执行 alike 民诉
- 1）incl:救命钱+权利义务关系明确的
- 2）可以复议 一次

## 4.证据
### 4.1 举证 **（客）
- 1）P prove

> incl:1.1）有原告资格；1.2）诉不作为证明申请过，exp 依职权的行为||D 的原因 P 不能提供： like 行政机关应该登记但是没有登记；1.3）损害证明，exp D导致P举证不能的，D 举证

- 2）举证期限:受到起诉状副本15days 内
- 3）延期举证：

> 3.1)P:受到起诉状副本15days内+书面；3.2)D/third party：法庭准许->法庭调查中提

- 4）延长举证期限：在举证期限届满前提出
- 5）案情复杂||证据较多 == 可以+庭前证据交换
- 6）涉密的证据要质证，but 不得在开庭时公开质证
- 7）依申请的证据由当事人质证，依职权调取的证据由法庭说明+听取当事人意见
- 8）未采纳的证据应当在裁判文书中说明理由
- 9）证人的交通费、住宿、就餐、误工费，由败诉方承担

### 4.2 行政赔偿、补偿案件中损害事实的分配责任 **（主）
- 1）usu P 对自己损害举证
- 2）exp D 对P的损害举证

> 2.1)无法认定损失的价值的，负有举证责任的一方申请鉴定；exp 行政行为做出时就应该评估的；2.2）应该负责鉴定但不鉴定的，承担不利后果；2.3）P 坐地起价的：当事人的损失因客观原因无法鉴定的，法院应当结合当事人的主张和在案证据，遵循法官职业道德，运用逻辑推理和生活经验、生活常识等，酌情确定赔偿数额。

## 5.判决

### 5.1 一审判决
#### 5.1.1 撤销判决
- 1）被告行政行为违法
- 2）原告起诉请求撤销，法院审查认为无效，判决无效

#### 5.1.2 履行判决
- 1）履行有意义的，判决履行
- 2）尚需调查||裁量的，判决被告针对原告的请求重新处理

#### 5.1.3 给付判决
- 1）抚恤金、最低生活保障金、工伤、医疗社会保险金
- 2）P 未先向行政机关提出给付请求&& 法院认为明显不属于行政机关权限范围的，裁定驳回起诉

#### 5.1.4 变更判决
- 1）行政处罚显失公正
- 2）其他行为款额有误
- 3）usu 不能加重；exp 利害关系人同为 p + 诉讼请求相反 ：like 2人打架，1人不服

#### 5.1.5 确认判决
##### 5.1.5.1 确认违法
- 1）违法+撤销损害国家、公共利益
- 2）程序轻微违法+对原告实际权力不产生影响：like 未通知、送达、处理期限的问题
- 3）违法+不具有可撤销内容：like 房子已经拆了
- 4）D 已经改了+P 仍然要求确认违法
- 5）履行判决已经没有意义

##### 5.1.5.2 确认无效

> 前提：行为无效->无主体资格、无法律依据、内容在客观上不可能实现

- 1）请求撤销但是超过法定起诉期的，裁定+驳回起诉
- 2）拒绝变更诉讼请求+坚持要求确认无效，判决+驳回诉讼请求

### 5.2 二审判决
#### 5.2.1 维持

#### 5.2.2 改判
- 1）一审事实不清，或改或发

#### 5.2.3 发回
- 1）一审程序严重违法
- 2）一审漏了必要当事人
- 3）一审漏了必须判决的诉讼请求
- 4）一审漏了赔偿请求，二审认为不予赔偿，判决驳回赔偿请求
- 5）一审漏了赔偿请求，二审认为应该赔偿，1st 调解；2nd 赔偿部分发回
- 6）二审提出新的赔偿请求，1st 调解；2nd 告知另诉
- 7）只能发回一次：二审法院不得再次发回重审

## 6.执行
- 1）申请执行的期限为2years
- 2）行政机关作出裁决后，权利人可以在6mon 内申请强制执行
- 3）执行中的财产保全：应当提供担保
- 4）D 不履行的，从期满之日起，对行政机关 负责人 50-100/day
- 5) 社会影响恶劣+不履行 == 可以司法拘留
