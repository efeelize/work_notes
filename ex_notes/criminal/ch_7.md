# 2.行贿受贿

> 主观上都是为了谋取不正当利益。ie.将不确定的利益变为确定性利益

| 行贿人 | 受贿人 | 行贿人罪名 | 受贿人罪名 |
| ----- | ----- | -------- | -------- | 
| only 自然人 | 国工 | 行贿 | 受贿 |
| 单位（公司、企业、事业单位、机关、团体） |  国工 |  1）单位行贿；2）if 利益归了个人，则转化为行贿罪  |  受贿 |
| ALL | 1）国有单位（国机、国有公司、企业、事业单位、人民团体）；2）！incl:社会团体、民营企业 | 对单位行贿罪 | 1）单位受贿罪，必须为他人谋取利益+情节严重；2）单位在账外暗中收取回扣、手续费的|
| ALL | 1）公司、企业、其他单位的工作人员/非国家工作人员；2）Only refer 上班族 | 对非国家工作人员行贿罪 | 非国家工作人员受贿罪 |
| ALL | 1)国工近亲属||关系密切人；2）离职国工；3）离职国工的近亲属||关系密切人 | 对有影响力的人行贿罪 | 利用影响力受贿罪 |

## 2.1 行贿
       1）if 为了谋取正当利益；only 受贿罪
       2）if 被索贿+ ！获得不正当利益；！行贿
             else if 获得了；行贿
       3）只要将财物置于国工的实际控制范围，就完成行贿的行为
       4）自首
          4.1）if 行贿人在被追诉前 + 主动交代; 可以从减
                  if 重大立功表现的；可以减免
          4.2）if 出卖受贿人；成立自首
                  if 揭发他人犯罪，不在成立立功
                     cuz 已经评价为自首
          4.3）if 行贿人自首后 + 受贿人主动投案；坦白 

## 2.2 受贿
       ie.权钱交易
       1）自首
          前提：一开始就没有受贿的故意
          及时退还||上交；不是受贿
       2）罪数
          usu 并罚
          exp:从一重罪
              徇私枉法、民事行政枉法裁判、执行判决裁定失职罪、执行判决裁定滥用职权
       3）为他人谋取利益
          3.1）实际||承诺 
          3.2）明知他人请托事项
          3.3）事后基于该履职事由 收受他人财物

### 2.2.1 行为
          1）索贿
             ！care 是否为他人谋取利益
          2）受贿
             care:为他人谋利的意思
             ！care:明示默示、真实虚假、是否正当、是否实现、收钱先后
          3）经济受贿
             收受回扣、手续费 + 归个人所有
          4）斡旋受贿
            notes:!incl:单位
            case_甲让乙给丙办事
                 4.1）甲利用的是面子
                 4.2）乙利用的是职权
                 4.3）乙对甲拿钱不知情
                        else if 知情；甲乙都构成受贿的共犯
          5）事先约定+离退休后 收受

## 2.3 利用影响力受贿
       1）前提是国工不知情；else 受贿共犯
       2）if 行贿人是概括的故意；！care 谁拿钱
             if 国工拿钱；行贿->受贿
             if 国工身边人拿钱；对有影响力的人行贿->利用影响力受贿

## 2.4 医院、学校
       if 职务行为；
          if 在公办；受贿
          if 在私营；非国家工作人员受贿
       if 业务行为；
          all 非国家工作人员受贿
       if 收病人家属的红包；法律不调整

## 2.5 为亲友非法牟利罪
       1）主体
          国有公司、企业、事业单位的工作人员
       2）客观方面
          2.1）利用职务便利
          2.2）实施以下三种行为之一
               1）把本单位的盈利业务交给亲友经营
               2）高价从亲友那里进货 || 低价向亲友销售
               3）从亲友那里买 不合格的产品
          2.3）使国家利益遭受到了重大损失

## 2.6 赃款赃物的评价问题
       version 赃款赃物 没有返还请求权
       version 赃款赃物 属于普通财产

                like:不付嫖资
             if else 放弃财物；诈
       4) 受贿后事没有办成+不退钱；
