                甲只是单纯的发现

### 1.3 结果
#### 1.3.1 class 
    1)结果犯/侵害犯
      like:生产销售类：
           劣药；不符合安全标准产品；伪劣农药、兽药、化肥、种子；不符合安全标准化妆品

    2）危险犯
      2.1）抽象危险犯/行为犯
           incl:生产销售假药、有毒有害食品

      2.2）具体危险犯
           incl:破坏交通工具罪
                放火、决水、爆炸、投放危险物质、以危险方法危害公共安全
                破坏电力设备、破坏易燃易爆设备罪

      2.3）判断
           if 刑法分则条文只是单纯规定了某种犯罪行为，没有对危险状态做出具体规定；行为犯
              if else; 具体危险犯
           like:...，危害公共安全 || 足以..... ->具体危险犯

#### 1.3.2 结果加重犯
     1）对加重结果可以是故意||过失
        like:1.1）抢劫致人重伤、死亡，行为人对重伤、死亡既可以是故意，也可以是过失
             1.2）故意伤害致人重伤，行为人对重伤既可以是故意，也可以是过失

     2）对加重结果 only 过失
       like:故意伤害致人死亡，对死亡只能是过失；
            else 故杀

     3）盗窃
       if 想偷较大数额，实际投了特别巨大的数额
          按照实际的数额量刑，对加重结果无需故意

     4）拐卖妇女儿童
        only 造成被拐卖的妇女、儿童或者其他亲属重伤、死亡或者其他严重后果的 是结果加重犯
        else 都是情节加重犯

     5）Only 改变量刑，不改变罪数
        5.1）incl
             污染环境罪、非法行医致人死亡
        5.2）!incl:
             遗弃、侮辱、诽谤、强制猥亵、盗窃、诈骗、绑架致人死亡
                  cuz:基本犯罪行为具有造成加重结果的高度可能性

### 1.4 因果关系
    1）if 介入行为不中断 + 两个行为都有因果关系 ; 均有关
          like:打到濒死的状态+医生一般过失
       if 没有联络+分别都能独立导致结果的发生 ; 均有关
          like:甲乙碰巧同时开枪，均打中乙,击中心脏死亡
       if 没有联络+单独不都 导致结果的发生 ; 均有关
          like:甲乙碰巧同时开枪，均未打中要害,乙失血过多死亡
    
          case_甲把乙从楼顶推下，丙正好朝窗外开枪，乙被打死
          analyze:甲属于故杀未遂->
                  因果关系中断，不再向前追溯->
                  丙属于意外事件

    2）自杀不中断因果关系 的exp
       ie.导致生不如死
       2.1）虐待
       2.2）暴力干涉婚姻自由
       like:甲向重病的乙投毒，乙喝完之后觉得很难受自杀；中断

    3）特殊体质不中断
      3.1）like：
           3.1.1）甲打中乙的胸部，乙是血友病患者，失血过多死亡
           3.1.2）甲黑夜中朝乙射击，乙有心脏病被吓死
                  甲均成立故杀既遂
      3.2）diff：
           3.2.1）甲推到乙，乙后脑勺撞在地上，颅脑损伤，抢救无效死亡
                  甲应当预见到推到会致使碰到地上的石头等；主观上有过失；成立过失死
           3.2.2）甲与不认识的乙发生口角，推了乙的肩膀一下，乙突然心脏病发作死亡
                  甲对乙的心脏病无认识可能性，主观上无过失，属于意外事件


    4）自然事件中断
       like:伤害被害人后送去医院+医院发生火灾

> 故意、过失都是针对结果来讲的

### 1.1 故意

             2)甲心想：丙你就看运气吧

          2.2）数额较大
               ie.对于其他的事情是否明知不影响盗窃罪的成立

### 1.2 过失
    1）过于自信的过失
      1.1）采取措施+有效果
      1.2）基于经验、技术

    2）过失犯罪，应当预见的结果是该罪的构成要件所规定的具体的特定的结果
       like 过失致人死亡、过失致人重伤应当预见重伤/既有重伤也有死亡；else 意外事件

    3）共同犯罪不承认共同过失犯罪，分别认定各自的犯罪，但是要证明因果关系
       3.1)if 甲乙以为是兔子但是打到了人+无法区分是谁干的
              甲乙不构成犯罪
       3.2）？？？其他情形

### 1.3 事实认识错误
#### 1.3.1 同一犯罪构成间

     1）打击错—>具体符合说 && 法定符合说
        case_1.1）具体：甲对乙是故杀未遂，对丙是过失死，现象竞合
             1.2）法定：甲对乙是故杀未遂，对丙是故杀既遂，一个行为，属于故杀既遂
        $ : 对象错误两种学说的结论是相同的

     2）因果关系错误
       2.1）狭义因果关系错误
            like:甲本来想淹死乙，但是推入井中摔死了乙
            $:不影响犯罪既遂的成立

       2.2）事前故意/犯罪构成的推后实现
            case_甲对乙先实施了第一个行为A，以为其已经死亡，为了毁尸灭迹，实施了第二个行为 B,B 导致了乙的死亡
            version 1）：if实施 B 行为的时候;对死亡持有间接故意，整体上成立一个故杀既遂.if 实施 B 行为的时候，相信死亡结果已经发生，成立故杀未遂与过失致人死亡
            version 2）将两个行为视为一个整体，将支配行为视为概括的故意，只成立一个故杀既遂

       2.3）犯罪构成的提前实现
            depends 实施A 行为是否已经着手
            case_毒咖啡案：
                 2.3.1）乙固定在2h 后才回家，甲成立故杀预备未遂+过失致死的既遂 想象竞合
                 2.3.2）乙随时可能回家，甲故杀既遂

#### 1.3.2 不同犯罪构成间
> 没有因果关系的错误

    1）对象错误
     1.1）usu:在主客观重合的范围里成立犯罪

                 like:盗窃 && 侵占

     2）打击错误
       depends 是否注意
               if Y;间接故意
                 else ; 打击错误
       2.1）esp:主观 && 客观之间 没有重合的部分->A 未遂+B既遂 的想象竞合

# III.Trifle
## 1.刑法的解释
     1）强奸中的暴力、胁迫 alike 猥亵
     2）非法经营罪中的买卖:做买卖
     3）使用假币指的是伪造的货币
        !incl:变造的货币
     4）未经著作权人许可大量播放其音像制品:侵犯机械表演权
        发行：非法制作侵犯著作权的产品，通过出售、出租、出借、散发等方式提供复制品的行为
             ie.发行是通过一定载体向公众传播侵权复制品
     5）普通人轻罪可以不追究
     6）if 外国人在国外打中国人;
           3年以上+双重犯罪 == 可以追究
     7）规范的构成要件
        ie.专业的法律概念
        以行为人在实施其行为时说认识到的一般人的判断结论，认定行为人是否有故意
           ie.if 只要行为人认识到一般人会认为其贩卖的是淫秽物品 + 事实上是淫秽物品；
                 认定其认识到了
     8)不真正不作为犯
       ie.通常要作为实现，但是不作为也可以实现
          like:放火、故杀
