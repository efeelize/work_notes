## 1.1 主体
###   1）不真正身份犯/加减身份
> ie.不影响定罪量刑的身份
         刑法中关于加减身份的规定不适用于不具有加减身份的人
             like:非法拘禁，国工利用职权的，从中触犯

###   2) 单位犯罪
> depends:名利双收

####  2.1）！incl:
#####     1) 贷款诈骗、信用卡诈骗、有价证券诈骗
             if 单位实施这些犯罪；
                   直接追究直接责任人贷款诈骗的责任
#####     2）不具有法人资格的独资、私营企业、事业单位
#####     3）以单位名义+实施自然人犯罪
#####     4）管家分钱：总裁、领导

####  2.2）incl:
#####     1) 拒不执行判决裁定罪、虐待被监护人、看护人罪
#####     2）分支机构以单位名义+违法所得归分支机构 
               like:后勤处干的
#####     3）if 单位被吊销、破产;like:保险诈骗罪
                only 追究自然人的责任，仍然是单位犯罪
#####     4) if 单位合并；
                P 列原单位，双罚
#####     5）行政机关、法院、军事等
#####     6）东家分钱
            like:股东集体讨论决定，钱分给了股东

#### 2.3）trait:
         1)单位犯罪的，单位 && 直接责任人 之间不成立共犯
              cuz:人格不独立
         2）单位的主要责任人员之间 可以成立共犯
         3）双罚制：对单位only 适用罚金
            单罚制：only 罚自然人

## 1.2 作为、对象
> 作为、不作为与犯罪性质没有必然的联系，刑法分则各章中都存在不作为犯罪

### 1.2.1 不作为义务

     notes:1)作为、不作为评价的是实行行为
           ！incl:预备行为
           2)作为||不作为 的故杀 only 在构成要件符合性的层面上 diff 
           3)属于作为犯
             3.1)持有型犯罪
                    cuz 禁止人们持有特定的物品（违反了法律的禁止性规范）
             3.2)侵占
                    cuz  非法占为己有，拒不返还是一个意思
           4）不纯正不作为犯
              only 是间接故意
                   like:重大责任事故罪是过失犯罪
           5）不真正不作为犯
              其能够评价为犯罪在于作为和不作为在造成危害结果上有 等值性

#### 1）有义务
        1）遗弃罪的不作为来源于：不履行抚养义务
        2）出租车 && 乘客：不作为的强奸的帮助犯
        3）共同危险行为有相互救助的义务
           like 探险
           unless 特别说明
        4）合同即使过期，人身安全保障义务也不能免除
        5）紧急避险人 在危险消除后 对第三人
        6）特殊防卫+侵害结束+有死亡的紧迫危险

#### 2）！义务
       1）成年人出去玩（incl 漂流）+是当场唯一能救的人 == 没有义务
           unless：唯一 + 可控制的范围内
       2）喝酒
           like:甲乙深夜喝酒，甲醉后，钱包掉了出来。乙拾得。甲让乙送自己回家，乙拒绝。
                甲在回家途中醉倒冻死
                analyze:乙对甲的死亡不构成不作为的故意杀人
       3）单纯发现了危险：
           like:在深山里发现了弃婴
                if 乙掉进河里，甲虽然发现，但误以为是别人家的小孩，没有救助导致乙死亡；
                        成立不作为的故杀
                          cuz only 是对自己有无救助义务产生了错误认识，属于违法性认识错误
       4）内心没有承担风险的意思
           like:把弃婴报到了加油站||福利院||医院门口

#### 3）共犯问题
#####   1）甲投毒，乙看见了不阻止，乙成立不作为的教唆犯
#####   2）甲要救被自己撞下水的人，乙劝阻甲救人，乙成立不作为犯罪的教唆犯
#####   3）if 甲以强奸的故意致丙昏迷实施奸淫，随后乙来到也进行了奸淫+窃取财物
               甲承担轮奸+盗窃
#####   4）if 甲乙共同入户抢劫，甲捆绑丙实施奸淫，乙在旁边拿财物;乙也成立强奸
#####   5）if 甲乙共谋入户盗窃 + 分别在不同的房间找东西 + 乙发现了被害人丙准备杀害 + 甲知道没有阻止；甲只承担盗窃的责任
> cuz 盗窃不会使人有生命危险->甲只是单纯的发现
#####  6）明知他人实施重婚而提供帮助的，成立重婚的帮助犯，二者之间成立共同犯罪

#### 4）不作为犯罪必须有结果回避的可能性
> ie.不履行义务的行为和后果之间要有因果关系
           like:甲伤害乙后逃逸，事后证明即使送医，甲也不可能存活
                 甲不成立不作为犯，其事后逃跑的行为没有创造不被允许的风险

              if else;

### 1.3.2 结果加重犯的要件
> 1)轻罪无结果加重犯；2）罪名变化不是；3）没有因果关系不是

#### 1）基本行为导致，有因果关系
#### 2）主观
            2.1）对加重结果可以是故意||过失
                  like:1）抢劫致人重伤、死亡，行为人对重伤、死亡既可以是故意，也可以是过失
                       2）故意伤害致人重伤，行为人对重伤既可以是故意，也可以是过失
            2.2）对加重结果 only 过失
                 like:故意伤害致人死亡，对死亡只能是过失；else 故杀
            2.3）盗窃
                 if 想偷较大数额，实际偷到了特别巨大的数额；
                        按照实际的数额量刑，对加重结果无需故意 
#### 3）基于法律规定
           Only 改变量刑，不改变罪数
           1）incl：
              污染环境罪、非法行医致人死亡
           2）!incl:
              遗弃、侮辱、诽谤、强制猥亵、盗窃、诈骗、绑架致人死亡
                         cuz:基本犯罪行为具有造成加重结果的高度可能性
           3）拐卖妇女儿童
              only 造成被拐卖的妇女、儿童或者其他亲属重伤、死亡或者其他严重后果的 是结果加重犯
              else 都是情节加重犯  
           4）绑架致使被害人死亡
              ！结果加重犯
              以绑架+过失死 == 想象竞合  
           5）强奸多人的、轮奸的 是情节加重犯  

## 1.4 因果关系
        notes:有...的目的||动机，就有...的因果关系 （False）
                cuz:目的||动机 属于主观判断、是责任要件
                    因果关系 属于客观判断、是违法构成要件

        顺序：if 两个人同时；
                if 任何一个行为都能导致结果的发生；二重因果关系
                      ie.if 没有联络+分别都能独立导致结果的发生 ; 均有关
                             like:甲乙碰巧同时开枪，均打中乙,击中心脏死亡
                
                if 结合才能导致结果的发生；重叠的因果关系
                      ie.if 没有联络+单独不都 导致结果的发生 ; 均有关
                             like:甲乙碰巧同时开枪，均未打中要害,乙失血过多死亡
              
              else if 有先后；介入因素判断
                              if 中断因果关系；条件说失效，禁止溯及前行为
                                 case_甲把乙从楼顶推下，丙正好朝窗外开枪，乙被打死
                                      cuz:甲属于故杀未遂->
                                          因果关系中断，不再向前追溯->
                                          丙属于意外事件
                              if 介入行为不中断 + 两个行为都有因果关系 ; 均有关
                                     like:打到濒死的状态+医生一般过失

             if 存疑有利于被告
                if 过失犯罪；无罪
                if 故意犯罪；未遂
                   notes:共同犯罪不承认共同过失犯罪，分别认定各自的犯罪，但是要证明因果关系
                   case_1)if 甲乙以为是兔子但是打到了人+ 同时开枪+ 只有一个弹孔 + 无法区分是谁干的
                             甲乙不构成犯罪;
                             cuz无法证明各自的因果关系
                        2)if 甲乙以为是兔子但是打到了人+ 同时开枪+ 只有一个弹孔 + 该弹孔有两颗子弹穿过
                             甲乙分别过失死；
                             cuz 有因果关系，不承认共同过失，按照各自的责任处罚

### 1.4.1 介入因素的判断
    1）if 自然事件;中断
          like:伤害被害人后送去医院+医院发生火灾     

    2）自杀中断因果关系 
       ie.导致生不如死
       2.1）虐待
       2.2）暴力干涉婚姻自由
       like:甲向重病的乙投毒，乙喝完之后觉得很难受自杀；中断

    3）特殊体质不中断
      3.1）like：
           3.1.1）甲打中乙的胸部，乙是血友病患者，失血过多死亡
           3.1.2）甲黑夜中朝乙射击，乙有心脏病被吓死
                  甲均成立故杀既遂
      3.2）diff：
           3.2.1）甲推到乙，乙后脑勺撞在地上，颅脑损伤，抢救无效死亡
                  甲应当预见到推到会致使碰到地上的石头等；主观上有过失；成立过失死
           3.2.2）甲与 不认识的乙 发生口角，推了乙的肩膀一下，乙突然心脏病发作死亡
                  甲对乙的心脏病无认识可能性，主观上无过失，属于意外事件

## 2.违法阻却事由

| items |  正当防卫 | 紧急避险  |
| ----- | -------- | -------- |
| 学说 | 防卫意识：认识到不法侵害正在进行（防卫意图） | usu 需要 避险意识->行为无价值理论->善意善果 |
| 特殊 | 针对的不是犯罪行为的，不属于正当防卫的范围，不涉及正当防卫的问题:like 两车并行 | 下手时间可以早于正当防卫：like 甲知道乙要下楼杀丙，在甲下楼的时候就把房门锁上 |
| 过当 |  ？ | 应当减免 |

      2.1）防卫过当
          1）if 造成死亡结果 + 是先前行为+不救助 共同导致的
                if 对结果是一般过失；
                          不作为的过失致人死亡
                if 对结果是间接故意；
                          不作为的故意杀人
                like:甲对实施一般侵害的乙正当防卫致其重伤（在限度内），乙已经无侵害能力，
                     乙请甲将其送往医院，甲不理会而去
                     1）乙死亡结果不是由甲不救助行为独立造成的
                       甲有救助的义务
                     2）甲防卫过当
          2）防卫过当也是防卫行为，需要满足
             2.1）起因
                  1）防卫的对象要有紧迫性、进攻性、破坏性
                         but 不要求同时具备
                  2）对国家利益、社会利益 普通人也可以正当范围
                         but 不能超过必要限度，最多重伤
             2.2）时间
                  1）财产犯罪中
                         if 行为既遂+现场可以挽回损失；应当认为不法侵害尚未结束，
                  2）人身犯罪不可以
                            cuz 人身犯罪一旦既遂，危险就已经结束
                            only 最大限度是重伤，超过则防卫过当
             2.3）对象
                  only 侵害人本人
             2.4）主观
                  1）防卫认识（有人在干坏事）
                  2）防卫意志（保护特定的利益）

      2.2）trifle
           1）甲盗窃后遇见乙，乙紧拽住甲要求分成，甲打乙轻伤逃走
              analyze: 对 黑吃黑 也可以正当防卫
                       甲成立正当防卫
           
           2）甲扭送罪犯乙的途中，死死按住乙导致窒息死亡
              analyze:2.1)扭送属于法令行为，！正当防卫
                      2.2)死死按住乙导致窒息死亡 属于扭送的不当行为，！正当防卫
                      2.3)if 甲主观有过失；过失死
                             else if;意外事件
          
          3）乙追赶抢劫犯甲，车辆并行时撞到护栏，甲死亡
             analyze:3.1)正当防卫要 看上去是犯罪行为
                         乙的追赶行为 不可能是犯罪
                     3.2）甲的死亡是自己驾车导致的
                          中断因果关系
                     3.3）即使认为有因果关系，乙的行为 ！ 正当防卫
                         cuz:抢劫行为已经得逞，乙已经取得了财物->
                             不法侵害紧迫性已经消失->
                             只能进行一般的正当防卫->
                             致使死亡的，防卫过当
                        
                          alike:甲持刀抢劫出租车司机乙，下车逃跑，甲发动汽车将乙撞死
                                cuz:视为不法侵害尚未结束->
                                    but 已经不具有危及人身的危险性，只能一般防卫->
                                    甲撞死乙构成防卫过当
         
         4）甲使用丙的财物 反击不法侵害人乙
            ie.正当防卫 && 紧急避险的 竞合
               甲对丙是紧急避险；对乙是正当防卫
         
         5）if 紧急避险中 个人利益>公共利益
               可以通过损害公共利益的方法保护个人利益
                  like:司机发现甲被乙撞成重伤，危在旦夕；
                       超速闯红灯将被害人送到医院救治
                       成立紧急避险，不构成危险驾驶
         6）if 正当行为；
               if + 造成具体结果：不认为是犯罪
               if + 创造了风险：有义务排除
            
            if 甲见义勇为追赶小偷乙,乙跳入河中，甲见状离去，乙突然抽筋淹死;
                     甲不构成过失犯罪,cuz 甲并没有认识到自己的救助义务
            if 甲开车溅起石子砸伤了乙，乙流血不止
                     甲有救助的义务

      2.3）被害人承诺
           1）推定承诺，即使事后不同意也没有用
                 like:甲的家里发生火灾，乙闯入甲的家中抢救
           2）重伤承诺无效
              ie.only 砍掉小拇指
           3）自由可以处分
              ie.可以同意被拐卖、非拘

                                       故伤
              if 骗 A 别人都捐了1W，A 捐了2W  +  最后捐给了灾区；
                                       无罪
                  if + 吞没了捐款；
                          诈骗
           5）危险接受
              depends 被害人自己对结果是否能起到支配作用
              if 乙明知甲醉酒开车仍然坐车，交通事故致乙死亡；
                               甲交通肇事，乙没有同意牺牲生命 
              if 甲乙丙三人决定试试深水区的水，牵手一起下去，丙死亡；
                                甲乙不构成犯罪，被害人自己支配了结果的发生
              if 甲乙驾车冬天到水库玩耍，甲在冰面上开车，乙表示同意；后乙溺亡；
                                危险完全掌握在甲手中，甲过失死

      2.4）对来自动物的侵害
           1）野生动物- 紧急避险
           2）有主物
              if 主人教唆；正当防卫
              if 主人过失；正当防卫
              if 动物自发侵害；
                     主人存在侵害行为的不作为，正当防卫

      2.5）假想防卫
           if 有过失；
                 过失犯罪
                   if 有特定的紧迫性；
                          适度提前预备行为（入户抢劫、杀人、进入房间就可以防卫） 
                    like:甲抢劫出租车司机乙，乙开车将甲撞成重伤抢回财物
                         乙构成正当防卫
                              cuz:财产性犯罪延长防卫时间 +  不宜造成他人死亡 
           else if;意外事件

      2.6）偶然防卫
           way 1 （通说）行为无价值论：成立故意犯罪
                             cuz 从更早的阶段规范行为
           way 2：结果无价值论：不以犯罪论处
                             cuz:出现结果||有结果的高度可能性才处罚

# II.责任要件 
## 1.责任要件符合性
     故意、过失都是针对结果来讲的
        like:交通肇事中，行为人对违章行为本身是故意的
                       but 对于造成他人死亡时过失的

### 1.1 故意
       1）if 1)甲欲杀乙，乙丙同在一间房子里，甲锁死门窗之后放火;
             2)甲心想：丙你就看运气吧
          cuz丙必死无疑
             $:甲对丙是 直接故意

       2）盗窃需要明知的是：
          2.1）他人占有之物；
          2.2）数额较大
               ie.对于其他的事情是否明知不影响盗窃罪的成立

       3）间接故意存在未遂形态
          like:丙被甲乙合法追捕，只有1发子弹，丙对甲乙都有杀人故意，子弹从两人中间穿过
               丙对两人都成立故杀未遂
                  cuz 丙明知自己的行为会产生损害结果，既可能是直接故意，也可能是间接故意

### 1.2 过失
    1）过于自信的过失（不相信危害结果能发生）
      1.1）采取措施+有效果
      1.2）基于经验、技术
      1.3）diff && 间接故意(放任结果发生)
           case_1)交通肇事(过于自信的过失) 
                  以危险方法危害公共安全（间接故意）
                2)甲欲杀死妻子乙，认识到可能毒死儿子丙，还是投毒
                           对丙的死亡是间接故意
                       cuz 没有采取防止危害结果出现的措施
