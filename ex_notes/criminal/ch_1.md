# ch_1 犯罪论
# I.违法构成要件

## 1.构成要件符合性
### 1.1 主体
#### 1.1.1 不真正身份犯/加减身份，不影响定罪量刑的身份->刑法中关于加减身份的规定不适用于不具有加减身份的人

> like:非法拘禁，国工利用职权的，从中触犯

#### 1.1.2 单位犯罪
> depends:名利双收

    1）！incl:
      1.1) 不具有法人资格的独资、私营企业、事业单位
      1.2）贷款诈骗、有价证券诈骗、信用卡诈骗
      1.3）以单位名义+实施自然人犯罪
      1.4）管家分钱：总裁、领导

     2）incl:
      2.1) 拒不执行判决裁定罪
      2.2）分支机构以单位名义+违法所得归分支机构 
          like:后勤处干的
      2.3）单位被吊销、破产 == 追究自然人的责任，仍然是单位犯罪
      2.4）行政机关、法院、军事等
      2.5）东家分钱
           like:股东集体讨论决定，钱分给了股东

     3）notes:
      3.1)单位犯罪的，单位 && 直接责任人||直接责任人 之间不成立共犯
          cuz:人格不独立

### 1.2 作为、对象
> 作为、不作为与犯罪性质没有必然的联系，刑法分则各章中都存在不作为犯罪

#### 1.2.1 不作为义务

     1）有义务
      1.1）遗弃罪的不作为来源于：不履行抚养义务
      1.2）出租车 && 乘客：不作为的强奸的帮助犯
      1.3）共同危险行为有相互救助的义务
           like 探险
           unless 特别说明
      1.4）合同即使过期，人身安全保障义务也不能免除
      1.5）紧急避险人 在危险消除后 对第三人
      1.6）特殊防卫+侵害结束+有死亡的紧迫危险


     2）！义务
      2.1）成年人出去玩+是当场唯一能救的人 == 没有义务
           unless：唯一 + 可控制的范围内
      2.2）共同去风景区
      2.3）单纯发现了危险：
           like:在深山里发现了弃婴
      2.4）内心没有承担风险的意思
           like:把弃婴报到了加油站||福利院||医院门口

     3）共犯问题
       3.1）甲投毒，乙看见了不阻止，乙成立不作为的教唆犯
       3.2）甲要救被自己撞下水的人，乙劝阻甲救人，乙成立不作为犯罪的教唆犯
       3.3）甲以强奸的故意致丙昏迷实施奸淫，随后乙来到也进行了奸淫+窃取财物->甲承担轮奸+盗窃
       3.4）甲乙共同入户抢劫，甲捆绑丙实施奸淫，乙在旁边拿财物，乙也成立强奸
       3.5）甲乙共谋入户盗窃+分别在不同的房间找东西+乙发现了被害人丙准备杀害+甲知道没有组织 == cuz 盗窃不会使人有生命危险->甲只是单纯的发现->甲只承担盗窃的责任

### 1.3 结果
#### 1.3.1 class 
    1)结果犯/侵害犯
      incl：污染环境罪、非法行医致人死亡
      !incl:遗弃、侮辱、诽谤、强制猥亵、盗窃、诈骗、绑架致人死亡

    2）危险犯
      2.1）抽象危险犯
           incl:生产销售假药、有毒有害食品
      2.2）具体危险犯
           incl:破坏交通工具罪
                放火、决水、爆炸、投放危险物质、以危险方法危害公共安全
                破坏电力设备、破坏易燃易爆设备罪

#### 1.3.2 结果加重犯
    1）对加重结果可以是故意||过失
       like:1.1）抢劫致人重伤、死亡，行为人对重伤、死亡既可以是故意，也可以是过失
            1.2）故意伤害致人重伤，行为人对重伤既可以是故意，也可以是过失

     2）对加重结果 only 过失
       like:故意伤害致人死亡，对死亡只能是过失-> else 故杀

     3）盗窃
       3.1）想偷较大数额，实际投了特别巨大的数额->按照实际的数额量刑，对加重结果无需故意

### 1.4 因果关系
    1）介入行为不中断 + 两个行为都有因果关系 == 均有关
       like:打到濒死的状态+医生一般过失
    2）没有联络+分别都能独立导致结果的发生 == 均有关
       like:甲乙碰巧同时开枪，均打中乙,击中心脏死亡
    3）没有联络+单独不都 导致结果的发生 == 均有关
       like:甲乙碰巧同时开枪，均未打中要害,乙失血过多死亡
    4）甲把乙从楼顶推下，丙正好朝窗外开枪，乙被打死
       analyze:甲属于故杀未遂->因果关系中断，不再向前追溯->丙属于意外事件
    5）自杀不中断因果关系->导致生不如死
       5.1）虐待
       5.2）暴力干涉婚姻自由
    6）特殊体质不中断
    7）自然时间中断
       like:伤害被害人后送去医院+医院发生火灾
