### 1.4 因果关系
        notes:有...的目的||动机，就有...的因果关系 （False）
                cuz:目的||动机 属于主观判断、是责任要件
                    因果关系 属于客观判断、是违法构成要件

        顺序：if 两个人同时；
                if 任何一个行为都能导致结果的发生；二重因果关系
                      ie.if 没有联络+分别都能独立导致结果的发生 ; 均有关
                             like:甲乙碰巧同时开枪，均打中乙,击中心脏死亡
                
                if 结合才能导致结果的发生；重叠的因果关系
                      ie.if 没有联络+单独不都 导致结果的发生 ; 均有关
                             like:甲乙碰巧同时开枪，均未打中要害,乙失血过多死亡
              
              else if 有先后；介入因素判断
                              if 中断因果关系；条件说失效，禁止溯及前行为
                                 case_甲把乙从楼顶推下，丙正好朝窗外开枪，乙被打死
                                      cuz:甲属于故杀未遂->
                                          因果关系中断，不再向前追溯->
                                          丙属于意外事件

                              if 介入行为不中断 + 两个行为都有因果关系 ; 均有关
                                     like:打到濒死的状态+医生一般过失

          1）if 造成死亡结果;是先前行为+不救助 共同导致的

             2.3）对象

# II.责任要件 
## 1.责任要件符合性
     故意、过失都是针对结果来讲的
        like:交通肇事中，行为人对违章行为本身是故意的
                       but 对于造成他人死亡时过失的

### 1.1 故意
       1）if 1)甲欲杀乙，乙丙同在一间房子里，甲锁死门窗之后放火;
             2)甲心想：丙你就看运气吧
          cuz丙必死无疑
             $:甲对丙是 直接故意

       2）盗窃需要明知的是：
          2.1）他人占有之物；
          2.2）数额较大
               ie.对于其他的事情是否明知不影响盗窃罪的成立

### 1.2 过失
    1）过于自信的过失
      1.1）采取措施+有效果
      1.2）基于经验、技术

    2）过失犯罪，应当预见的结果是该罪的构成要件所规定的具体的特定的结果
       like 过失致人死亡、过失致人重伤应当预见重伤/既有重伤也有死亡；else 意外事件

    3）共同犯罪不承认共同过失犯罪，分别认定各自的犯罪，但是要证明因果关系
       3.1)if 甲乙以为是兔子但是打到了人+ 同时开枪+ 只有一个弹孔 + 无法区分是谁干的
              甲乙不构成犯罪;
                 cuz无法证明各自的因果关系
       3.2)if 甲乙以为是兔子但是打到了人+ 同时开枪+ 只有一个弹孔 + 该弹孔有两颗子弹穿过
              甲乙分别过失死；
                 cuz 有因果关系，不承认共同过失，按照各自的责任处罚

### 1.3 事实认识错误
#### 1.3.1 同一犯罪构成间

     1）打击错
        version 具体符合说：甲对乙是故杀未遂，对丙是过失死，现象竞合
        version 法定符合说：甲对乙是故杀未遂，对丙是故杀既遂，一个行为，属于故杀既遂
        $ : 对象错误，都是既遂
            打击错误，只有具体符合说出现未遂

     2）因果关系错误
       2.1）狭义因果关系错误
            like:甲本来想淹死乙，但是推入井中摔死了乙
            $:不影响犯罪既遂的成立

       2.2）事前故意/犯罪构成的推后实现
            case_甲对乙先实施了第一个行为A，以为其已经死亡，为了毁尸灭迹，实施了第二个行为 B,B 导致了乙的死亡

            version 2）将两个行为视为一个整体，将支配行为视为概括的故意，只成立一个故杀既遂

       2.3）犯罪构成的提前实现
            depends 实施 第一个行为 是否已经着手
            case_毒咖啡案：
                 2.3.1）乙固定在2h 后才回家，甲成立故杀预备未遂+过失致死的既遂 想象竞合
                 2.3.2）乙随时可能回家，甲故杀既遂

# III.Trifle
## 1.刑法的解释
     1）强奸中的暴力、胁迫 alike 猥亵
        非法经营罪中的买卖:做买卖
        使用假币指的是伪造的货币
           !incl:变造的货币
        把信用卡解释为借记卡、把弹头、弹壳解释为弹药 扩大解释

     2）未经著作权人许可大量播放其音像制品:侵犯机械表演权
        发行：非法制作侵犯著作权的产品，通过出售、出租、出借、散发等方式提供复制品的行为
             ie.发行是通过一定载体向公众传播侵权复制品

     3）if 外国人在国外打中国人;
           3年以上+双重犯罪 == 可以追究
        if 普通人;轻罪可以不追究

     4）规范的构成要件
        ie.专业的法律概念
           like:侮辱、诽谤
        以行为人在实施其行为时说认识到的一般人的判断结论，认定行为人是否有故意
