# ch_5 侵犯财产犯罪
# 1.财产犯罪的共性问题
## 1.1 class 
       1)if depends 利用职务便利；贪、职侵、挪公款、特定款物
       
       2)if depends 非法占有目的；挪公款、特定款物、故意毁坏财物 不具有
                  notes:1)非法取财后又出卖的
                          depends 是否 侵犯新的法益 && 期待可能性
                          1.1）违禁品 usu 并罚
                          1.2）普通财物 depends 坑了几个人
       
       3)if depends 转移占有；侵 不具有
                  notes:1)转移占有
                          1.1)行为
                              ie.他人占有
                          1.2）意识
                              ie.知道是别人的
                        2）认识错误
                           2.1）甲想要 盗窃，实际上 他人遗忘物
                           2.2）甲想要 侵占，实际上 盗窃
                           $ : 抽象认识错误，在主客观统一的范围内定侵占
                        3）在转移占有之后才有 非法占有目的的
                           属于侵占
       
       4)if depends 和平取财；盗、诈

## 1.2 单位内部的人占有单位财物
       depends  是否利用职务之便
                ie.自己负责管理控制财物
                   like:出纳 无论采用什么方法 拿单位的钱，都属于 职侵
                        老板派甲出去代为收款，甲据为己有，属于 职侵

## 1.3 竞合
       1）职侵 && 侵
          优先适用职侵
       2）盗、诈、侵 && 职侵
          优先适用职侵
       3）偷走自己借给别人的东西 + 骗他人赔偿
          盗 && 诈 从一重罪，定盗窃。
             ie.后行为视为不可罚的行为
       4）IP 类犯罪 单独定罪
          like:if 以胁迫方式获取商业秘密；侵犯商业秘密罪
               if            财物；    敲诈

# 2.抢劫
## 2.1 263 普通抢劫
       2.1.1 当场
             1）实施暴力、胁迫+趁对方没注意，当场取走财物
                cuz:对方过于惊恐
             2）实施暴力、胁迫+对方吓得跑了，当场取走财物
                notes:if 逃跑的时候钱包掉了，捡走的
                         介入因素异常；中断因果关系->抢劫的未遂

       2.1.2 加重情节
             1）入户
                1.1）！抢劫：合法入户+户内施暴
                1.2）入院子 视为 入户
                1.3）非法入户目的 incl:
                     1.3.1)抢劫
                     1.3.2)事实其他人身、财产犯罪
                           like:杀、盗
             2）在公共交通工具上
                2.1）在运营中
                2.2）！incl:出租车
                2.3）在车上扒窃被发现，失主追下来，抗拒抓捕->
                     269转化抢劫
                        cuz:没有危及到车上的人的安全
             3）抢劫银行||其他金融机构
                ！incl：取出来||正要存 的钱
                  like:抢劫储户取出来的钱不属于加重
             4）冒充军警
               4.1）！incl:真军人、警察 以 军人、警察的身份
             5）持枪
               5.1）使用、显示
               5.2）！incl:仿真枪

        2.1.3 飞车抢夺
              cuz 抢夺是对物暴力，抢劫是对人暴力
              依据263 定罪

## 2.2 267-2 携带凶器抢夺
- 1）！incl:使用、显示；else 263
- 2) 非管制类凶器 + 为了犯罪而携带 == 构成本罪
- like:背着斧头砍柴中抢夺，定抢夺

## 2.3 269 转化抢劫
      1）前提
         1.1）盗、诈、夺 要求着手，不要求既遂
         1.2）特殊盗窃
              depends 是否具有财产犯罪的属性
              1）盗伐林木、盗窃枪支
                analyze:评价为盗窃->适用转化型抢劫
              2）盗窃国有档案、尸体的
                 不构成269
      2)暴力程度是足以压制反抗
        2.1）有可能使得对方遭受危险
        2.2）与263暴力程度相同
      3）他人incl；
         3.1)财物所有人
         3.2）占有人
         3.3）抓捕者
         3.4）阻止者
         $ 打击错误，可以转化抢劫

## 2.4 289 聚众打砸抢
- 法律拟制，毁坏不具有非法占有的目的

## 2.5 罪数
      1）绑架+当场劫取被害人 随身携带财物
         cuz:控制人质既是绑，又是抢的一部分
         绑+抢，择一重罪
      2）抢 信用卡 + 使用；抢劫
