                快递甲趁监督人员上厕所 偷走快递；盗窃

###  2)if depends 非法占有目的；挪公款、特定款物、故毁财 不具有
> 非法占有：排除意思（伪装退货、取得商品对价，从超市拿出商品，认定为有排除的意思） + 利用意思（除非有证据证明，否则推定有利用意思）
####   2.1)非法取财后 + 出卖
            depends 是否 侵犯新的法益 && 期待可能性
            1）违禁品 usu 并罚
            2）普通财物 depends 坑了几个人

             if 报销垫付的钱；！
###    2.3）if 没有利用的意思；故毁财
            incl:单纯的放置不用
            if 后来才产生非法占有的目的；侵占
                like:本来想要毁坏手机，后来觉得很好玩拒不返还

###  3)if depends 转移占有；Only侵 不具有
####    3.1)转移占有
####        1)客观行为(他人占有)
              if 短暂离开主人仍然占有；盗窃
                  like:上个厕所、帮忙看店、刚下出租车就想起包还在车上+司机一踩油门把车开走
              else if 属于转移占有；侵占
                  like:出差几天
####        2）主观意识(知道是别人的)->犯罪故意的问题->法律问题->法官判断
                  like:遗忘在飞机上、公交车上的大件物品都属于盗窃
####    3.2）侵占
             1)if 在转移占有之后才有 非法占有目的;侵占
                  like:甲借了乙的车使用，过几天不想还了
             2)侵占的占有 
               incl: 基于委托、租赁、借用、担保、无因管理
                  like:干洗店的老板将顾客衣物里的财物据为己有； 
                       甲的卡 + 乙的钱 -> 钱在甲的占有之下；
####    3.3）认识错误
> $ : 抽象认识错误，在主客观统一的范围内定侵占

| 偷钱  | 偷到了枪 |  盗窃 + 非法持有枪支 == 数罪并罚（cuz 盗窃！incl 枪支） |

###  4)if depends 和平取财；盗、诈
       1）only  盗窃的既遂标准采取失控说
             else 财产犯罪不采取
       2）敲诈、诈骗
             if 取得财产与手段之间没有因果关系；只能成立未遂

# 2.抢劫
> 没有携带凶器抢劫这种罪名，最终都只是评价为一个抢劫罪
## 2.1 263 普通抢劫
> 既遂（之一就构成）：1）劫财;2）造成他人轻伤以上
###  1) 当场
####   1.1）趁对方不注意
            depends 原因行为
               if 实施暴力、胁迫 导致对方过于惊恐 + 当场取走财物；
                            抢劫
               if（单纯）趁对方不注意 + 取走财物；
                            盗窃
####   1.2）实施暴力、胁迫 + 对方吓得跑了 
             1)+ 钱包掉了，捡走财物;抢劫未遂            
                  cuz:介入因素异常+对结果起到决定作用->
                      中断因果关系
             2)+ 要求对方把钱包留在现场；抢劫既遂
####   1.3) 有因果关系
            1）if 劫匪把刀放在甲的脖子上，旁边的乙看见了给甲钱，甲乙是路人；抢劫既遂
                          cuz 乙给钱是害怕劫匪伤害甲，取财和暴力手段有因果
            2）甲发现乙被绑在树上 + 趁机取走财物；盗
####  1.4）暴力手段
#####    1）被害人潜入麻醉状态
            depends 是如何陷入的；
              if 自己陷入 + 取走财物；盗窃
              else if 行为人使其陷入 + 取走财物；抢劫
                 like:甲想要抢劫乙，在乙的杯中投入麻醉药，2h 后回来，发现乙外出，取走财务
                        抢劫 未遂  + 盗窃 既遂 == 并罚
#####    2）if 甲入室盗窃，对方要求不要拿；甲不予理会；不构成抢劫
              cuz 没有强制手段
####  1.5)飞车抢夺

                          依据263 定罪（暴力的目的是为了当场取财）

               2.1）if 掉包、调虎离山；盗窃

                    if 在道路上 + 开走；诈骗
                             cuz:把车从4S 店占有转为试车人占有
           3）学说
               verison 抽象：对种类有认识就构成既遂；诈

           4）case
              1) if 甲以为自己在输入验证码，实际上是在转账；乙盗窃
                    cuz 乙实质上将甲作为工具加以利用，在其不知情的情况下，把甲的财产转移到自己的账号中
