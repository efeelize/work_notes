## 2.3 轮奸
       1）规则
         1.1）量刑规则
              ie.纯事实判断
                 only 成立||不成立的问题，必须实际遭受到了损害才能使用
              $ : 没有未遂的问题
         1.2）加重情节的犯罪构成
              ie. 存在未遂
       2）if 2人准备实施轮奸，1人尚未实施就被抓住
             version 量刑规则：没有轮奸发生，适用强奸的基本法定刑
             version 加重情节：基本犯既遂+加重犯未遂（已经着手开始实施了犯罪），以轮奸未遂定罪，比照轮奸既遂从轻减轻处罚
          
          if 3人准备实施轮奸，1人尚未实施就被抓住
             构成2人以上轮奸的法定刑

### 2.3.1 诈骗、盗窃 数额巨大、特别巨大 属于加重构成
          1） if 想要取得数额巨大||特别巨大，实际上一般的 ; 
                 特别巨大数额的未遂，10年以上，比照既遂从轻减轻处罚
              like:if 以1w 谎称 100W 进行诈骗；
                   1.1）加重构成
                        1w 既遂 + 99w 未遂
                   1.2）量刑规则
                        1W 既遂，在基本犯罪的法定刑幅度内
          2） if 部分既遂、部分未遂的
                if 在同一量刑幅度内；以诈、盗既遂处罚
                if else 不同量刑幅度的；以处罚较重的处罚
              like: 想要偷1B,实际偷了1W；以1B 未遂处罚，10年以上
