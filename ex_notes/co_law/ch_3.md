# ch_3  破产
# I.程序的转换
# 1.破产（期间：受理-裁定终结破产程序）
    1）申请->
    2) 受理 + 指定管理人 ->
    3) 申报债权\破产分配方案 ->
    4) 裁定破产宣告（1st 裁定）
      （做出前可以转为重整、和解）（一旦做出程序不可逆）
    5）破产分配方案（清偿债务）->
    6) 裁定终结破产程序 （2nd 裁定）->
    7）10days 管理人注销登记

## 1.破产宣告前
     if 债务人 提破产申请；
                 选择：重整||和解||清算
            if 债务人选择清算；
                       债权人可以申请重整
                       债务人可以申请和解
            else if 债务人选择重整；
                      债权人异议 + 法院审查认为重整；
                            裁定债务人重整（法院拥有最终裁定权）

     if 债权人 提破产申请；
                 选择：重整||清算
            if 债权人选择清算；
                         债务人||出资额 占 债务人注册资本1/10以上，可以申请和解、重整
                         if 债务人有异议；
                               自收到法院通知之日起7days 向法院提出

     $:1）可以单独提出重整、和解
       2）债权人只需要证明债务人不能清偿到期债务
       3）重整期间：自法院 裁定债务人重整之日 到 重整程序终止

## 2.破产宣告后
     ie.标志破产案件不可逆的进入清算程序，不得再转换

# 2.和解
    1）only 债务人申请  + 同时提出和解协议草案 ->
    2）法院受理 ->
    3）对和解协议草案表决 （人头过半数 + 代表债权额占无担保债权总额的2/3）->
    4）裁定终止和解程序 ->
    5）债务人执行 ->
       5.1）恢复生产经营
       5.2）进入破产程序

# 3.重整（期间：受理到裁定终止重整程序为重整期间）
    1）债务人、出资人、债权人 + 申请 ->
    2）法院受理 + 指定管理人 ->
    3）制作重整计划 ->
    4) 分组表决 （人头过半数 + 债权额占该组总额的2/3）->
    5) 裁定终止重整程序（裁定做出，重整终止，批准重整计划） ->
    6）债务人执行
       6.1）恢复生产经营
       6.2）进入破产程序

# 4.破产原因：
    incl:1）不能清偿债务；2）资不抵债；3）明显缺乏清偿能力
    if 债权人提；证明1）
    if 债务人提：ALL 需要证明

# II.清算（解散清算diff破产清算）

# 1 清算组职权
## 《公司法》：处理剩余财产
> if 资>债；剩余财产分配给sh;else if 资<债；交给法院走破产程序
### 1）允许
     1.与法院相关
       1）清理公司财产，编制资产负债表、财产清单、制定清算方案 
          + 1.1）自行清算：报股东（大）会、确认
          + 1.2）指定清算：报法院 确认 
       2）清算组负责人代表公司参与民事诉讼活动。
     2.处理财产
       1）处理与清算有关的公司未了结的业务
       2）清缴所欠税款以及清算过程中产生的税款;
       3）清理债权、债务;
       4）处理公司清偿债务后的剩余财产; 
     3.通知、公告债权人;
### 2）禁止
     1.(禁止个别清偿)
        在债权申报期间，清算组不得对债权人进行清偿;
     2.(禁止开展新业务)
        清算期间，公司存续，但不得开展与清算无关的经营活动;

## 《破产法》：处理资不抵债 ->管理人职权
      1）清理债权、债务
       2）处理公司清偿债权、债务后的剩余财产
       3）代表公司参与民诉
          if 清算中；
                 法代参加诉讼
          if 清算结束后，注销登记前 诉讼（法人还在）；
                     公司名义 + 清算组负责任参加诉讼

# 3. 通过
## 3.1 和解协议
> only 债务人提 + 和解的成立必须经过法院的裁定认可
###    1）债权人会议的决议
            usu:出席过半数 + 债权额 占 无财产担保债权的 1/2以上
            exp:和解协议 
                出席会议过半数 + 债权额 占+ 无财产担保的债权 + 2/3以上
###    2）债权人委员会 =< 9
             incl:债权人会议选任的债权人代表 + 1个债务人职工代表||工会代表
###    3) if 和解协议表决未获得通过 || 法院不认可；
                法院裁定终止和解程序 + 宣告债务人破产
          depends:表决后法院审查批准,即使表决未通过法院可以强行批准

## 3.2 重整计划
###    1）制作
             if 债务人自行管理 财产&&营业事务；
                                     债务人制作
             else if 管理人负责管理 财产&&营业事务
                                     管理人制作
###    2）表决（分组表决 + 全部小组都通过 == 重整计划通过）
          1)分组
            depends 法定债权分类
          2）每一组 双重多数决
             出席人数过半数 + 债权额 占 该组债权总额的2/3以上
###    3）执行
          If 自重整计划通过之日起10days 内 ；
               债务人||管理人向 法院提出批准申请；
                    if 法院审查符合；
                         自收到申请之日起30days 内批准 + 终止重整 + 执行重整计划

## 2.一般取回权（基于返还原物请求权 + 不经破产程序 + 直接取回）

             在破产财产变价方案、和解协议、重整计划提交表决前 + 向管理人提出；

               可以取回

### 2）所有权保留买卖中的取回权（if 买方破产）
####  2.1)if 合同解除；卖方取回标的 + 返还价款
           else if 合同继续履行；付款义务在法院受理破产申请时 视为已到期

           exp:买方支付75%以上||TP 善意取得

## 5.破产抵销（diff civil_law:无所谓是否到期、标的品质种类）
### 5.1）在受理前取得 + 债权人 + 可以 向管理人主张抵销
### 5.2）在受理后 + 
####     1）次债务人取得 他人对债务人的债权 + 不得主张
             int:甲（100）——>债务人（100）——>乙
                 if 乙将100转让给甲
                    甲不得向债务人主张抵销
                           cuz 恶意创设债权
####     2）债务人'sh. 对债务人的负债
           2.1）欠缴出资、抽逃出资
           2.2）滥用股东权利、关联关系 损害co利益

     case_甲乙买卖合同，甲发货，乙尚未付清全部货款时候甲向法院提出破产申请，法院裁定受理
          1）货到后，所有权转移给乙
             cuz 法院裁定受理
          2）乙已经进入破产程序，债权人不能再行使取回权
          3）甲对乙的请求支付价款的请求权变为破产债权
  
     case_法院受理乙的破产申请，管理人甲发现：
          1）丙的一项债权有房产抵押
                必须经过破产债权申报和确认后才能行使抵押权，不能在破产受理后就直接行使
          2）乙丙之间有尚未履行的合同
                乙申请破产后，针对正在履行中的合同，管理人可以选择继续履行||解除合同
          3）乙曾经租赁给丙一套设备被毁损 + 侵权人向丙支付了赔偿金
                only 作为普通债权在破产清偿程序中受偿
                    cuz 赔偿金的支付发生在破产前，权利人不能行使取回权
          4）丁对乙负有债务，在破产受理后受让了 乙||丙 的债权，
                不能主张抵消
                    cuz 发生在破产受理后

# 2.管理人追回
## 2.1 无效（% 当然违法 %）
       1）为逃避债务转移、隐匿财产的
             like:假借条、假离婚
       2）虚构债务||承认不真实的债务的
## 2.2 可撤销
###    1）% (受理破产申请前)1年无偿不合理，放弃担保未到期 ->推定违法 %
         S 民总债权人撤销权
###    2）% 存在破产原因 + 受理破产前6mon + 对个别债权人清偿->当然违法 %

###    3）不得撤销
          1）水电费
          2）劳动报酬、个人损害赔偿金
          3）债务人以自有财产设定担保物权的债权进行个别清偿
                  ie.消灭担保物权
                  exp:清偿时担保财产的价值<债权额
          4）if 经过诉、裁 + 对个别债权人进行清偿;
                usu:不得撤销
                exp:债权人&&债务人 恶意串通 损害债权人利益的
          5）else: 使债务人受益的个别清偿

## 2.3 追回权（属于民总：基于投资关系产生的缴付出资请求权）
       尚未完全履行出资义务 + 不受出资期限的限制

## 2.4 特别追回权（企业高层）
      董监高利用职权 + 拖欠职工工资 + 非正常收入 && 侵占企业的财产
           like:发高昂报酬、绩效奖金

# IV.破产财产的清偿
    1）破产费用>共益债务>劳动债权（工资、医疗、伤残补助、抚恤费用）>社保、税款>普通破产债权
    2）同一顺位的，按照比例（可分配财产/未清偿费用总额）清偿
    $: % 外部看先后，内部看比例 %
       like:破产财产500w.破产费用300w.共益债务400w
            500-300=200
            200/400=1/2

# 1.破产费用（为破产程序进行，由债务人财产优先支付的费用）
        incl:1)诉讼费用（不需要预交）
             2）管理、变价、分配债务人财产的费用
                   like:仓储
             3）管理人 + 执行职务 + 费用、报酬、聘任工作人员
                   like:差旅费

###  1.1）时间：受理前成立的债务 + 受理后 双方当事人均未履行完毕
###  1.2）管理人有权决定

           else if 解除；
                自受理之日起2mon 未通知|| 自收到对方催告之日起30days内 未答复
## 2）法定之债：无因管理、不当得利
## 3）侵权之债
       3.1）管理人执行职务致人损害、债务人财产致人损害
       3.2）为债务人继续营业支付的劳动报酬、社保费
                 like:管理人为了维持公司运行，支付的水电费、保安费
