# 9.合伙
| items | LP  | GP | 
| ---- | --- | --- | 
| 执行人 |  不得执行事务  |  1）确定执行人的，其他合伙人不再执行合伙事务；2）only  执行人可以对合伙事务的执行提出异议 |
| 利润分配 |  全部利润 可以 分配给部分人（% 可以不同甘 %） | 不得  |
| 亏损承担 |  全部亏损 不得 分配给部分人（% 必须共苦 %） | 不得 | 
| 无限人 |  不得要求其退伙，cuz 无行为能力要求 |  一致同意转为有限人 |
| 丧失偿债能力 |  不得要求其退伙，cuz 以其交的钱为限担责 |  当然退伙（ % 人死财空没资格 % ） | 
| 入伙 |  全体一致同意 ，exp 另有约定   |  全体一致同意  + 书面入伙协议  | 
| 入伙前的债务 |  以认缴的出资额为限担责 | 1）无限连带；2）if 老合伙人故意隐瞒负债情况；新合伙人先对旧债负责，清偿之后再向其他老合伙人追偿  |
| 退伙前的债务 |  以退伙时从有限合伙中取回的财产担责 |  无限连带 |
| 身份转化 |  全体一致同意 + 对其作为 LP 期间的债务 无限连带 | 全体一致同意 + 对其作为 GP 期间的债务 无限连带 |
| 资格继承 |  当然继承 |  1）合伙协议的约定or 全体合伙人一致同意；从继承之日开始取得合伙人资格；2）if 继承人不愿意or 未取得资格；退换被继承人的财产份额 | 

## related：GP 
### 1）当然退伙
     % 人死财空没资格 %
     1）死亡||宣告死亡
     2）法人||其他组织 被吊销营业执照、责令关闭、撤销||
     3）个人丧失偿债能力
     4）合伙人在合伙企业中的全部财产份额被 法院强制执行
     5）合伙人变为无限人
        if 其他合伙人一致同意；
                     转变为 LP，普通合伙转变为优先合伙
        else if 未能一致同意；
                     从退伙事由发生之日 为退伙生效日
### 2）协议退伙
       if 未约定合伙期限 + 不给企业造成不利影响 + 提前30days 通知；
                                                      可以退伙
### 3）特普责任
       if 故意||重大过失 + 造成合伙企业债务；
                    行为人 无限连带 + else合伙人 以财产份额为限
       else 其他债务；
               所有合伙人 都是无限连带

## related：LP
### 1）LP 可以做的/不视为执行事务
      1）参与 决定 GP 的入伙、退伙
      2）对企业的经营管理提出建议
      3）涉及自身利益 + LP可以查阅 会计账簿
      4）为本企业提供担保
      5）诉讼

         5.2）执行人怠于行使权利 +  损害本企业的利益  +  以自己的名义起诉
