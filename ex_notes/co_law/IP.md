# 3.商标权的消灭（1个商品可以使用2个以上的注册商标:ie.联合商标）
## 3.1 注销
       if 申请 + 商标局核准；
                    自商标局 受到申请之日终止
       else if 宽展期满不续期 || 申请未获得批准;
                    注销
## 3.2 撤销
       1)only 商标局可以撤销
       2）依职权：商标退化
       3）依申请：防御商标 + 连续3years不使用
       4）自商标局公告之日起终止
## 3.3 无效（在公告期后3mon 向商标局提异议 + 已经履行不构成不当得利）
       1）绝对无效异议
         1.1）禁止使用、注册、欺骗等不正当手段获取 + 任何人可以提
              but 经过使用有了显著特征以后可以进行注册
         1.2）only 商标局宣告无效；
              else 都是商标评审委宣告无效
       2）相对无效异议
         2.1）cuz:
             1）抢注 + 驰名
             2）恶意注册 +地理标志
             3）恶意注册
             4）混淆
             5）抢注 + 他人已经使用&&有一定影响的
             6）在先申请人的异议
             7）损害他人在先权利（可扩大解释为权益）
         2.2）期限
             usu:自商标注册之日起5years
                 if 题中未标明“驰名商标”；
                          在先权利人要在5years 请求商标评审委宣告无效
             exp:驰名商标 无期限保护
                 related:未注册的驰名商标：不予注册+禁止使用
                         已经注册的：不相同不类似的产品也保护

## 3）善意侵权
> 1）usu:不构成侵权；2）exp:支付合理对价不用停止
### 3.1）出版者
         if 尽到了合理注意义务 + 不知出版涉及侵权；
                        停止侵权 + 返还利润
### 3.2）专利
        1）独占实施
           1.1）发明、实用：制、使、销（允诺销售：ie.要约）、进
           1.2）外观：！incl:使用 
        2）善意侵权
           if 为了生产经营目的 + 使、销（cuz 制、进有更高的注意义务） + 不知情；
                          停止侵权 + 不赔
           if 使用者  + 支付合理对价；
                          继续使用  + 不赔
### 3.3）商标
        1）销售者 + 合法取得 + 说明提供者 + 不知情；
                                           不赔
        2）注册商标权人的证明责任：自己之前3years 内曾经使用过该商标 + 造成损失
        3）if 未注册商标 + 别人抢注 ；
              使用人：不赔 + 停止侵权 （即使被认定为驰名商标）
              先用权人：在异议期内提出异议反对别人注册||核准注册后向商标局宣告无效（！撤销）
