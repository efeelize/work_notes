            else if ;撤

## 1.2 上诉||二审抗诉的撤回起诉
       if 期限内；无需审查+应当准许
                 一审的判决、裁定在 上诉、抗诉期满之日起 生效
       if 期满后；二审法院 应当审查 
                 if 1）一审裁判正确；准许撤诉，准许撤诉的裁定书送达之日起 生效
                    2）检院抗诉 + 不派员出庭 + 不说明原因；
                          法院 可以裁定 按照撤诉处理
                    else if 1）一审错了；继续审理
                            2）死立即 + 被告人上诉案件 + 开庭审理后
                                 应当不予准许，继续按照上诉案件审理
        if 上级检院 认为抗诉不当；
           可以向同级法 撤回抗诉 + 通知下级检院

## 1.3 再审申诉的撤回
       if 申诉人 再审期间内 撤回申诉；
             应当裁定准许
       if 申诉人 不来||先走；应当裁定 按撤回申诉处理
          exp 申诉人不是原审当事人

## 1.4 再审抗诉的撤回
       if 检院 再审期间内 撤回抗诉；
             应当裁定准许
       if 期满后；S 上诉||二审抗诉的撤回 
       if 检院 不来||先走；可以裁定 按撤回申诉处理

##  1.5 减刑假释撤销
         if 执行机关+书面+撤回建议+在法院作出裁定前;
            法院决定是否准许
         
         if 法院裁定减刑假释；
                          1）7days 内

                             + 送达 社区矫正机构||继承组织

# 3.抗诉
      1）一审判决确有错误
      2）只能书面

## 3.1 裁定
       1）不能申请抗诉
          cuz 不影响当事人的实体权利
       2）检院自己 可以针对法院的裁定抗诉

## 3.2 二审抗诉
      1）only 通过原审交抗诉书
          cuz 二审抗诉是 向上一级抗，通过同级检  对 未生效裁判 提抗诉
      2）县检 only 提二审抗诉

      3）程序
         if 地方各级检院 抗诉；
            应当通过原审法提出抗诉书 + 抄送上一级 检院 ->
                原审法 应当将抗诉书、案卷、证据 移送 上一级法 + 将抗诉书副本 送交 当事人
         if 上级检 认为 下级检 二审抗诉不当；
                可以直接向 同级法 撤回抗诉 + 通知下级检院

## 3.3 再审抗诉
       % 死、抗 必上 审委会 %
       1）上级检 对 下级 生效的裁判 提抗诉
       2）最高检 only 提再审抗诉
       3）程序
          if 上级检 发现下级法 已生效 判决裁定错误；
             1）直接向同级法提出抗诉
             2）指令下级法的 上一级检 向 同级法提出抗诉
       4）like:
          市检 发现 基院 错；向中院提再审抗诉
          省检 发现 基院 错；向高院提再审抗诉||指令市检 向中院提再审抗诉

# II.二审
## 1 被上（incl 自诉人上诉）、被抗 可加刑
     1)一审可以加
       if 二审发回重审 + 有新的犯罪事实||检院 补充起诉
     2）确实判少了 再审改
       if 刑法畸轻 || 少了使用附加刑 + 必须依法改判；
            在二审判决裁定 生效后；再审改
     3）可以加重罪名

## 2.审理
      1）判决
        if 没错；维持
        if 法律、量刑错；应当改判
        if 事实不清证据不足；可改可发
        if 程序违法；N 次发
        
        1.1）程序违法 incl:
             违反公开审判、回避制度、剥夺限制当事人的法定诉讼权利、审判组织的组成不合法

      2）宣判
         2.1）自行宣判
         2.2）委托 原审法 代为宣告 + 
              自己 向当事人送达 +
              自己 直接 向同级检 送达

## 3.复核
     notes:1）可以判但是不生效
           2）一案一报，层层上报

### 3.1 死立即
### 3.1.1 二审程序
          if 中院一审 死立即 + 上诉||抗诉 
              if 高院 裁定维持；
                    10days 内报最高法核准
                  else if 高院认为不应当判处死刑;
                    直接用二审改判

### 3.1.2 复核程序
           if 中院一审 死立即 + 未上诉、抗诉 + 期满后10days 内 报高院复核
              if 高院同意；
                    10days 内报最高法核准
                 else if;
                    应当按照二审 提审||发回重审

           if 高院一审 死立即 + 未上诉、抗诉 + 期满后10days 内 
              报 最高院复核

### 3.2 死缓高院核准
        if 中院一审 死缓 + 未上诉、抗诉 ；
            期满后 报高院核准

### 3.1.3 复核结果
