# ch_5 二审 && 再审 

# I.申诉（***）
# 1.一般申诉向检院提(depends 谁侵犯了自己的权利)
    if 公法；向同级检院提
    if 检院；向上一级检院提
            exp:酌定不诉，向原决定检

# 2.再审申诉
## 2.1 主体
       1）当事人 及其 法代、近亲属
          incl:被害人（可以申诉，不可以上诉）
       2）案外人提；法院 应当审查处理
       3）申诉 可以委托律师 

## 2.2 最晚申诉的时间
### 1）usu ：刑罚执行完毕2years 内
       从裁判生效之日起就可以提申诉
### 2）exp：超过2years 也应当受理
       1）可能宣告无罪
       2）期限内提了 + 法院未受理
       3）重大、疑难、复杂

## 2.3 二级申诉
### 1st 申诉
#### 1）usu:应当向 作出法院 提
#####  1.1)if 越一级申诉(直接向 做出法院 上一级法 提)
            1）让他交
                 ie.告知申诉人 向 做出法院提
            2）自己交
                 ie.直接交 做出法院
            3）重大疑难复杂的，自己审
                 ie.直接审查处理
#####  1.2)if 越两级(直接向 上上级法（高||最高）申诉)
            1）自己提审
            2）指令
                2.1）usu 指令原审 以外的
                2.2) exp 指令原审
                         if 原审有利于查明事实，纠正错误的

#### 2) exp
#####  2.1）二审法 裁定准予撤回上诉
                  向 一审||二审 提
#####  2.2）死立即 && 死缓
            原核准法（高||最高） 直接处理 or 原审法 
               if 向 原审法 提；
                   原审法 应当写出审查报告，提出处理意见 + 层报 原核准法（高||最高）

###  2nd 申诉(应当向 作出法院 上一级法 提)
#### 1)if 第一次申诉 不具有法定情形；
             应当说服申诉人  撤回申诉；
                   if 申诉人坚持；应当书面通知驳回
#### 2)if 第二次申诉 也被驳回 + 继续申诉；
               一般不受理
#### 3)if 对最高法的再审裁判||驳回申诉 仍然不服要申诉的；
               不予受理

# 3. 对公安不立案的申诉
           C_ch_4
# 4.对检院不起诉的申诉
           C_ch_4

# II.二审
# 1.上诉(可以口头)
## 1）可以上诉的裁定
        1）不予受理
        2）驳回起诉
           if 自诉人不服 不予受理、驳回起诉 + 上诉；
              二审法 认为一审错；应当撤销原裁定+同时指令 一审法 立案受理||进行审理
        3）没收
## 2）方式
###  2.1）if 通过原审法院提;原审法 应当审查
            if 符合规定；应当在上诉期满后 3days 内 
                   案卷材料移送上一级法 + 上诉状副本 送交 同级检院 && 对方当事人
###  2.2）if 直接向二审法院提；二审法 收到材料后 3days 内交原审法
            一审法按照1）的程序再走一遍
## related:二审抗诉only 通过原审交抗诉书
     cuz 二审抗诉是 向上一级抗，通过同级检  对 未生效裁判 提抗诉

# 2.抗诉（only书面）
> 1）only 申请检院对判决抗诉；2）对裁定抗诉 only 检院自己来
## 2.1 二审抗（县检 only 提二审抗诉）
###   1）if 基原一审生效裁判；
         区检 向 中院 提二审抗诉（下抗上）
         市检 代 区检 出庭
         区检 文书 送 基院（文书同级送）
###   2）if 上级检 认为 下级检 二审抗诉不当；
              可以直接向 同级法 撤回抗诉 + 通知下级检院

# related:再审抗诉(最高检 only 提再审抗诉)
> % 死、抗 必上 审委会 %
## 1)if 上级检 发现下级法 已生效 判决裁定错误；
          1）直接向同级法提出抗诉
             市检 发现 基院 错；向中院提再审抗诉
          2）指令下级法的 上一级检 向 同级法提出抗诉
             省检 发现 基院 错；向高院提再审抗诉||指令市检 向中院提再审抗诉
## 2）同级抗诉
      县法做出生效判决；市检向市中院抗诉

# 3.判决
## 3.1 结果
        if 没错；维持
        if 法律、量刑错；应当改判
        if 事实不清证据不足；可改可发
        if 程序违法；N 次发
           incl:没公开、回避、剥夺限制法定诉讼权利、审判组织的组成不合法
## 3.2 if 二审发回，原审法不得加刑；
         exp:有新的犯罪事实 + 检院补充起诉

# III.再审
# 1.提起主体
## 1.1 法院院长 && 审委会
###   1)if 针对本院生效判决；做出生效判决的 院长&&审委会
            ie.院长发现后，应当提交审委会讨论决定 是否再审
###   2)if 针对其他法院生效判决；最高法、上级法 (if 上级法发现 生效判决 有错)
####   2.1）自己提审
            1）适用法律错
            2）疑难重大复杂
            3）原审法 不宜审理
####   2.2）指令再审
            1）usu 指令原审 以外的
            2) exp 指令原审
                     if 原审有利于查明事实，纠正错误的

## 1.2 检院再审抗诉
       C_3.1

# 2.特点
## 2.1）必须经 有权法||检院 审查 才能提起
## 2.2）方式
         1）决定再审
         2）指令再审
         3）决定提审
         4）再审抗诉
## 2.3）usu 不加重刑罚
        exp:1)检院抗诉
            2）只针对部分被告抗诉，不得加重同案犯的刑罚
## 2.4）再审期间对原判决、裁定的执行
         usu:不停止
         exp:if 被告人可能经再审改判无罪、可能经再审减轻院判刑罚导致刑期届满的；
                   可以决定中止原判决、裁定的执行；
                      if 必要时；
                            可以对被告人采取取保、监居

# IV.复核
# 1.死立即复核（only 2个应当：检院应当讯问 原审D、最高院应当结果通报最高检）
> 1）可以判但是不生效;2）一案一报，层层上报
## 1.1 二审程序
        if 中院一审 死立即 + 上诉||抗诉 
              if 高院 裁定维持；
                    10days 内报最高法核准
               else if 高院认为不应当判处死刑;
                    直接用二审改判

## 1.2 复核程序
###   1）if 中院一审 死立即 + 未上诉、抗诉 + 期满后10days 内 报高院复核
            if 高院同意；
                    10days 内报最高法核准
            else if;
                    应当按照二审 提审 || 发回重审
###   2）if 高院一审 死立即 + 未上诉、抗诉 + 期满后10days 内 
              报 最高院复核

## 1.3 复核结果
>  % 没错就核准，见错就发回，1/N 不该死可改判 % ; % 1人1罪 绝对不会改判 %

### 1）if 没错 || 引用法律条款有瑕疵；
                   补正瑕疵之后 裁定核准
### 2）if 数罪 + 每个罪都是死立执 + 量刑错；
                   可以 + 判决改判 + 判决核准其他人

# 2 死缓核准结果
## 2.1）核准 S 死立即
        1)直接核准
        2）纠正后核准
## 2.2）发回、改判 S 二审判决
> only DIFF: 二审改判可以加重，复核不加刑

# 3.法定刑以下判处刑罚的复核
> 1)上一级法only 对案件在 程序上 同意||不同;2)上诉抗诉的，按照二审审理 + 再复核
## 3.1.服一审（int:if 中院判;不生效;上诉抗诉期满3days 内 报高院 复核
###    1)if 高院 同意；书面+层报 最高法 核准
           if 最高法认为原裁判 正确；裁定核准
           else if ；不予核准+应当撤销原判决、裁定+发回原审||指令其他下级法重新审判
###    2)else if 高院 不同意；
           应当裁定发回 ||  改变管辖按照一审重新审理
                like: if 基层判； 高院 可以指定中院 按照一审重新审理        

## 3.2.上诉、抗诉的(先经过二审程序，再复核)
###   1)if 无理；应当裁定驳回；维持原判
              按照 1 的程序 层报 最高院
###   2)if 有理；应当依法改判

# V.撤
# 1 一审中的撤诉（ALL 审查）（***）
## 1.1）if 公诉 + 判决宣告前；审查有罪||无罪
         if 有罪；不允撤诉；
         else if ;撤
## 1.2）if 自诉；审查有无违反自愿
         自诉撤回之后 不能 再起诉
         （ D && 民诉：一审撤回起诉后可以再起诉，cuz 重一审）

# 2 二审
> 1)期满前撤诉；一审的判决、裁定在 上诉、抗诉期满之日起 生效 2）期满后撤诉；准许撤诉的裁定书 送达之日起生效
## 2.1 撤回上诉
###   1）if 期限内；无需审查+应当准许 
        撤回之后还可以再上诉（以期满前最后一次议事表示为准）
       （ D 民诉：二审只能上诉一次，撤回后不能再上诉，刑诉重二审）
###   2）if 期满后；二审法院 应当审查 
          if 1）一审裁判正确；准许撤诉
             2）检院抗诉 + 不派员出庭 + 不说明原因；
                               法院 可以裁定 按照撤诉处理
          else if 1）一审错了；继续审理
                  2）死立即 + D 上诉案件 + 开庭审理后
                                  应当不予准许，继续审理 

## 2.2 撤回抗诉
###    1）if 期限内；一审法不再向上一级法移送案件
###    2）if 期满后；可以裁定准许 + 通知一审法、当事人

# 3 再审申诉的撤回
## 3.1）if 申诉人 再审期间内 撤回申诉；
             应当裁定准许
## 3.2）if 申诉人 不来||先走；应当裁定 按撤回申诉处理
          exp 申诉人不是原审当事人

# 4 再审抗诉的撤回
## 4.1）if 检院 再审期间内 撤回抗诉；
             应当裁定准许
## 4.2）if 期满后；S 上诉||二审抗诉的撤回 
## 4.3）if 检院 不来||先走；可以裁定 按撤回申诉处理

# 5 减刑假释撤销
## 1）if 执行机关+书面+撤回建议+在法院作出裁定前;
            法院决定是否准许
## 2）if 法院裁定减刑假释；
###  2.1）7days 内 
                送达 报请的执行机关、同级检院、本人
### 2.2）if 假释  + 送达 社区矫正机构||继承组织
### 2.3）if 撤；
          1）一经做出，立即生效，不得上诉抗诉 
          2）撤销裁定书 
             2.1）送交 罪犯居住地的 县级司法行政机关->
                             县级司法行政机关 将犯罪交付执行
             2.2）同时抄送  罪犯居住地 同级 公检

# 6 附民中撤
## 6.1）if 检院 撤回起诉（曾提过）；
               对附民调解，不成的 应当裁驳 + 告知另诉
## 6.2）if 检院 部分撤销案件（视为没提过）、不起诉
               对附民调解，不成的 应当裁驳 + 告知另诉
