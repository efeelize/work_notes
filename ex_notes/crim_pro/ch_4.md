> 公安 + 应当 + 收到申请后30days 内 + 做出决定 + 书面告知申请人

              原机关 在收到复议申请后 7days 内 做出决定 + 书面告知控告人；

###  3rd: if 重大复杂；

## 2.2 找检院 控告申诉检察部门 投诉；
### 1) 公安没错
#### 1.1) if 不应当追究刑事责任；
                  及时答复投诉人||行政执法机关
#### 1.2) if 不属于公安管；
                  告知投诉人有管辖权的机关 + 建议向该机关控告、移送
#### 1.3) if 公安还没做出不予立案的决定；
                  移送公安

### 2) 公安错
> % 先要求说理，后通知立案 %
#### 1st: ALL 检察长批准 + 应当要求公安书面说明理由
          1)不立案；
          2)违法立案；
                like:插手民事、经济纠纷、利用立案报复陷害、敲诈勒索
          3)违法采取强制措施； 
#### 2nd: if 检院要求公安说明理由； 公安 应当在7days 内说明
          if 理由不成立；
                 检察长||检委会决定 + 应当通知公安+ 立案||撤案
             公安应当 1)15days 内 立案
                     2)立即 撤案

### 3) 公安已经立案，随后撤案(已经不适用立案监督的方式，直接公转自)
      like:公安对甲涉嫌过失致人死亡立案，随后撤案->
             死者乙的近亲属应当直接向法院对甲提出刑附民

# III.补侦
# 1.批捕（公提请批捕，公安补侦）
> 审批阶段的补侦以不批准逮捕为前提，此时案子在公安
## 1）程序
       if 检院批捕；
            公安应当立即执行 + 执行情况及时通知检院
       else if 不批；
               检院 应当说明理由
                       if 需要补侦；
                            应当同时通知公安补侦
## 2）救济
     if 公安认为不批捕有错误；
                 向原检院复议，向上一级检院复核

# 2.审查起诉
- 检院无论在何时发现事实不清、证据不足都可以补侦，but ABC 3个检总共只有2次补侦的机会(incl 改变管辖前后)
- 存疑不起诉不以补侦为前提，以事实不清、证据不足为前提
- 2次补侦是退回了2次，第3次不能再退回

## 2.1）检院审查起诉部门自行侦查
         在审查起诉期限内完成-> 
         不给多余的时间
## 2.2）退回公安补侦
         1次1mon.最多2次 
## 2.3）if 经过2次补侦 + 检院审查起诉中又发现了新的犯罪事实的；应当移送侦查机关立案侦查；
           if 有已经查清的犯罪事实；
                    应当依法提起公诉
        like:甲乙共同实施抢劫 + 经过2次补侦后检院发现甲曾经实施过诈骗；
                将新发现的犯罪移送公安立案侦查 + 对查清的犯罪事实提起公诉

# 3.监察委补充调查
> 监察委没有侦查

    if 事实清楚、证据充分+依法应当追究刑事责任；
                    应当起诉
    else if 需要 补充核实；
                    1）应当退回监察机关 补充调查
                       1次1mon.最多2次
                    2）检院自行 补充侦查

# 4.审判（only 检院补侦）
## 4.1 检院主动提
###    1) if 检院提出延期审理建议；
             法院 应当同意
             检院 应当自行侦查 + 必要时 公安协助
####     1.1） if 在期限内 + 检院补充证据移送法院了；
                 应当通知 辩、诉 查阅、摘抄、复制
####     1.2） else if 超出期限 + 经法院通知 + 检院未移交案件到法院 + 未说明原因；
                 按照撤诉处理

###    2) if 在宣告判决前 + 检院发现 + 
####    2.1) D的真实身份||犯罪事实 与起诉书不符||事实证据没有变化 + 罪名、适用法律与起诉书不一致的；
             可以变更起诉
####    2.2) 发现遗漏同案犯 || 罪行可以一并起诉审理的；
             可以追加、补充起诉

## 4.2 法院建议补
###   1）if 新的立功线索；
          if 一审被告人提出新的立功线索；
                可以建议 检院 补充侦查
          if 合议庭自己发现 检院遗漏了 立功等量刑情节；
                应当通知 检院 补充移送
###   2）if 改变管辖；
          1）可以通过 原受理检院 退回 原公安 补侦
          2）原 检院 自己补侦

          应当当庭询问被告人对指控的犯罪事实的意见 + 告知被告人适用简易程序审理的规定 + 
              确定被告人是否同意适用简易程序
       3）被告人 对适用简易程序没有异议
          ！incl:被害人、法代、辩

## 1.2 消极条件
       % 有缺陷、有影响、不认罪 % 
       （民诉：下落不明重再审，人多公益要撤销）
       1）盲聋哑
       2）半疯傻
          ie.尚未完全丧失辨认||控制自己行为能力的精神病人
       3）重大社会影响
       4）共犯中部分人 不认罪|| 不想适用简易
       5）辩护人 无罪辩护||轻罪辩护
       6）有犯罪事实 但是不构成犯罪

# 2.审理
    % 刑诉：独任一定简易，简易未必独任，陪审员只有一审可以有 %
    % 民诉：简易一定独任，独任未必简易，陪审员只有一审普通可以有 %

    1）一审基层
    2）<3years,可独可合，20days 内审结
       >3years,必须合，1.5mon 内审结
    3）应当通知 辩护人出庭
       if 法援；必须有辩护人
          else if;辩护人在开庭前可以将辩护意见提交法庭
    4）一般 应当 当庭宣判
      diff && 速裁程序；必须当庭宣判

# 3.简易转普通
    ie.不适用简易的消极条件
    3.1）审理期限从 决定转为 普通之日 开始计算
    3.2）原起诉仍然有效，检院无需另行起诉
