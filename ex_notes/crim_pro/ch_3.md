# I.变更强制措施
> 任何一种都可以变更为更轻||更重的强制措施

# 1.羁押必要性审查（only逮捕后）（***）
## 1）启动
###  1.1）依职权
         刑事执行检察部门 对本院||同级法 决捕 + 应当初审
###  1.2）依申请
         嫌疑人、D && 其 法、近、辩 申请 + 应当说明理由

## 2）初审
### 2.1）立案
        检察长、分管副检察长 + 批准
### 2.2）不予立案
        检察官 + 决定 + 书面告知申请人

## 3）立案后的审查
### 3.1）公开审查
        exp:国密、商业秘密、个人隐私
### 3.2）综合评估
        Incl:身体状况、案件进展情况、可能判处刑罚
### 3.3）量化方式（得分作为参考）
        incl:加分、减分、否决项目等具体标准

## 4）检院应当建议变更羁押，公安实施(else都是可以建议)
> % 量刑非羁押、刑期已折抵、羁押已超期 %

      1）可能判处拘、管、独立附加、免于刑事处罚、判决无罪
      2）继续羁押将超过 依法可能判处的刑期
      3）符合 取保||监居
      4）没有证据证明有犯罪事实 || 有证据证明不是他干的

# 2.变更逮捕
## 2.1）应当变更（% 期限届满 %）
> 公检法 收到变更强制措施的申请后，应当在3days内 做出决定；
    1）一审判处管制、宣告缓刑、单独附加刑 + 判决尚未生效
    2）二审 + 羁押时间已经满了一审判处的刑期
    3）案件不能再法定期限内 审结
# 2.2）程序
#### 1）if 公检法+发现采取不当强制措施；
                应当撤销||变更
#### 2）if 公 释放 || 变更 逮捕；
                应当通知 + 原批准检院 + 备案
#### 3）if 下级检院 发现 上级检决捕错了；
                应当自行释放||变更 + 向上一级检院 + 报告
