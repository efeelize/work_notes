# ch_2 制度
# I.管辖
## 1.级别管辖
### 1.1 一般规则
       1）上可审下
          ie.必要时
       2）下不可审上
          usu:应当由上级管辖的，不能再指定下级
          exp:if 缓刑、假释期间又犯新罪
                 审判新罪的法院 一并 撤销缓刑、假释
              if 上级法作出的缓刑、假释
                 下级法可以撤销
       3）就高不就低
          usu:一人||一罪 属于 上级法，全案移送
          exp:未成年人可以分案

### related 
           1）漏罪
              原审地法院管辖
              更为合适的；罪犯服刑地||犯罪地
           2）新罪
              服刑地法院管辖
              unless 犯罪地抓获+发现其在脱逃期间犯的罪；犯罪地
              if 死缓期间脱逃+在犯罪地抓获;死缓执行地中院 执行死刑
           3）强制医疗
              % 暴力行为地为主，被申请居住地为辅，基层法院 + 合议庭 % 
              3.1）usu 被申请人实施暴力行为所在地 + 基院
              3.2) exp 更合适的 + 被申请人居住地 + 基院
              3.3）精神病不能适用简易程序
           4）减刑、假释
              4.1）无期
                   服刑地+高院+裁定
              4.2）有期（incl 无期减为有期）、拘、管
                   服刑地+中院+裁定
                   4.2.1）无期减为有期
                          从裁定减刑之日 起算刑期
                          已经执行的部分，不计入减刑后的刑期
                   4.2.2）其他减刑
                          已经执行的部分，计入减刑后的刑期

### 1.2 指定管辖
#### 1.2.1 情形
           1）管辖不明
              inti:B 中院为受案法，高院 指定 A中院审理, 
                   if 公诉；
                      案卷退回 B 检院；B 检院 移送 A 检院；A 检院 向 A 中院起诉
                   if 自诉
                      案卷直接移送 A中院
           2）管辖不宜
              ie.A省 省委书记 不能在 A省 审判
                 上一级法 自己管||指定 提出请求的同级其他法
           3）规避管辖
              ie.针对检院规避法院的级别管辖
                 inti: 中院 一审，高院 二审 ，A 上诉
                       if 高院 二审发回重审，市检撤诉，
                          市检 让 区检 向 基院 提起公诉
                       else 基院 应当层报 高院
                            高院 可以决定移送 中院 || 其他法院
           4）监察法
              ie.可上可下
              4.1）上级  可以指定 下级||其他 监察机关管辖
              4.2）重大复杂 ，可以 报请 上级  监察机关管辖   

#### 1.2.2 程序
           1）争议
             1.1）都有管辖权
                  S_2.1 3）
             1.2）管辖权争议
                  1）应当在审限内协商
                  2）不成的，分别逐级报请 共同上级法 指定管辖
             1.3）监察法
                  由共同上级 监察机关确定
           2）上级法的处理
              需要移送中院的，应当在报请院长决定后,最迟审限届满前15day前+书面请求移送
              
              if 受理前；中院接到申请后10days 内做出决定
                       if 不同意移送；下达不同意移送决定书
                          else if 同意；下达同意移送决定书 + 书面通知同级检 
                
              if 受理后；应当依法审判

# III.证据制度
## 1.基本原则
### 1.1 证明力
        1）关联性规则
           1.1）证据本身与案件联系的紧密程度
           1.2）关联性是证据证明你的原因
                but  有关联性的证据未必具有可采性
        2）补强证据规则
           2.1）2个证据都要有证据能力
           2.2）有独立的来源
                   like:有利的人说了有利的话
                        限人作证 + 可以正确认知、表达
                   ！:根据被告人供述找到的证据，只是互相印证

### 1.2 证据能力
        1）证据裁判规则
              diff && 神明裁判

        2）自由心证
              like：存在多个鉴定意见的时候，由法官进行判断

        3）自白任意||非任意自白排除规则
           3.1)^基本确立
           3.2)严禁刑讯逼供
           3.3)diff && 非法证据排除

        4）传闻证据规则
           ie.真的证人+没有出庭
           ^ 未确立严格的传闻证据排除规则

        5）意见证据规则
           5.1）only 证人
                ！incl:专家辅助人、鉴定人
           5.2）可以采纳
                5.2.1）醉酒、麻醉、中毒 + 能正常感知||正确表达
                5.2.2）猜测、评论性、推断性 + 根据一般生活经验判断符合事实

        6）最佳证据规则||原始证据规则
           6.1）only 文字材料
           6.2）^ 未明确规定

        7）证据转换规则
           7.1）行政 转 刑事
                7.1.1）物证可以在刑诉中作为证据
                       1）物证、书证、视听资料、电子数据
                       2）勘验检查笔录 + 检院审查认为符合要求
                7.1.2）言辞
                       1）usu 应当重新收集
                       2）exp
                         2.1)鉴定意见 + 检院审查认为符合要求
                         2.2）if 供述、证言、陈述;
                                +来源、收集程序合法+其他证据相互印证+检院审查认为符合要求
           7.2）监察 转 刑事
                都可以转，cuz 是按照刑事标准收集的
