# ch_6 特殊程序

# I.非法证据排除(***)
    1）if 嫌、D 及其辩护人 申请排除非法证据;
              应当提供涉嫌非法取证的人员、时间、地点、方法、内容等相关线索||材料
    2）非法证据只对书证、物证进行排除；
       其只是排除规则中的一种

# 1 排除
##   1）言辞证据 绝对排除
> 引诱 + 取得嫌疑人、D供述 、证人证言、被害人陈述 不排除
###    1.1）嫌疑人、D + 供述(！incl:辩解)
            1）刑讯
               incl:肉体、精神
            2）冻、饿、晒、烤、疲劳审讯
            3）在规定的办案场所以外 + 取得的供述
            4）未依法全程录音录像的（无期、死刑）
            5）威胁 + 本人 && 近亲属 + 痛苦
            6）非法拘禁等限制自由
            4）重复自白/供述
                 if 先受到刑讯，后基于先前影响作出重复性供述；排除
                 else if 更换了侦察主体||更换了诉讼阶段；！排
###    1.2）证人证言、被害人陈述
             1）暴力、威胁
             2）限制自由
##   2）物证书证  相对排除
           只要可以补正||作出合理解释

# 2. 侦查、审查起诉期间 
##  1）启动
###   1.1）依申请
          if 嫌疑人、辩+ 向检院申请 + 提供线索；
                               检院 应当 调查核实
          else if 只是说自己受到刑讯；
                               检院 可以 调查核实      
###   1.2）依职权
           重大案件 + 驻看守所检察人员 + 应当在侦查终结前讯问嫌疑人 + 同步录音录像

##   2) if 确实刑讯了
###   2.1)侦查期间
####      1）检院 
             应当 向侦查机关 提出纠正意见
####      2）侦查机关应当予以排除，不得作为提请批捕、移送审查起诉的根据
####      3）重大案件
             驻所 检察人员 应当 + 侦查结束终结前核查是否存在刑讯、非法取证 + 同步录音录像
####      4）非法取证的人员
             1）应当依法作出处理
             2）另行指派 原机关的 侦查人员 重新调查取证
###   2.2）审查逮捕、起诉期间
####      1）讯问时
             应当告知 有权申请排除非法证据
####      2) 应当 依法排除相关证据 && 提出纠正意见，必要时检院自行调查取证
####      3）被排除的非法证 应当 随案移送 + 写明是 依法排除的非法证据
                   cuz 应当由法官进行判断

# 3. 一审排除
## 3.1 庭审前 ；有线索 应当在开庭前提
> 法院 向 被告人 及其 辩护人 送达起诉书副本 + 应当告知 其有权申请；
###   1)if 提出线索；
         1）法 + 应当在开庭前 + 申请书&&相关线索材料的复印件 送交 检院
         2）法 + 应当召开 庭前会议审查 ->
                        检院 应当 通过出示有关证据材料的方式 对证据收集的合法性进行说明 ->
                        公诉人、被告人、辩护人 可以就证据收集的合法性进行协商->
                        法院 可以 核实情况，听取意见
###   2)if 检院 撤回 有关证据；
             撤回的证据 + 没有新的理由，不得在庭审中出示
###   3)if 被告人 && 辩护人 撤回 申请；
               没有新的线索||材料；不得再次提排除申请

## 3.2 庭审中 ；被告人、辩 应当说明理由
### 1)if 庭审期间才发现线索的；应当进行审查
      + 法院 可以选择 在申请时||法庭调查结束前一并审查 + 决定是否进行调查
### 2)$:
#### 2.1) if 庭前提 + 对合法性有疑问的；
             应当进行调查
          if 庭前会议中 公诉人、被告人、辩护人 就证据收集合法性 未达成一致意见 + 法院有疑问;
                    应当在庭审中进行调查
#### 2.2) if 庭审中提 + 对合法性有疑问的；
             应当 先行当庭调查 || 在法庭调查结束前进行调查       

# 4. 法庭 调查核实
> 庭审中提的；应当当庭说明情况和理由
## 1) 没有引起疑问；直接驳回 + 继续法庭审理 
> 法院宣读 + 出示结合其他证据 确定是否作为定案根据
###  1.1）被告人、辩护人 没提供线索||证据；
###  1.2）被告人、辩护人提了 + 法院认为合法        
###  1.3) 庭前会议中就证据收集合法性 未达成一致意见 + 法院认为合法 + 没有新的线索和材料；
          法院可以决定不再进行调查

## 2) 引起疑问;控方对证据合法性进行举证
###  2.1）侦查人员出庭作证
####     1）检院向法院申请，法院依 检院申请 通知
#####      1.1）主体
                侦查人员、看守、监管、同监、录音、录像、翻译人员
#####      1.2）材料 
                1）讯问笔录、提讯笔录、体检记录、
                2）采取强制措施||侦查措施的法律文书、
                3）侦查终结前+对讯问合法性的核查、
                4）有针对性的+播放讯问录音录像
####     2）法院 可以依职权 通知
            1）此时仍然是侦查人员 
            2）被告人、辩护人向法院申请 + 法院认为有必要
            3) if 被告人、辩护人 提出线索 + 申请法院播放特定时段 的录音录像;
                      侦查人员出庭；应当向法院说明证据收集过程 + 就相关情况接受发问
                           if 发问方式不当||内容无关；
                                         法庭应当制止
####     3）侦查人员提要求，法院通知
###  2.2）提供书面材料说明
          1）证明力稍弱
          2）公诉人提交的取证过程合法的说明材料，应当经有关侦查人员签名 + 加盖公章
                    else if;不得作为证据使用
          3）公诉人、被告人、辩护人 进行质证、辩论

# 5.调查后法庭的处理
## 5.1）general principle
###    1）调查后，应当当庭做出是否排除有关证据的决定；
                if 必要；可以宣布休庭 + 合议庭评议||提交审委会讨论 + 再次开庭时宣布
###    2）做出决定之前，不得对有关证据 宣读、质证

## 5.2）details
###    1) if 确认||不能排除 非法取证；
                 应当予以排除
###    2) if 公诉人证据确实充分；
                 应当结合被告人当庭公诉 && 其他证据 确定 能否作为定案依据
###    3) if 公诉人不提供证据||提供了不充分、确实的证据 + 证据不能作为定案依据；
                 直接排除 + 不宣读、质证、出示

## 5.3）通知
        1）调查结论 应当告知 公诉人、当、辩、诉
        2）审查、调查结论 应当在裁判文书中写明 + 说明理由

# 6. 二审排除
## 6.1 if 一审未审+以有关证据作为定案依据+可能影响公正审判；
             二审法 可以裁定撤销原判，发回重审
## 6.2 if 检院、被告人、法代 上诉、抗诉 + 就证据收集合法性审查、调查结论 + 提出异议；
       二审法 应当审查
        but 检院在一审中就应出示全部的证据
             if 一审中检院未出示证据证明 + 法院依法排除有关证据；
                    在二审中不得出示之前未出示的证据
## 6.3 if 被告人、辩护人 + 一审后才发现线索 + 应当说明理由
             二审法 应当审查

# 7. 二审审理中  应当排除的证据；法院 可以依法排除
          if 原判对；应当 裁定驳回上诉、抗诉
          if 法律错||量刑错；应当改判
          if 事实不清，证据不足；可以裁定撤销原判，发回重审

# II.公诉案件的和解，之后检院酌不诉
> 公诉和解制度 构建了 多元化的纠纷解决机制:cuz 是特别程序
# 1 适用条件
> % 民间人财三年下，七年过失渎职外 %

      1）民间纠纷引起
       2）刑法4、5章 人身权、财产权
       3）可能判处有期3years 以下（宣告刑）
       4）7years 以下过失犯罪 + 渎职犯罪除外
          ！incl:暴力干涉婚姻自由
       5）嫌疑人、被告人+5years 内未曾故意犯罪
          ！care 现在犯的是什么罪||曾经的罪是否追究
       6）嫌疑人、被告人+真诚悔罪
       7）案件事实基本清楚
       8）嫌疑人、被告人 获得 被害人谅解 +  被害人自愿和解

# 2 和解范围 
       1）incl 民责
               like:赔偿损失、赔礼道歉
       2)!incl 法院职权范围内
               like:事实认定、证据采信、法律适用、定罪量刑

# 3 和解主体
       1）被害人
       2）嫌疑人、被告人、法代、近亲属
          only 赔礼道歉 必须本人
       3）法院 主持制作和解协议
          if 事实清楚、证据充分，法院应当告知当事人可以自行和解；
               if 当事人提出申请的；
                      法院可以主持双方当事人协商 达成和解

# 4 程序
## 4.1 if 一开始；当事人自行和解
## 4.2 if 在公安；公安 可以 向检院 提出不起诉的建议 || 建议检院向法院提出从宽的建议
            检院在审查逮捕、审查起诉的时候 应当充分考虑 公安的建议
            和解协议 应当及时履行
## 4.3 if 在检院；
                1）if 应当起诉；可以 向法院提出从宽的建议
                2）if 犯罪情节轻微+不需要判处刑罚；酌不诉
                3）if 未成年；附条件不起诉
           应当在签署后 till 检院做出从宽处罚决定前  立即履行
              if 被害人同意+提供有效担保；可以分期
## 4.4 if 在法院；only 不能判无罪
###    1）对公检主持制作的和解协议书；
                   if 当事人异议；法院应当审查
                      if 自愿合法；予以确认+无需重新制作和解协议书
                         else if；应当认定无效

###    2）对达成和解协议的；法院可以从宽处理（应当从轻处罚）
                   if 符合非监禁；应当适用非监禁
                   if 判法定最低刑仍然重；可以减轻处罚
                   if 犯罪情节轻微+不需要判处刑罚；可以免除刑事处罚
                   if  共犯;对达成和解协议的部分人 可以从宽处罚
            if 不能及时履行全部赔偿；法院应当制作 附民调解书
###    3）if 侦查、审查起诉 + 已经和解 + 又提附民 （S 附民调解）
                    不予受理
                    else if 1.2.1) 没有履行协议
                            1.2.2) 违反自愿
                            1.2.3) 违反合法
                            可以再受理

# III.附民
# 1. 成立条件（***）
## 1.1 刑诉成立为前提
       exp:if 公诉案件 + 检院撤回起诉；
                        对附民调解，不成另提民诉
## 1.2 P 
### 1）检院公益诉讼,检院为 P
> only 在刑附民中，在追究犯罪的同时代表国家||集体的时候提
         1）国家、集体财产遭受损失
           ！incl:个人
         2）前面的人没有提附民
         3）判决 是向受损的单位、单位继受人 赔偿
              if 没有继受人；应当判令向检院 交付赔偿款->检院上交国库
### 2）限人为 P（incl：未成年 + 精神病） + 法代代为提起附民
### 3）死人（incl 植物人）+ 近亲属为 P

## 1.3 D
###   1)不能列为D
         1）亲友
            自愿代为赔偿的，应当准许
         2）共犯中在逃的
               cuz 刑责不能缺席判
###   2）可以列为D
####   2.1)死刑犯’遗产继承人
          直接将遗产继承人列为 D + only 调解结案，不成另民诉
             cuz 执行之后刑诉不存在
####   2.2）共犯中 死的人’遗产继承人
           直接列为附民 D，正常审理
             cuz 刑诉还在

## 1.4 犯罪行为引起的物质损失
### 1）不赔
####  1.1）死亡赔偿金、残疾赔偿金
            exp:交通肇事 + 附民提
####  1.2）精神损失费
           单独提民诉也不赔
####  1.3）属于国赔的部分
            like:交管所的人执法导致...
                 虐待被监管人员...
                 非法搜查...
####  1.4）赃款赃物（走退赔程序，在刑诉中解决）
         1）3W 项链
           1.1）卖了1K;退赔1K;剩下2.9W 另外提民诉
                cuz 退赔程序是刑诉的终结
           1.2 卖了5W;退赔3W;剩下2W 追缴国库
         2）合同诈骗获利200W;追缴
### 2）赔
#### 2.1 误工费
         证人不补
#### 2.2 直接损失
        if 安装假肢 + 每隔3year 清洗一次；赔
## 1.5 符合管辖

# 2. 程序
## 2.1 附民应当制作调解书
###  1）if 没有达成调解||当事人反悔的； 附民 应当 一并判决
        调解协议可以分期履行
###  2) if 达成协议 + 当时执行完毕 == 可以不制作调解书
### related：自诉
       1）公转自不能调 
       2）亲告、轻微+当时执行完毕 == 也要制作调解书

## 2.2 提起
###   1）从立案到一审判决宣告前
###   2）if 二审提附民；调解不成，另提民诉
###   3）侦查、审查起诉提 + 已经调解 + 又提附民;不予受理
           else if; 可以再受理
                    1) 没有履行协议
                    2) 违反自愿
                    3) 违反合法

## 2.3 保全（refer 民诉部分）
###   1)诉中 + 3个人 + 可以
        检院、法院(P 不提)、P
###   2）诉前(诉前 ！equal 开庭前)
        财产所在地、被申请人居住地、有管辖权法

## 2.4 审理
###   1）一审
####    1.1）出庭
               P 不来||先走 + 1次通知 == 撤诉
               D 不来||先走 + 1次通知 == 缺席判
                 (Diff:自诉2次通知)
####    1.2）审理
####        1)if 被告没有犯罪；
                 对附民进行调解，不成的 应当一并做出刑附民判决
                        cuz:此时只是不承担 刑责
####        2)同一审判组织审理(刑庭审理、2个判决、1个判决书、分开生效)

###   2）二审
####   2.1）上诉(上诉、抗诉 期限 obey 刑诉)
#####      1)if 只上诉刑事；
                    附民生效；
                      if 民事有错；
                            民事再审纠错
#####      2)if 只上诉民事；
                    刑事生效；
                      if 刑事有错；
                            刑事再审纠错 + 对民事部分一并审理
                               cuz 刑事部分错必然导致民事部分错

####   2.2）审理
#####      1)if 原告增加独立的诉讼请求 || 提出反诉；
                  调解，不成的 + 告知另诉
#####      2)if 原告增加赔偿数额
                  调解，不成的 + 一并裁判
#####      3)if 被告人对刑事上诉 + 附民原告只对民事上诉；
                      二审不得加刑

# V.没收（***）
# 1 适用条件
> % 贪恐重大逃1年 %

       1）贪污贿赂犯、恐怖活动 等 重大犯罪案件
       2）嫌疑人、被告人 + 逃匿1年后不能到岸 || 死亡
       3）应当追缴违法所得 && 其他涉案财产 + 检院可以 向法院提出没收违法所得申请
          违法所得incl:直接||间接||转化（投资）

# 2 申请程序
| 阶段 |  主体  |   程序  |
| ---- | ----- | ------ |
| 侦查 |  公安 | 县级以上公安机关负责人批准+公安应当写出没收违法所得意见书 + 移交检院 |
| 审查起诉 | 检院 | 直接启动没收 |
| 审判 | 检院 | 提出没收违法所得申请 | 

# 3 法院受理程序
## 3.1）应当公告
        1）15days 内发出公告
        2）期间6mon(alike 除斥期间)
        3）公告期满后审理
              cuz：防止没收了别人的财产
        4）if 属于违法所得的财产；
                     除了依法返还被害人外，应当裁定予以没收
                  else if 不属于违法所得的财产;
                             应当裁定驳回申请 
## 3.2）利害关系人申请参加诉讼
        1）usu:在公告期间提
           exp:合理说明理由的，应当准许
        2）开庭
           C_ch_1
## 3.3)结果
###    1）审理中
####     1.1) if 普通程序中；
                if 死；裁定终止审理
                if 逃；裁定中止审理
                      if 检院 另行 向法院提出没收；启动没收程序
####     1.2) if 没收程序中；
                if 抓获；终止没收程序
                if 死；继续没收程序，可以由同一审判组织 对 是否没收违法所得 继续进行审理
###    2）审结
              if 不在案；案外人提异议，再审纠错
              if 到案；被告人对没收提异议；
                      检院 向原做出裁定法院提起 公诉的；可以由同一审判组织审理

# VI.强疗 
# 1 适用条件
> % 暴力无责有危险 %
       1）实施暴力行为+危害公共安全||人身
       2）法定程序鉴定+依法不负刑责+精神病
       3）继续危害社会可能
   
# 2 启动

| 主体 | 先  |  如何符合强疗条件 |   后   |
| ---- | --- | ------------- | ------ |
| 公安 | 应当依法进行鉴定 | 应当撤销案件+写出强疗意见书，移送检院 | 在法院决定强疗前，由公安采取临时性保护措施 |
| 检院 | 应当做出不起诉决定 | 应当同时向法院提出强疗申请 |     |
| 法院  | 1）审理普通案件发现的，应当进行法医精神病鉴定+可以做出强疗决定；2）检院申请强疗+符合条件+应当做出强疗决定| etc | 做出决定5days 后，向公安送达强疗决定书&&强疗执行通知书，由公安将被决定强疗的人送交 强疗机构  |

# 3 程序
##  3.1 检院申请强疗
          if Y;
               法院应当 做出强疗决定
          if N; 
               应当决定 驳回强疗申请 + 退回检院
          if 精神病+不需要强疗；
               应当决定 驳回强疗申请
                   if 被告人已经造成危害结果；
                         应当 + 责令家属||监护人 看管医疗

## 3.2 一审普通中强疗
         if 普通发现强疗；
            应当进行法医精神病鉴定
               if Y; 
                 检院 应当在 庭审中 发表意见
                    （only 这里检院不是事后监督）
                 法院 应当判决被告不负刑事责任+做出强疗决定
             
               if N;
                  应当按照普通程序继续审理
             
               if 精神病+不需要强疗；
                  应当判决无罪||不负刑事责任
                     if else 被告人已经造成危害结果；
                                  应当 + 责令家属||监护人 看管医疗

## 3.3 二审普通中强疗
          if 二审普通中发现可能符合强疗；
             else 1)可以 按照强疗程序处理
                  2）可以 裁定发回重审

## 3.4 救济
###   1) 不服
          if 被决定强疗的人、被害人 及其 法代、近亲属 不服；
             向上一级法 申请复议；
                 复议期间不停止执行
                    cuz：决定一经做出，立即生效，不得上诉
          
          if 检院不服；在收到决定书后 20days 内提出书面纠正意见；
                     法院另行组成合议庭 +1mon 做出决定
                     (此处 S 减刑假释的监督)

###   2) 解除
####   2.1 申请
#####     1）强疗人 及其 近亲属 ; 向决定强疗 法院 提出
              if 第一次申请被驳回 + 6mon 后再次提出；
                    法院应当受理
              if 强疗机构未提供评估报告；
                    申请人可以申请法院调取
#####     2）强疗机构 ; 定期进行诊断评估
              if 不需要强疗的；
                    应当及时提出解除意见 + 报 决定强疗 法院 批准
                        if 强疗机构未提供评估报告；
                             法院应当要求其提供
####    2.2 法院审查
           1）应当组成合议庭 + 1mon 做出决定
                 if 决定解除；
                       应当通知强疗机构 在收到决定书的当日 解除
           2）做出决定的5days 后 决定书送达：
                 incL: 强疗机构、申请人、被强疗的人、检院
