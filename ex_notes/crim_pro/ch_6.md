# 1.非法证据排除
    if 嫌、被告 及其辩护人 申请排除非法证据;
              应当提供涉嫌非法取证的人员、时间、地点、方法、内容等相关线索||材料
## 1.1 排除
       1）言辞证据 绝对排除
          1.1）嫌疑人、被告人 + 供述
               1.1.1）！incl:辩解
               1.1.2）威胁+本人 && 近亲属+痛苦
               1.1.3）限制自由
               1.1.4）刑讯+痛苦
               1.1.5）重复性供述
                      if 先受到刑讯，后基于先前影响作出重复性供述；排除
                                   else if 更换了侦察主体||更换了诉讼阶段
          1.2）证人证言、被害人陈述
               1）暴力、威胁
               2）限制自由
        2）物证书证  相对排除
           只要可以补正||作出合理解释

## 1.2 侦查、审查逮捕、起诉期间 
        if 嫌疑人、辩+ 向检院申请 + 提供线索；
             检院 应当 调查核实
                else if 只是说自己受到刑讯；
                            检院 可以 调查核实
             
        if 确实刑讯了
           1)侦查期间
              1.1）检院 应当 向侦查机关 提出纠正意见
              1.2）侦查机关应当予以排除，不得作为提请批捕、移送审查起诉的根据
              1.3）重大案件
                   驻所 检察人员 应当 + 侦查结束终结前核查是否存在刑讯、非法取证 + 同步录音录像
              1.4）非法取证的人员
                   1）应当依法作出处理
                   2）另行指派 原机关的 侦查人员 重新调查取证
                  
            2）审查逮捕、起诉期间
               2.1）讯问的时候 应当告知 有权申请排除非法证据
               2.2）应当 依法排除相关证据 && 提出纠正意见，必要时检院自行调查取证
               2.3）被排除的非法证 应当 随案移送 + 写明是 依法排除的非法证据
                       cuz 应当由法官进行判断

## 1.2 一审排除
### 1.2.1 庭前 先审查
       if 庭审前 ；
          if 法院 向 被告人 及其 辩护人 送达起诉书副本 + 应当告知 其有权申请；
             有线索 应当在开庭前提
             if 提出线索；1）法院 + 应当在开庭前 + 申请书&&相关线索材料的复印件 送交 检院
                        2） 法院 + 应当召开 庭前会议审查 ->
                            在庭前会议中，检院 应当 通过出示有关证据材料的方式 对证据收集的合法性进行说明 ->
                            公诉人、被告人、辩护人 可以就证据收集的合法性进行协商->
                            法院 可以 核实情况，听取意见
                if 检院 撤回 有关证据；
                   撤回的证据 + 没有新的理由，不得在庭审中出示
                if 被告人 && 辩护人 撤回 申请；
                   没有新的线索||材料；不得再次提排除申请

       if 庭审中 ；被告人、辩 应当说明理由
           if 庭审期间才发现线索的；1）法庭 应当进行审查
                                2）法院 可以选择 在申请时||法庭调查结束前一并审查
                                   + 决定是否进行调查   

     <关于办理刑事案件严格排除非法证据若干问题的规定>
          if 庭前提 + 对合法性有疑问的；
             应当进行调查
                if 庭前会议中 公诉人、被告人、辩护人 就证据收集合法性 未达成一致意见 + 法院有疑问;
                    应当在庭审中进行调查
          if 庭审中提 + 对合法性有疑问的；
             应当 先行当庭调查 || 在法庭调查结束前进行调查       

### 1.2.2 法庭 调查核实
#### 1.2.2.1 没有引起疑问
             if 1）被告人、辩护人 没提供线索||证据；
                2）被告人、辩护人提了 + 法院认为合法
                else 法院宣读 + 出示结合其他证据 确定是否作为定案根据
                          
             if 庭前会议中 公诉人、被告人、辩护人 就证据收集合法性 未达成一致意见 + 法院认为合法 + 没有新的线索和材料；法院可以决定不再进行调查
             
             $:没有引起疑问；应当当庭说明情况和理由 + 直接驳回 + 继续法庭审理 

#### 1.2.2.2 引起疑问;控方对证据合法性进行举证
             1）侦查人员出庭作证
                1.1）检院向法院申请，法院依 检院申请 通知
                    1）主体incl
                       侦查人员、看守、监管、同监、录音、录像、翻译人员
                    2）材料 incl:
                       2.1）讯问笔录、提讯笔录、体检记录、
                       2.2）采取强制措施||侦查措施的法律文书、
                       2.3）侦查终结前+对讯问合法性的核查、
                       2.4）有针对性的+播放讯问录音录像
                1.2）法院 可以依职权 通知
                     1）此时仍然是侦查人员 
                     2）被告人、辩护人向法院申请 + 法院认为有必要
                     3) if 被告人、辩护人 提出线索 + 申请法院播放特定时段 的录音录像;
                           侦查人员出庭；应当向法院说明证据收集过程 + 就相关情况接受发问
                              if 发问方式不当||内容无关；
                                    法庭应当制止
                1.3）侦查人员提要求，法院通知
             2）提供书面材料说明
                2.1）证明力稍弱
                2.2）公诉人提交的取证过程合法的说明材料，应当经有关侦查人员签名 + 加盖公章
                        else if;不得作为证据使用
            3）公诉人、被告人、辩护人 进行质证、辩论

#### 1.2.2.3 调查后法庭的处理
             1）general principle
                1.1）调查后，应当当庭做出是否排除有关证据的决定；
                     if 必要；可以宣布休庭 + 合议庭评议||提交审委会讨论 + 再次开庭时宣布
                1.2）做出决定之前，不得对有关证据 宣读、质证
             2）details
                if 确认||不能排除 非法取证；
                       应当予以排除
                if 公诉人证据确实充分；
                       应当结合被告人当庭公诉 && 其他证据 确定 能否作为定案依据
                if 公诉人不提供证据||提供了不充分、确实的证据 + 证据不能作为定案依据；
                       直接排除 + 不宣读、质证、出示
             3）通知
                3.1）调查结论 应当告知 公诉人、当、辩、诉
                3.2）审查、调查结论 应当在裁判文书中写明 + 说明理由

## 1.3 二审排除
       if 一审未审+以有关证据作为定案依据+可能影响公正审判；
             二审法 可以裁定撤销原判，发回重审

       if 检院、被告人、法代 上诉、抗诉 + 就证据收集合法性审查、调查结论 + 提出异议；
             二审法 应当审查
             but 检院在一审中就应出示全部的证据
                 if 一审中检院未出示证据证明 + 法院依法排除有关证据；
                    在二审中不得出示之前未出示的证据

       if 被告人、辩护人 + 一审后才发现线索 + 应当说明理由
             二审法 应当审查

## 1.4 二审审理
       if 应当排除的证据；法院 可以依法排除
          if 原判对；应当 裁定驳回上诉、抗诉
          if 法律错||量刑错；应当改判
          if 事实不清，证据不足；可以裁定撤销原判，发回重审

# 2.公诉案件的和解，之后检院酌不诉
## 2.1 适用条件
       % 民间人财三年下，七年过失渎职外 %
       1）民间纠纷引起
       2）刑法4、5章 人身权、财产权
       3）可能判处有期3years 以下（宣告刑）
       4）7years 以下过失犯罪 + 渎职犯罪除外
          ！incl:暴力干涉婚姻自由
       5）嫌疑人、被告人+5years 内未曾故意犯罪
          ！care 现在犯的是什么罪||曾经的罪是否追究
       6）嫌疑人、被告人+真诚悔罪
       7）案件事实基本清楚
       8）嫌疑人、被告人 获得 被害人谅解 +  被害人自愿和解

## 2.2 和解范围 
       1）incl 民责
               like:赔偿损失、赔礼道歉
       2)!incl 法院职权范围内
               like:事实认定、证据采信、法律适用、定罪量刑

## 2.3 和解主体
       1）被害人
       2）嫌疑人、被告人、法代、近亲属
          only 赔礼道歉 必须本人

## 2.4 程序
       if 一开始；当事人自行和解
       
       if 在公安；公安 可以 向检院 提出不起诉的建议 || 建议检院向法院提出从宽的建议
            应当及时履行

       if 在检院；
                1）if 应当起诉；可以 向法院提出从宽的建议
                2）if 犯罪情节轻微+不需要判处刑罚；酌不诉
                3）if 未成年；附条件不起诉
           应当在签署后 till 检院做出从宽处罚决定前  立即履行
              if 被害人同意+提供有效担保；可以分期

       if 在法院；only 不能判无罪
                1）对公检主持制作的和解协议书；
                   if 当事人异议；法院应当审查
                      if 自愿合法；予以确认+无需重新制作和解协议书
                         else if；应当认定无效

                2）对达成和解协议的；法院可以从宽处理
                                  ie.应当从轻处罚
                   if 符合非监禁；应当适用非监禁
                   if 判法定最低刑仍然重；可以减轻处罚
                   if 犯罪情节轻微+不需要判处刑罚；可以免除刑事处罚
                   if  共犯;对达成和解协议的部分人 可以从宽处罚
            if 不能及时履行全部赔偿；法院应当制作 附民调解书

       if 侦查、审查起诉 + 已经和解 + 又提附民 （S 附民调解）
                    不予受理
                    else if 1.2.1) 没有履行协议
                            1.2.2) 违反自愿
                            1.2.3) 违反合法
                            可以再受理

# realted :附民
## 1. 当事人
#### 1.1 检院公益诉讼
         1）国家、集体财产遭受损失
           ！incl:个人
         2）前面的人没有提附民
         3）判决 是向受损的单位、单位继受人 赔偿
            if 没有继受人；应当判令向检院 交付赔偿款->检院上交国库

#### 1.2 不能列为附民被告
         1）亲友
            自愿代为赔偿的，应当准许
         2）共犯中在逃的
               cuz 刑责不能缺席判

#### 1.3 不赔
         1）死亡赔偿金、残疾赔偿金
            exp:交通肇事 + 附民提
         2）精神损失费
         3）抢夺手机->责令退赔
         4）属于国赔的部分
            like:交管所的人执法导致...
                 虐待被监管人员...
                 非法搜查...

#### 1.4 调解
         if 没有达成调解||前手前当事人反悔的；
            附民 应当 一并判决

         1）附民
            1.1）应当制作调解书
                 exp:达成协议 + 当时执行完毕 == 可以不制作调解书
            1.2）if 侦查、审查起诉 + 已经调解 + 又提附民
                    不予受理
                    else if 1.2.1) 没有履行协议
                            1.2.2) 违反自愿
                            1.2.3) 违反合法
                            可以再受理
         2）自诉
            2.1）公转自不能调 
            2.2）亲告、轻微+当时执行完毕 == 也要制作调解书

#### 1.5 程序
##### 1.5.1 一审
            1）出庭
               P 不来||先走 + 1次通知 == 撤诉
               D 不来||先走 + 1次通知 == 缺席判
                 diff && in 自诉:2次通知
            2）审理
               if 被告没有犯罪；
                     对附民进行调解，不成的 应当一并做出刑附民判决
                        cuz:此时只是不承担 刑责

##### 1.5.2 二审
            1）上诉
               if 只上诉刑事；
                    附民生效；
                      if 民事有错；
                            民事再审纠错
               if 只上诉民事；
                    刑事生效；
                      if 刑事有错；
                            刑事再审纠错 + 对民事部分一并审理
                               cuz 刑事部分错必然导致民事部分错

            2）审理
               if 原告增加独立的诉讼请求 || 提出反诉；
                  调解，不成的 + 告知另诉
               if 原告增加赔偿数额
                  调解，不成的 + 一并裁判
               if 被告人对刑事上诉 + 附民原告只对民事上诉；
                      二审不得加刑

#### 1.6 error
         1)不收诉讼费
         2）同一审判组织审理
            exp:确实不能
                 like 死了
         3）上诉、抗诉 期限 obey 刑诉
         4）保全措施
            4.1）被保全财产所在地、被申请人居住地、案件有管辖权的法院
            4.2）诉中财产保全，可以责令提供担保
            4.3）诉前 ！equal 开庭前

# 3.没收
## 3.1 适用条件
       % 贪恐重大逃1年 %
       1）贪污贿赂犯、恐怖活动 等 重大犯罪案件
       2）嫌疑人、被告人 + 逃匿1年后不能到岸 || 死亡
       3）应当追缴违法所得 && 其他涉案财产 + 检院可以 向法院提出没收违法所得申请
          违法所得incl:直接||间接||转化（投资）

## 3.2 申请程序
| 阶段 |  主体  |   程序  |
| ---- | ----- | ------ |
| 侦查 |  公安 | 县级以上公安机关负责人批准+公安应当写出没收违法所得意见书 + 移交检院 |
| 审查起诉 | 检院 | 直接启动没收 |
| 审判 | 检院 | 提出没收违法所得申请 | 

## 3.3 法院受理程序
       1）管辖
          犯罪地||嫌疑人、被告人 居住地中院 组成合议庭
       2）应当公告
         2.1）15days 内发出公告
         2.2）期间6mon(alike 除斥期间)
         2.3）公告期满后审理
              cuz：防止没收了别人的财产
       3）利害关系人申请参加诉讼
          3.1）usu:在公告期间提
               exp:合理说明理由的，应当准许
          3.2）开庭
               C_ch_1
       4)结果
         4.1）审理中
             if 普通程序中；
                if 死；裁定终止审理
                if 逃；裁定中止审理
                      if 检院 另行 向法院提出没收；启动没收程序
             if 没收程序中；
                if 抓获；终止没收程序
                if 死；继续没收程序，可以由同一审判组织 对 是否没收违法所得 继续进行审理
         4.2）审结
              if 不在案；案外人提异议，再审纠错
              if 到案；被告人对没收提异议；
                      检院 向原做出裁定法院提起 公诉的；可以由同一审判组织审理
                      if 裁定正确的；予以维持+不再对涉案财产作出判决
                         else if;应当撤销原裁定+在判决中对涉案财产一并作出处理
         4.3）上诉、抗诉
              1）嫌疑人、被告人 的近亲属 有独立上诉权
              2）其他利害关系人 需要授权
              3）检院

                 1）在调查中
                 2）depends

            2.3）询问
                 1）调查中
                 2）讯问证人等

## related:监察委留置权
### 1.条件
      1）代替拘留、逮捕
      2）已经掌握部分违法犯罪事实、证据

### 2.可以留置的情形
      1）S 拘留的两种情形：
           自杀逃跑在逃的
           毁灭伪造串供的
      2）涉及案情重大、复杂的
      3）有其他有碍调查行为的

         if 省级监察及噶啥了；
               应当报国家监察委员会 + 备案
         if 监察机关要求公安配合；
               公安应当协助

### 4.时间
      1）期限
         3+3（省级以下应当报上一级监察机关+批准）
         if 不应当留置；
                应当及时解除
       2）折抵
          留置1days
              折抵管制 2days
              折抵拘役、有期 1days
       3) 保障
          incl:饮食、休息、安全、医疗
               合理安排讯问时间、时长

# 4.强疗 
## 4.1 适用条件
       % 暴力无责有危险 %
       1）实施暴力行为+危害公共安全||人身
       2）法定程序鉴定+依法不负刑责+精神病
       3）继续危害社会可能
   
## 4.3 启动

| 主体 | 先  |  如何符合强疗条件 |   后   |
| ---- | --- | ------------- | ------ |
| 公安 | 应当依法进行鉴定 | 应当撤销案件+写出强疗意见书，移送检院 | 在法院决定强疗前，由公安采取临时性保护措施 |
| 检院 | 应当做出不起诉决定 | 应当同时向法院提出强疗申请 |     |
| 法院  | 1）审理普通案件发现的，应当进行法医精神病鉴定+可以做出强疗决定；2）检院申请强疗+符合条件+应当做出强疗决定| etc | 做出决定5days 后，向公安送达强疗决定书&&强疗执行通知书，由公安将被决定强疗的人送交 强疗机构  |

## 4.4 程序
### 4.4.1 检院申请强疗
          if Y;
               法院应当 做出强疗决定
          if N; 
               应当决定 驳回强疗申请 + 退回检院
          if 精神病+不需要强疗；
               应当决定 驳回强疗申请
                   if 被告人已经造成危害结果；
                         应当 + 责令家属||监护人 看管医疗

### 4.4.2 一审普通中强疗
         if 普通发现强疗；
            应当进行法医精神病鉴定
               if Y; 
                 检院 应当在 庭审中 发表意见
                    （only 这里检院不是事后监督）
                 法院 应当判决被告不负刑事责任+做出强疗决定
             
               if N;
                  应当按照普通程序继续审理
             
               if 精神病+不需要强疗；
                  应当判决无罪||不负刑事责任
                     if else 被告人已经造成危害结果；
                                  应当 + 责令家属||监护人 看管医疗

### 4.4.3 二审普通中强疗
          if 二审普通中发现可能符合强疗；
             else 1)可以 按照强疗程序处理
                  2）可以 裁定发回重审

## 4.5 救济
### 4.5.1 不服
          if 被决定强疗的人、被害人 及其 法代、近亲属 不服；
             向上一级法 申请复议；
                 复议期间不停止执行
                    cuz：决定一经做出，立即生效，不得上诉
          
          if 检院不服；在收到决定书后 20days 内提出书面纠正意见；
                     法院另行组成合议庭 +1mon 做出决定
                     (此处 S 减刑假释的监督)

### 4.5.2 解除
#### 4.5.2.1 申请
           1）强疗人 及其 近亲属
                向决定强疗 法院 提出
              if 第一次申请被驳回 + 6mon 后再次提出；
                    法院应当受理
              if 强疗机构未提供评估报告；
                    申请人可以申请法院调取
           2）强疗机构
                 定期进行诊断评估
              if 不需要强疗的；
                    应当及时提出解除意见 + 报 决定强疗 法院 批准
                        if 强疗机构未提供评估报告；
                             法院应当要求其提供

#### 4.5.2.2 法院审查
           1）应当组成合议庭 + 1mon 做出决定
                 if 决定解除；
                       应当通知强疗机构 在收到决定书的当日 解除
           2）做出决定的5days 后 决定书送达：
                 incL: 强疗机构、申请人、被强疗的人、检院

### related ：减刑假释检院的监督
        if 检院不服裁定；
               收到裁定书后 20days 内提出书面纠正意见；
                    法院应当另行组成合议庭 + 1mon 做出决定
                       if 检院认为最终裁定还是不合法；
                              应当向同级法院提出纠正意见

### realted:暂予监外执行的监督
        if 监狱、看守所提出 暂予监外执行的书面意见；
              应当将书面意见的副本 抄送 检院
                   检院 可以 向决定机关||批准机关 提出书面纠正意见
             
        if 检院不服；
              接到通知之日起 1mon 内  提出书面纠正意见 + 送交批准机关
                   批准机关 应当立即重新核查

### realted:暂予监外执行的机关
#### 交付执行前 
        由交付执行的法院 决定
            if 法院决定 暂予监外执行；
                  法院负责组织： 1）病情诊断
                               2）妊娠检查
                               3）生活不能自理的鉴别
                                         
                  公安负责：对罪犯送交 执行                          

#### 交付执行后
         if 有期；
               监狱 提出书面意见+ 报省级 监狱管理机关 批准
         if 拘役；
               看守所、拘役所 提出书面意见 + 市级公安 批准
